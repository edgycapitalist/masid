**Architecture Document: In-Memory Task Queue**

## Overview

The in-memory task queue is designed to manage tasks with priorities, track task status, and support task retry with configurable max retries. The system consists of the following components:

1. **Task Queue**: A data structure that stores tasks with their respective priorities.
2. **Task Manager**: Responsible for adding tasks to the queue, assigning tasks to worker threads, and tracking task status.
3. **Worker Threads**: Consume tasks from the queue, execute them, and report task status.
4. **Task Status Tracker**: Maintains the status of each task (pending, running, completed, failed).

## System Components

### Task Queue

* **Data Structure**: A priority queue (implemented using a heap) to store tasks with their priorities.
* **Operations**:
	+ `add_task(task, priority)`: Adds a task to the queue with the specified priority.
	+ `get_task()`: Retrieves the highest-priority task from the queue.

### Task Manager

* **Responsibilities**:
	+ Adds tasks to the queue.
	+ Assigns tasks to worker threads.
	+ Tracks task status.
* **Methods**:
	+ `add_task(task, priority)`: Adds a task to the queue.
	+ `assign_task(worker_thread)`: Assigns a task to a worker thread.
	+ `update_task_status(task_id, status)`: Updates the status of a task.

### Worker Threads

* **Responsibilities**:
	+ Consume tasks from the queue.
	+ Execute tasks.
	+ Report task status.
* **Methods**:
	+ `consume_task()`: Retrieves a task from the queue.
	+ `execute_task(task)`: Executes a task.
	+ `report_task_status(task_id, status)`: Reports the status of a task.

### Task Status Tracker

* **Responsibilities**:
	+ Maintains the status of each task.
* **Methods**:
	+ `get_task_status(task_id)`: Retrieves the status of a task.
	+ `update_task_status(task_id, status)`: Updates the status of a task.

```python
import heapq
import threading
import time
from enum import Enum

class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id, func, priority, max_retries=3):
        self.task_id = task_id
        self.func = func
        self.priority = priority
        self.max_retries = max_retries
        self.retries = 0
        self.status = TaskStatus.PENDING

class TaskQueue:
    def __init__(self):
        self.queue = []
        self.lock = threading.Lock()

    def add_task(self, task):
        with self.lock:
            heapq.heappush(self.queue, (task.priority, task))

    def get_task(self):
        with self.lock:
            if self.queue:
                return heapq.heappop(self.queue)[1]
            return None

class TaskManager:
    def __init__(self, task_queue):
        self.task_queue = task_queue
        self.worker_threads = []
        self.task_status_tracker = {}

    def add_task(self, task):
        self.task_queue.add_task(task)
        self.task_status_tracker[task.task_id] = task.status

    def assign_task(self, worker_thread):
        task = self.task_queue.get_task()
        if task:
            worker_thread.task = task
            worker_thread.start()

    def update_task_status(self, task_id, status):
        self.task_status_tracker[task_id] = status

class WorkerThread(threading.Thread):
    def __init__(self, task_manager):
        super().__init__()
        self.task_manager = task_manager
        self.task = None

    def run(self):
        if self.task:
            self.task.status = TaskStatus.RUNNING
            self.task_manager.update_task_status(self.task.task_id, self.task.status)
            try:
                self.task.func()
                self.task.status = TaskStatus.COMPLETED
            except Exception as e:
                self.task.status = TaskStatus.FAILED
                self.task.retries += 1
                if self.task.retries < self.task.max_retries:
                    self.task_manager.add_task(self.task)
            finally:
                self.task_manager.update_task_status(self.task.task_id, self.task.status)
```

```python
        self.max_retries = max_retries
        self.retries = 0
        self.status = TaskStatus.PENDING

class TaskQueue:
    def __init__(self):
        self.queue = []
        self.lock = threading.Lock()

    def add_task(self, task):
        with self.lock:
            heapq.heappush(self.queue, (task.priority, task))

    def get_task(self):
        with self.lock:
            if self.queue:
                return heapq.heappop(self.queue)[1]
            return None

class TaskManager:
    def __init__(self, task_queue):
        self.task_queue = task_queue
        self.worker_threads = []
        self.task_status_tracker = {}
```
```python
        self.shutdown = threading.Event()

    def add_task(self, task):

        self.task_status_tracker[task.task_id] = task.status

    def assign_task(self, worker_thread):
        task = self.task_queue.get_task()

            worker_thread.start()

    def update_task_status(self, task_id, status):
        self.task_status_tracker[task_id] = status

    def shutdown_manager(self):

        for worker_thread in self.worker_threads:
```

```python
class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id, func, priority, max_retries=3):
        self.task_id = task_id
        self.func = func
        self.priority = priority
        self.max_retries = max_retries
        self.retries = 0
        self.status = TaskStatus.PENDING

class TaskQueue:
    def __init__(self):
        self.queue = []
        self.lock = threading.Lock()

    def add_task(self, task):
        with self.lock:
            heapq.heappush(self.queue, (task.priority, task))

    def get_task(self):
        with self.lock:
            if self.queue:
                return heapq.heappop(self.queue)[1]
            return None

class TaskManager:
```
```python
    def __init__(self, task_queue, max_workers=5):
        self.task_queue = task_queue
        self.max_workers = max_workers
        self.worker_threads = []
        self.task_status_tracker = {}
        self.shutdown = threading.Event()

    def add_task(self, task):
        self.task_queue.add_task(task)
        self.task_status_tracker[task.task_id] = task.status

    def assign_task(self, worker_thread):
        task = self.task_queue.get_task()
        if task:
            worker_thread.task = task
            worker_thread.start()

    def update_task_status(self, task_id, status):

        for worker_thread in self.worker_threads:

        while not self.task_manager.shutdown.is_set():
            task = self.task_manager.task_queue.get_task()
            if task:
                task.status = TaskStatus.RUNNING
                self.task_manager.update_task_status(task.task_id, task.status)
                try:
                    task.func()
                    task.status = TaskStatus.COMPLETED
                except Exception as e:
                    task.status = TaskStatus.FAILED
                    task.retries += 1
                    if task.retries < task.max_retries:
                        self.task_manager.add_task(task)
                finally:
                    self.task_manager.update_task_status(task.task_id, task.status)
            else:
                time.sleep(0.1)
```

### Architecture Document

The system consists of the following components:

*   **Task**: Represents a single task with a unique ID, a function to execute, a priority, and a maximum number of retries.
*   **TaskQueue**: A priority queue that stores tasks. It provides methods to add tasks and retrieve the task with the highest priority.
*   **TaskManager**: Manages the task queue and worker threads. It provides methods to add tasks, assign tasks to worker threads, and update the status of tasks.
*   **WorkerThread**: A thread that executes tasks. It has a reference to the task manager and the task it is currently executing.

The system works as follows:

1.  **Task Creation**: A task is created with a unique ID, a function to execute, a priority, and a maximum number of retries.
2.  **Task Addition**: The task is added to the task queue.
3.  **Task Assignment**: A worker thread is assigned the task with the highest priority from the task queue.
4.  **Task Execution**: The worker thread executes the task. If the task fails, it is retried up to the maximum number of retries.
5.  **Task Status Update**: The status of the task is updated in the task manager.
6.  **Task Completion**: If the task completes successfully, its status is updated to completed. If all retries fail, its status is updated to failed.
