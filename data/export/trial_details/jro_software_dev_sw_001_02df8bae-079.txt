**Architecture Document: URL Shortener Library**

## Overview

The URL Shortener Library is a Python library designed to provide a hash-based URL shortening functionality, supporting reverse lookup, collision detection, input validation, and thread safety. This document outlines the high-level structure, module decomposition, interfaces, data flow, and technology choices for the library.

## Module Decomposition

The library will consist of the following modules:

1. **url_shortener**: The main module responsible for URL shortening and reverse lookup functionality.
2. **hash_generator**: A module responsible for generating a unique hash for a given URL.
3. **collision_detector**: A module responsible for detecting collisions and handling them accordingly.
4. **validator**: A module responsible for input validation, ensuring that the provided URL is in a valid format.
5. **database**: A module responsible for storing and retrieving the mapping between short and long URLs.

## Interfaces

The library will expose the following interfaces:

1. **shorten_url(url: str) -> str**: Takes a long URL as input and returns a shortened version.
2. **get_original_url(short_url: str) -> str**: Takes a short URL as input and returns the original URL.
3. **is_valid_url(url: str) -> bool**: Takes a URL as input and returns a boolean indicating whether the URL is in a valid format.

## Data Flow

The data flow for the library is as follows:

1. The user provides a long URL to the **shorten_url** interface.
2. The **validator** module checks if the provided URL is in a valid format.
3. If the URL is valid, the **hash_generator** module generates a unique hash for the URL.
4. The **collision_detector** module checks if the generated hash is already in use.
5. If a collision is detected, the **collision_detector** module generates a new hash and repeats the process until a unique hash is found.
6. The **database** module stores the mapping between the short and long URLs.
7. The user can retrieve the original URL by providing the short URL to the **get_original_url** interface.

The library will use the following technologies:

1. **Python 3.x**: The library will be implemented using Python 3.x, ensuring compatibility with the latest version of the language.
2. **SHA-256**: The library will use the SHA-256 hashing algorithm to generate unique hashes for the URLs.
3. **SQLite**: The library will use SQLite as the database to store the mapping between short and long URLs.

# url_shortener.py
import hashlib
import sqlite3
import threading
from urllib.parse import urlparse

class URLShortener:
    def __init__(self, db_name):
        self.db_name = db_name
        self.lock = threading.Lock()
        self.conn = sqlite3.connect(db_name)
        self.cursor = self.conn.cursor()
        self.cursor.execute(
            """CREATE TABLE IF NOT EXISTS urls
            (short_url TEXT PRIMARY KEY, long_url TEXT)"""
        )
        self.conn.commit()

    def shorten_url(self, url):
        if not self.is_valid_url(url):
            raise ValueError("Invalid URL")
        hash_object = hashlib.sha256(url.encode())
        short_url = hash_object.hexdigest()[:6]
        with self.lock:
            if self.cursor.execute("SELECT * FROM urls WHERE short_url = ?", (short_url,)).fetchone():
                # Collision detected, generate a new hash
                short_url = self._generate_new_hash(url)
            self.cursor.execute("INSERT INTO urls VALUES (?, ?)", (short_url, url))
            self.conn.commit()
        return short_url

    def get_original_url(self, short_url):
        with self.lock:
            result = self.cursor.execute("SELECT long_url FROM urls WHERE short_url = ?", (short_url,)).fetchone()
            if result:
                return result[0]
            else:
                return None

    def is_valid_url(self, url):
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

    def _generate_new_hash(self, url):
        # Generate a new hash by appending a counter to the URL
        counter = 0
        while True:
            hash_object = hashlib.sha256((url + str(counter)).encode())
            short_url = hash_object.hexdigest()[:6]
            with self.lock:
                if not self.cursor.execute("SELECT * FROM urls WHERE short_url = ?", (short_url,)).fetchone():
                    return short_url
            counter += 1

# Example usage:

short_url = shortener.shorten_url("https://www.example.com")
print(short_url)
original_url = shortener.get_original_url(short_url)
