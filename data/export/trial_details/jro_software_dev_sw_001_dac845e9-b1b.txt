**Architecture Document: URL Shortening Library**

## Introduction

This document outlines the design and implementation of a Python library that provides URL shortening functionality using a hash-based approach. The library is designed to be thread-safe, handle collision detection, and perform input validation.

## System Components

The library will consist of the following components:

1. **URLValidator**: Responsible for validating the input URL format.
2. **URLShortener**: Generates a shortened URL using a hash-based approach and stores the mapping between the original and shortened URLs.
3. **URLMapper**: Provides reverse lookup functionality from shortened URL to original URL.
4. **CollisionDetector**: Detects and handles collisions that may occur during the hashing process.

## System Interfaces

The library will provide the following interfaces:

1. **shorten_url(long_url)**: Accepts a long URL and returns a shortened version.
2. **get_original_url(short_url)**: Accepts a shortened URL and returns the original URL.
3. **validate_url(url)**: Validates the input URL format.

## Data Flow

The data flow between components is as follows:

1. The user calls the `shorten_url` method, passing a long URL.
2. The `URLValidator` validates the input URL format.
3. If the URL is valid, the `URLShortener` generates a shortened URL using a hash-based approach.
4. The `URLMapper` stores the mapping between the original and shortened URLs.
5. If a collision is detected, the `CollisionDetector` handles it by generating a new shortened URL.
6. The user can call the `get_original_url` method to retrieve the original URL from a shortened URL.

## Technology Choices

The library will use the following technologies:

1. **Hashing algorithm**: SHA-256
2. **Data storage**: In-memory dictionary (for simplicity and performance)
3. **Thread safety**: Locks (using the `threading` module)

## Implementation

```python
import hashlib
import threading
import re

# URLValidator
class URLValidator:
    def __init__(self):
        self.url_pattern = re.compile(r"^https?://[^\s]+$")

    def validate_url(self, url):
        return bool(self.url_pattern.match(url))

# URLShortener
class URLShortener:
    def __init__(self):
        self.url_map = {}
        self.lock = threading.Lock()

    def shorten_url(self, long_url):
        with self.lock:
            hash_value = hashlib.sha256(long_url.encode()).hexdigest()[:6]
            if hash_value in self.url_map:
                # Collision detected, generate a new shortened URL
                hash_value = hashlib.sha256((long_url + str(hash_value)).encode()).hexdigest()[:6]
            self.url_map[hash_value] = long_url
            return hash_value

# URLMapper
class URLMapper:
    def __init__(self, url_shortener):
        self.url_shortener = url_shortener

    def get_original_url(self, short_url):
        with self.url_shortener.lock:
            return self.url_shortener.url_map.get(short_url)

# URLShorteningLibrary
class URLShorteningLibrary:
    def __init__(self):
        self.url_validator = URLValidator()
        self.url_shortener = URLShortener()
        self.url_mapper = URLMapper(self.url_shortener)

    def shorten_url(self, long_url):
        if not self.url_validator.validate_url(long_url):
            raise ValueError("Invalid URL format")
        return self.url_shortener.shorten_url(long_url)

    def get_original_url(self, short_url):
        return self.url_mapper.get_original_url(short_url)

# Example usage
library = URLShorteningLibrary()
long_url = "https://www.example.com"
short_url = library.shorten_url(long_url)
print(f"Shortened URL: {short_url}")
print(f"Original URL: {library.get_original_url(short_url)}")

5. If a collision is detected, the `CollisionDetector` handles it by generating a new shortened URL.
6. The user can call the `get_original_url` method to retrieve the original URL from a shortened URL.

import hashlib
import threading
import re

# URLValidator
class URLValidator:
    def __init__(self):
        self.url_pattern = re.compile(r"^https?://[^\s]+$")

    def validate_url(self, url):
        return bool(self.url_pattern.match(url))

# URLShortener
class URLShortener:
    def __init__(self):
        self.url_map = {}
        self.lock = threading.Lock()

    def shorten_url(self, long_url):
        with self.lock:
            try:
                hash_value = hashlib.sha256(long_url.encode()).hexdigest()[:6]
                if hash_value in self.url_map:
                    # Collision detected, generate a new shortened URL
                    hash_value = hashlib.sha256((long_url + str(hash_value)).encode()).hexdigest()[:6]
                    # Handle cases where the new hash value also collides
                    while hash_value in self.url_map:
                        hash_value = hashlib.sha256((long_url + str(hash_value)).encode()).hexdigest()[:6]
                self.url_map[hash_value] = long_url
                return hash_value
            except Exception as e:
                # Handle exceptions during the hashing process
                print(f"Error shortening URL: {e}")
                return None

# URLMapper
class URLMapper:
    def __init__(self, url_shortener):
        self.url_shortener = url_shortener

    def get_original_url(self, short_url):
        with self.url_shortener.lock:
            try:
                return self.url_shortener.url_map.get(short_url)
            except Exception as e:
                # Handle exceptions during the retrieval process
                print(f"Error retrieving original URL: {e}")
                return None

# URLShorteningLibrary
class URLShorteningLibrary:
    def __init__(self):
        self.url_validator = URLValidator()
        self.url_shortener = URLShortener()
        self.url_mapper = URLMapper(self.url_shortener)

    def shorten_url(self, long_url):
        if not self.url_validator.validate_url(long_url):
            raise ValueError("Invalid URL format")
        return self.url_shortener.shorten_url(long_url)

    def get_original_url(self, short_url):
        return self.url_mapper.get_original_url(short_url)

import hashlib
import threading
import re

# URLValidator
class URLValidator:
    def __init__(self):
        self.url_pattern = re.compile(r"^https?://[^\s]+$")

    def validate_url(self, url):
        return bool(self.url_pattern.match(url))

# URLShortener
class URLShortener:
    def __init__(self):
        self.url_map = {}
        self.lock = threading.Lock()

    def shorten_url(self, long_url):
        with self.lock:
            try:
                hash_value = hashlib.sha256(long_url.encode()).hexdigest()[:6]
                if hash_value in self.url_map:
                    # Collision detected, generate a new shortened URL
                    hash_value = hashlib.sha256((long_url + str(hash_value)).encode()).hexdigest()[:6]
                    # Handle cases where the new hash value also collides
                    while hash_value in self.url_map:
                        hash_value = hashlib.sha256((long_url + str(hash_value)).encode()).hexdigest()[:6]
                self.url_map[hash_value] = long_url
                return hash_value
            except Exception as e:
                # Handle exceptions during the hashing process
                print(f"Error shortening URL: {e}")
                return None

# URLMapper
class URLMapper:
    def __init__(self, url_shortener):
        self.url_shortener = url_shortener

    def get_original_url(self, short_url):
        with self.url_shortener.lock:
            try:
                return self.url_shortener.url_map.get(short_url)
            except Exception as e:
                # Handle exceptions during the retrieval process
                print(f"Error retrieving original URL: {e}")
                return None

# URLShorteningLibrary
class URLShorteningLibrary:
    def __init__(self):
        self.url_validator = URLValidator()
        self.url_shortener = URLShortener()
        self.url_mapper = URLMapper(self.url_shortener)

    def shorten_url(self, long_url):
        if not self.url_validator.validate_url(long_url):
            raise ValueError("Invalid URL format")
        return self.url_shortener.shorten_url(long_url)

    def get_original_url(self, short_url):
        return self.url_mapper.get_original_url(short_url)
