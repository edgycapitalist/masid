# Introduction
The URL shortening library is designed to provide a simple and efficient way to shorten URLs using a hash-based approach. The library will support reverse lookup, collision detection, input validation, and thread safety.

## System Components

*   **URLShortener**: This is the main class responsible for shortening URLs and performing reverse lookups.
*   **URLValidator**: This class validates the input URL to ensure it is in a valid format.
*   **HashGenerator**: This class generates a unique hash for each URL.
*   **CollisionDetector**: This class detects and handles collisions when two different URLs generate the same hash.

## System Flow

1.  **Shorten URL**:
    *   The user provides a long URL to the `URLShortener`.
    *   The `URLValidator` validates the input URL.
    *   The `HashGenerator` generates a unique hash for the URL.
    *   The `CollisionDetector` checks for collisions.
    *   If a collision is detected, the `HashGenerator` generates a new hash.
    *   The shortened URL is returned to the user.
2.  **Reverse Lookup**:
    *   The user provides a shortened URL to the `URLShortener`.
    *   The `URLShortener` performs a reverse lookup to find the original URL.

## Data Structures

*   **URL Mapping**: A dictionary that maps shortened URLs to their original URLs.
*   **Hash Set**: A set that stores the generated hashes to detect collisions.

## Thread Safety

*   The `URLShortener` class will use a lock to synchronize access to the `URL Mapping` and `Hash Set` to ensure thread safety.

## Error Handling

*   **Invalid URL**: If the input URL is not in a valid format, the `URLValidator` will raise a `ValueError`.
*   **Collision**: If a collision is detected, the `CollisionDetector` will raise a `CollisionError`.

import hashlib
import threading
from urllib.parse import urlparse

class URLValidator:
    def __init__(self):
        pass

    def validate(self, url):
        """
        Validate the input URL.

        Args:
            url (str): The input URL.

        Raises:
            ValueError: If the input URL is not in a valid format.
        """
        try:
            result = urlparse(url)
            if not all([result.scheme, result.netloc]):
                raise ValueError("Invalid URL")
        except ValueError as e:
            raise ValueError("Invalid URL") from e


class HashGenerator:
    def __init__(self):
        pass

    def generate(self, url):
        """
        Generate a unique hash for the URL.

        Args:
            url (str): The input URL.

        Returns:
            str: The generated hash.
        """
        return hashlib.sha256(url.encode()).hexdigest()[:6]


class CollisionDetector:
    def __init__(self):
        self.hashes = set()

    def detect(self, hash):
        """
        Detect collisions.

        Args:
            hash (str): The generated hash.

        Raises:
            CollisionError: If a collision is detected.
        """
        if hash in self.hashes:
            raise CollisionError("Collision detected")
        self.hashes.add(hash)


class CollisionError(Exception):
    pass


class URLShortener:
    def __init__(self):
        self.url_mapping = {}
        self.lock = threading.Lock()
        self.validator = URLValidator()
        self.hash_generator = HashGenerator()
        self.collision_detector = CollisionDetector()

    def shorten(self, url):
        """
        Shorten the URL.

        Args:
            url (str): The input URL.

        Returns:
            str: The shortened URL.
        """
        with self.lock:
            self.validator.validate(url)
            hash = self.hash_generator.generate(url)
            try:
                self.collision_detector.detect(hash)
            except CollisionError:
                # Handle collision by generating a new hash
                hash = self.hash_generator.generate(url + str(hash))
                self.collision_detector.detect(hash)
            shortened_url = f"http://short.url/{hash}"
            self.url_mapping[shortened_url] = url
            return shortened_url

    def reverse_lookup(self, shortened_url):
        """
        Perform a reverse lookup.

        Args:
            shortened_url (str): The shortened URL.

        Returns:
            str: The original URL.
        """
        with self.lock:
            return self.url_mapping.get(shortened_url)


# Example usage
if __name__ == "__main__":
    shortener = URLShortener()
    url = "https://www.example.com"
    shortened_url = shortener.shorten(url)
    print(f"Shortened URL: {shortened_url}")
    original_url = shortener.reverse_lookup(shortened_url)
    print(f"Original URL: {original_url}")
