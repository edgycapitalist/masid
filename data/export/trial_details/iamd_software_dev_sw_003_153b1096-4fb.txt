**Architecture Document: Python Data Pipeline**

## Overview
The Python data pipeline is designed to process CSV files with configurable schema validation, apply a chain of transformations, and output results to CSV or JSON. The pipeline also supports a plugin system for custom transformations and provides clear error reporting for malformed data.

## Components
The pipeline consists of the following components:

1. **Data Reader**: Responsible for reading CSV files with configurable schema validation.
2. **Transformation Chain**: Applies a chain of transformations (filtering, mapping, aggregation) to the data.
3. **Plugin System**: Allows for custom transformations to be added to the pipeline.
4. **Data Writer**: Outputs the transformed data to CSV or JSON.
5. **Error Reporter**: Provides clear error reporting for malformed data.

## Data Flow
The data flow of the pipeline is as follows:

1. The **Data Reader** reads a CSV file and validates its schema.
2. The validated data is passed to the **Transformation Chain**, which applies a series of transformations to the data.
3. The transformed data is then passed to the **Data Writer**, which outputs the data to CSV or JSON.
4. If any errors occur during the pipeline, the **Error Reporter** provides clear error reporting.

## Interfaces
The following interfaces are defined for the pipeline:

1. **Data Reader Interface**: `read_data(file_path: str, schema: dict) -> pd.DataFrame`
2. **Transformation Interface**: `transform(data: pd.DataFrame) -> pd.DataFrame`
3. **Plugin Interface**: `plugin_transform(data: pd.DataFrame) -> pd.DataFrame`
4. **Data Writer Interface**: `write_data(data: pd.DataFrame, output_format: str) -> None`
5. **Error Reporter Interface**: `report_error(error: Exception) -> None`

import pandas as pd
import json

# Data Reader
class CSVReader:
    def __init__(self, schema):
        self.schema = schema

    def read_data(self, file_path):
        try:
            data = pd.read_csv(file_path)
            self.validate_schema(data)
            return data
        except Exception as e:
            ErrorReporter().report_error(e)

    def validate_schema(self, data):
        for column in self.schema:
            if column not in data.columns:
                raise ValueError(f"Column {column} not found in data")

# Transformation Chain
class TransformationChain:
    def __init__(self, transformations):
        self.transformations = transformations

    def apply_transformations(self, data):
        for transformation in self.transformations:
            data = transformation.transform(data)
        return data

# Plugin System
class PluginSystem:
    def __init__(self, plugins):
        self.plugins = plugins

    def apply_plugins(self, data):
        for plugin in self.plugins:
            data = plugin.plugin_transform(data)
        return data

# Data Writer
class DataWriter:
    def __init__(self, output_format):
        self.output_format = output_format

    def write_data(self, data):
        if self.output_format == "csv":
            data.to_csv("output.csv", index=False)
        elif self.output_format == "json":
            data.to_json("output.json", orient="records")

# Error Reporter
class ErrorReporter:
    def report_error(self, error):
        print(f"Error: {error}")

# Example usage
if __name__ == "__main__":
    schema = {"name": str, "age": int}
    reader = CSVReader(schema)
    data = reader.read_data("input.csv")

    transformations = [FilteringTransformation(), MappingTransformation()]

    transformed_data = chain.apply_transformations(data)

    plugins = [CustomPlugin()]
    plugin_system = PluginSystem(plugins)
    transformed_data = plugin_system.apply_plugins(transformed_data)

    writer = DataWriter("csv")
    writer.write_data(transformed_data)
