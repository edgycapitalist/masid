**Architecture Document: In-Memory Task Queue**

## Introduction
The in-memory task queue is a critical component of our project, enabling the efficient execution of tasks with varying priorities. This document outlines the high-level design and architecture of the task queue, ensuring seamless integration with other project components.

## Overview
The task queue will be implemented as a Python module, utilizing a thread-safe data structure to store tasks. The design will incorporate the following key features:

1. **Task Priority**: Tasks can be added with a specified priority, influencing their execution order.
2. **Worker Threads**: Multiple worker threads will consume tasks from the queue, executing them concurrently.
3. **Task Status Tracking**: Tasks will have a status (pending, running, completed, failed), updated throughout their lifecycle.
4. **Task Retry**: Failed tasks can be retried with a configurable maximum number of attempts.
5. **Graceful Shutdown**: The task queue will support a controlled shutdown, ensuring tasks are completed or terminated cleanly.

## Module Decomposition

The task queue module will consist of the following sub-modules:

1. **Task**: Represents a single task with attributes (id, priority, status, max_retries, etc.).
2. **TaskQueue**: Manages the task queue data structure and provides methods for adding, consuming, and updating tasks.
3. **WorkerThread**: Responsible for executing tasks from the queue.
4. **TaskExecutor**: Handles task execution, including retry logic and status updates.

## Interfaces

The task queue module will expose the following interfaces:

1. **TaskQueue.add_task()**: Adds a task to the queue with a specified priority.
2. **TaskQueue.get_task()**: Retrieves a task from the queue for execution.
3. **TaskQueue.update_task_status()**: Updates the status of a task.
4. **TaskExecutor.execute_task()**: Executes a task and handles retry logic.

## Data Flow

The task queue module will operate as follows:

1. Tasks are added to the queue using **TaskQueue.add_task()**.
2. Worker threads consume tasks from the queue using **TaskQueue.get_task()**.
3. Tasks are executed by the **TaskExecutor**.
4. Task status is updated using **TaskQueue.update_task_status()**.
5. Failed tasks are retried by the **TaskExecutor** up to a configurable maximum number of attempts.

## Technology Choices

The task queue module will utilize the following technologies:

1. **Python**: As the primary programming language.
2. **threading**: For creating and managing worker threads.
3. **queue**: For implementing the task queue data structure.
4. **logging**: For logging task execution and errors.

import threading
import queue
import logging

# Task class
class Task:
    def __init__(self, id, priority, max_retries):
        self.id = id
        self.priority = priority
        self.max_retries = max_retries
        self.status = 'pending'
        self.retries = 0

# Task queue class
class TaskQueue:
    def __init__(self):
        self.queue = queue.PriorityQueue()
        self.lock = threading.Lock()

    def add_task(self, task):
        with self.lock:
            self.queue.put((task.priority, task))

    def get_task(self):
        with self.lock:
            if self.queue.empty():
                return None
            _, task = self.queue.get()
            return task

    def update_task_status(self, task, status):
        with self.lock:
            task.status = status

# Task executor class
class TaskExecutor:
    def __init__(self, task_queue):
        self.task_queue = task_queue

    def execute_task(self, task):
        try:
            # Execute task logic here
            logging.info(f'Task {task.id} executed successfully')
            self.task_queue.update_task_status(task, 'completed')
        except Exception as e:
            logging.error(f'Task {task.id} failed: {e}')
            if task.retries < task.max_retries:
                task.retries += 1
                self.task_queue.add_task(task)
            else:
                self.task_queue.update_task_status(task, 'failed')

# Worker thread class
class WorkerThread(threading.Thread):
    def __init__(self, task_queue, task_executor):
        super().__init__()
        self.task_queue = task_queue
        self.task_executor = task_executor

    def run(self):
        while True:
            task = self.task_queue.get_task()
            if task is None:
                break
            self.task_executor.execute_task(task)

# Example usage
if __name__ == '__main__':
    task_queue = TaskQueue()
    task_executor = TaskExecutor(task_queue)

    # Add tasks to the queue
    task1 = Task(1, 1, 3)
    task2 = Task(2, 2, 2)
    task_queue.add_task(task1)
    task_queue.add_task(task2)

    # Start worker threads
    worker_threads = []
    for _ in range(2):
        worker_thread = WorkerThread(task_queue, task_executor)
        worker_thread.start()
        worker_threads.append(worker_thread)

    # Wait for worker threads to complete
    for worker_thread in worker_threads:
        worker_thread.join()

import logging
import time

class Task:
    def __init__(self, id, priority, max_retries):
        self.id = id
        self.priority = priority
        self.max_retries = max_retries
        self.status = 'pending'
        self.retries = 0

class TaskQueue:
    def __init__(self):
        self.queue = queue.PriorityQueue()
        self.lock = threading.Lock()
        self.tasks = {}

    def add_task(self, task):
        with self.lock:
            self.queue.put((task.priority, task))
            self.tasks[task.id] = task

    def get_task(self):
        with self.lock:
            if self.queue.empty():
                return None
            _, task = self.queue.get()
            return task

    def update_task_status(self, task, status):
        with self.lock:
            task.status = status
            self.tasks[task.id] = task

    def get_task_by_id(self, task_id):
        with self.lock:
            return self.tasks.get(task_id)

# Task executor class
class TaskExecutor:
    def __init__(self, task_queue):
        self.task_queue = task_queue

    def execute_task(self, task):
        try:
            # Execute task logic here
            logging.info(f'Task {task.id} executed successfully')
            self.task_queue.update_task_status(task, 'completed')
        except Exception as e:
            logging.error(f'Task {task.id} failed: {e}')
            task.last_error = str(e)
            if task.retries < task.max_retries:
                task.retries += 1
                time.sleep(2 ** task.retries)  # Exponential backoff
                self.task_queue.add_task(task)
            else:
                self.task_queue.update_task_status(task, 'failed')

    def __init__(self, task_queue, task_executor):
        super().__init__()
        self.task_queue = task_queue
        self.task_executor = task_executor

        while not self.stop_event.is_set():
            task = self.task_queue.get_task()
            if task is None:
                time.sleep(0.1)
                continue
            self.task_executor.execute_task(task)

        self.stop_event.set()

import queue
import logging
import time

class Task:
    def __init__(self, id, priority, max_retries):
        self.id = id
        self.priority = priority
        self.max_retries = max_retries
        self.status = 'pending'
        self.retries = 0
        self.last_error = None

# Task queue class
class TaskQueue:
    def __init__(self):
        self.queue = queue.PriorityQueue()
        self.lock = threading.Lock()
        self.tasks = {}

    def add_task(self, task):
        with self.lock:
            self.queue.put((task.priority, task))
            self.tasks[task.id] = task

    def get_task(self):
        with self.lock:
            if self.queue.empty():
                return None
            _, task = self.queue.get()
            return task

    def update_task_status(self, task, status):
        with self.lock:
            task.status = status
            self.tasks[task.id] = task

    def get_task_by_id(self, task_id):
        with self.lock:
            return self.tasks.get(task_id)

# Task executor class
class TaskExecutor:
    def __init__(self, task_queue):
        self.task_queue = task_queue

    def execute_task(self, task):
        try:
            # Execute task logic here
            logging.info(f'Task {task.id} executed successfully')
            self.task_queue.update_task_status(task, 'completed')
        except Exception as e:
            logging.error(f'Task {task.id} failed: {e}')
            task.last_error = str(e)
            if task.retries < task.max_retries:
                task.retries += 1
                time.sleep(2 ** task.retries)  # Exponential backoff
                self.task_queue.add_task(task)
            else:
                self.task_queue.update_task_status(task, 'failed')

class WorkerThread(threading.Thread):
    def __init__(self, task_queue, task_executor):
        super().__init__()
        self.task_queue = task_queue
        self.task_executor = task_executor
        self.stop_event = threading.Event()

    def run(self):
        while not self.stop_event.is_set():
            task = self.task_queue.get_task()
            if task is None:
                time.sleep(0.1)
                continue
            self.task_executor.execute_task(task)

    def stop(self):
        self.stop_event.set()

import threading
import queue

import time

# Task class
class Task:
    def __init__(self, id, priority, max_retries):

        self.priority = priority
        self.max_retries = max_retries
        self.status = 'pending'
        self.retries = 0
        self.last_error = None

# Task queue class
class TaskQueue:
    def __init__(self):
        self.queue = queue.PriorityQueue()
        self.lock = threading.Lock()
        self.tasks = {}

    def add_task(self, task):
        with self.lock:
            self.queue.put((task.priority, task))
            self.tasks[task.id] = task

    def get_task(self):
        with self.lock:
            if self.queue.empty():
                return None
            _, task = self.queue.get()
            return task

    def update_task_status(self, task, status):
        with self.lock:
            task.status = status
            self.tasks[task.id] = task

    def get_task_by_id(self, task_id):
        with self.lock:
            return self.tasks.get(task_id)

# Task executor class
class TaskExecutor:
    def __init__(self, task_queue):
        self.task_queue = task_queue

    def execute_task(self, task):
        try:
            # Execute task logic here
            logging.info(f'Task {task.id} executed successfully')
            self.task_queue.update_task_status(task, 'completed')
        except Exception as e:
            logging.error(f'Task {task.id} failed: {e}')
            task.last_error = str(e)
            if task.retries < task.max_retries:
                task.retries += 1
                time.sleep(2 ** task.retries)  # Exponential backoff
                self.task_queue.add_task(task)
            else:
                self.task_queue.update_task_status(task, 'failed')

# Worker thread class
class WorkerThread(threading.Thread):
    def __init__(self, task_queue, task_executor):
        super().__init__()
        self.task_queue = task_queue
        self.task_executor = task_executor
        self.stop_event = threading.Event()

    def run(self):
        while not self.stop_event.is_set():
            task = self.task_queue.get_task()
            if task is None:
                time.sleep(0.1)
                continue
            self.task_executor.execute_task(task)

    def stop(self):
        self.stop_event.set()
