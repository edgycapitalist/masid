**Architecture Document: URL Shortener Library**

### Introduction

The URL shortener library is designed to provide a hash-based URL shortening functionality without relying on external services. The library will accept a long URL, generate a shortened version, and support reverse lookup from the short URL to the original URL. It will also handle collision detection, input validation, and ensure thread safety for concurrent usage.

### Module Decomposition

The library will consist of the following modules:

1. **UrlValidator**: Responsible for validating the input URL format.
2. **HashGenerator**: Generates a unique hash-based short URL.
3. **CollisionDetector**: Detects and handles hash collisions.
4. **UrlStore**: Stores the mapping between short URLs and original URLs.
5. **UrlShortener**: The main interface for the library, providing methods for shortening URLs and reverse lookup.

### Interfaces

1. **UrlValidator**:
	* `is_valid_url(url: str) -> bool`: Returns True if the URL is valid, False otherwise.
2. **HashGenerator**:
	* `generate_hash(url: str) -> str`: Generates a hash-based short URL.
3. **CollisionDetector**:
	* `detect_collision(short_url: str) -> bool`: Returns True if a collision is detected, False otherwise.
4. **UrlStore**:
	* `store_url(short_url: str, original_url: str)`: Stores the mapping between the short URL and original URL.
	* `get_original_url(short_url: str) -> str`: Returns the original URL for a given short URL.
5. **UrlShortener**:
	* `shorten_url(url: str) -> str`: Shortens a URL and returns the short URL.
	* `get_original_url(short_url: str) -> str`: Returns the original URL for a given short URL.

### Data Flow

1. The user provides a long URL to the **UrlShortener**.
2. The **UrlShortener** calls the **UrlValidator** to validate the input URL.
3. If the URL is valid, the **UrlShortener** calls the **HashGenerator** to generate a hash-based short URL.
4. The **UrlShortener** calls the **CollisionDetector** to detect any hash collisions.
5. If a collision is detected, the **UrlShortener** generates a new hash-based short URL.
6. The **UrlShortener** stores the mapping between the short URL and original URL using the **UrlStore**.
7. The user can call the **UrlShortener** to get the original URL for a given short URL.

import hashlib
import threading
from urllib.parse import urlparse

class UrlValidator:
    def is_valid_url(self, url: str) -> bool:
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

class HashGenerator:
    def generate_hash(self, url: str) -> str:
        return hashlib.sha256(url.encode()).hexdigest()[:6]

class CollisionDetector:
    def __init__(self):
        self.lock = threading.Lock()
        self.short_urls = set()

    def detect_collision(self, short_url: str) -> bool:
        with self.lock:
            return short_url in self.short_urls

    def add_short_url(self, short_url: str):
        with self.lock:
            self.short_urls.add(short_url)

class UrlStore:
    def __init__(self):
        self.lock = threading.Lock()
        self.url_map = {}

    def store_url(self, short_url: str, original_url: str):
        with self.lock:
            self.url_map[short_url] = original_url

    def get_original_url(self, short_url: str) -> str:
        with self.lock:
            return self.url_map.get(short_url)

class UrlShortener:
    def __init__(self):
        self.url_validator = UrlValidator()
        self.hash_generator = HashGenerator()
        self.collision_detector = CollisionDetector()
        self.url_store = UrlStore()

    def shorten_url(self, url: str) -> str:
        if not self.url_validator.is_valid_url(url):
            raise ValueError("Invalid URL")

        short_url = self.hash_generator.generate_hash(url)
        while self.collision_detector.detect_collision(short_url):
            short_url = self.hash_generator.generate_hash(url)

        self.collision_detector.add_short_url(short_url)
        self.url_store.store_url(short_url, url)
        return short_url

    def get_original_url(self, short_url: str) -> str:
        return self.url_store.get_original_url(short_url)

# Example usage:

print(short_url)
print(url_shortener.get_original_url(short_url))
