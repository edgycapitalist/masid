**Architecture Document: URL Shortener Library**

## Introduction

The URL Shortener Library is a Python library designed to provide a simple and efficient way to shorten URLs using a hash-based approach. The library supports reverse lookup, collision detection, input validation, and is thread-safe for concurrent usage.

## Components

1. **URLValidator**: Responsible for validating the input URL format.
2. **URLShortener**: Generates a shortened URL using a hash-based approach.
3. **URLMapper**: Maps the shortened URL to the original URL and vice versa.
4. **CollisionDetector**: Detects and handles collisions when generating shortened URLs.

## Interfaces

1. **shorten_url(long_url)**: Accepts a long URL and returns a shortened version.
2. **get_original_url(short_url)**: Accepts a shortened URL and returns the original URL.
3. **is_valid_url(url)**: Validates the input URL format.

## Data Flow

1. The user provides a long URL to the `shorten_url` method.
2. The `URLValidator` validates the input URL format.
3. The `URLShortener` generates a shortened URL using a hash-based approach.
4. The `URLMapper` maps the shortened URL to the original URL.
5. The `CollisionDetector` checks for collisions and generates a new shortened URL if necessary.
6. The user provides a shortened URL to the `get_original_url` method.
7. The `URLMapper` retrieves the original URL from the mapping.

## Technology Choices

1. **Hashing Algorithm**: SHA-256 for generating shortened URLs.
2. **Data Structure**: Dictionary (hash table) for storing the URL mappings.
3. **Thread Safety**: Using the `threading` module to ensure thread safety.

## Implementation Code

```python
import hashlib
import threading
from urllib.parse import urlparse

class URLValidator:
    def is_valid_url(self, url):
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

class URLShortener:
    def __init__(self):
        self.url_map = {}
        self.lock = threading.Lock()

    def shorten_url(self, long_url):
        if not URLValidator().is_valid_url(long_url):
            raise ValueError("Invalid URL format")

        with self.lock:
            hash_object = hashlib.sha256(long_url.encode())
            short_url = hash_object.hexdigest()[:6]
            if short_url in self.url_map:
                # Collision detected, generate a new shortened URL
                short_url = self._generate_new_short_url(long_url)
            self.url_map[short_url] = long_url
            return short_url

    def get_original_url(self, short_url):
        with self.lock:
            return self.url_map.get(short_url)

    def _generate_new_short_url(self, long_url):
        # Generate a new shortened URL by appending a counter
        counter = 0
        while True:
            new_short_url = f"{hashlib.sha256(long_url.encode()).hexdigest()[:6]}{counter}"
            if new_short_url not in self.url_map:
                return new_short_url
            counter += 1

class URLShortenerLibrary:
    def __init__(self):
        self.url_shortener = URLShortener()

    def shorten_url(self, long_url):
        return self.url_shortener.shorten_url(long_url)

    def get_original_url(self, short_url):
        return self.url_shortener.get_original_url(short_url)

    def is_valid_url(self, url):
        return URLValidator().is_valid_url(url)

# Example usage:
library = URLShortenerLibrary()
long_url = "https://www.example.com/very/long/url"
short_url = library.shorten_url(long_url)
print(f"Shortened URL: {short_url}")
print(f"Original URL: {library.get_original_url(short_url)}")
```

```python
import hashlib
import threading
from urllib.parse import urlparse

class URLValidator:
    def is_valid_url(self, url):
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

class URLShortener:
    def __init__(self):
        self.url_map = {}
        self.lock = threading.Lock()

    def shorten_url(self, long_url):
        if not URLValidator().is_valid_url(long_url):
            raise ValueError("Invalid URL format")

        with self.lock:
            hash_object = hashlib.sha256(long_url.encode())
            short_url = hash_object.hexdigest()[:6]
            if short_url in self.url_map:
                # Collision detected, generate a new shortened URL
                short_url = self._generate_new_short_url(long_url)
            self.url_map[short_url] = long_url
            return short_url

    def get_original_url(self, short_url):
        with self.lock:
            return self.url_map.get(short_url)

    def _generate_new_short_url(self, long_url):
        # Generate a new shortened URL by appending a counter
        counter = 0
        while True:
            new_short_url = f"{hashlib.sha256(long_url.encode()).hexdigest()[:6]}{counter}"
            if new_short_url not in self.url_map:
                return new_short_url
            counter += 1

class URLShortenerLibrary:
    def __init__(self):
        self.url_shortener = URLShortener()

    def shorten_url(self, long_url):
        return self.url_shortener.shorten_url(long_url)

    def get_original_url(self, short_url):
        return self.url_shortener.get_original_url(short_url)

    def is_valid_url(self, url):
        return URLValidator().is_valid_url(url)

# Example usage:
```
