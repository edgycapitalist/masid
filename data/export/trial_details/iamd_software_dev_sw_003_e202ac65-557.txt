## Introduction
The goal of this project is to design and implement a Python data pipeline that can read CSV files, apply a chain of transformations, and output the results to CSV or JSON. The pipeline should also support a plugin system for custom transformations and provide clear error reporting for malformed data.

## System Components
The data pipeline will consist of the following components:

1. **Data Reader**: Responsible for reading CSV files with configurable schema validation.
2. **Transformation Chain**: Applies a series of transformations (filtering, mapping, aggregation) to the data.
3. **Plugin System**: Allows for custom transformations to be added to the pipeline.
4. **Data Writer**: Outputs the transformed data to CSV or JSON.
5. **Error Reporter**: Provides clear error reporting for malformed data.

## System Flow
The data pipeline will operate as follows:

1. The **Data Reader** reads a CSV file and validates its schema.
2. The **Transformation Chain** applies a series of transformations to the data.
3. The **Plugin System** allows for custom transformations to be added to the pipeline.
4. The **Data Writer** outputs the transformed data to CSV or JSON.
5. The **Error Reporter** reports any errors that occur during the pipeline execution.

## Data Reader
The **Data Reader** will use the `pandas` library to read CSV files. It will also use the `schema` library to validate the schema of the CSV file.

## Transformation Chain
The **Transformation Chain** will consist of a series of transformations that can be applied to the data. These transformations will include:

* **Filtering**: Removing rows from the data that do not meet certain conditions.
* **Mapping**: Applying a function to each row of the data.
* **Aggregation**: Combining multiple rows of the data into a single row.

## Plugin System
The **Plugin System** will allow for custom transformations to be added to the pipeline. This will be achieved through the use of a plugin interface that defines the methods that a transformation must implement.

## Data Writer
The **Data Writer** will use the `pandas` library to output the transformed data to CSV or JSON.

## Error Reporter
The **Error Reporter** will use the `logging` library to report any errors that occur during the pipeline execution.

import pandas as pd
from schema import Schema
import logging
import json

# Define the plugin interface
class TransformationPlugin:
    def apply(self, data):
        raise NotImplementedError

# Define the filtering transformation
class FilteringTransformation(TransformationPlugin):
    def __init__(self, condition):
        self.condition = condition

    def apply(self, data):
        return data[self.condition(data)]

# Define the mapping transformation
class MappingTransformation(TransformationPlugin):
    def __init__(self, function):
        self.function = function

    def apply(self, data):
        return data.apply(self.function, axis=1)

# Define the aggregation transformation
class AggregationTransformation(TransformationPlugin):
    def __init__(self, function):
        self.function = function

    def apply(self, data):
        return data.agg(self.function)

# Define the data reader
class DataReader:
    def __init__(self, schema):
        self.schema = schema

    def read(self, file_path):
        data = pd.read_csv(file_path)
        if not self.schema.validate(data):
            raise ValueError("Invalid schema")
        return data

# Define the data writer
class DataWriter:
    def write(self, data, file_path, format):
        if format == "csv":
            data.to_csv(file_path, index=False)
        elif format == "json":
            data.to_json(file_path, orient="records")
        else:
            raise ValueError("Invalid format")

# Define the error reporter
class ErrorReporter:
    def __init__(self):
        self.logger = logging.getLogger(__name__)

    def report(self, error):
        self.logger.error(error)

# Define the pipeline
class Pipeline:
    def __init__(self, data_reader, data_writer, error_reporter):
        self.data_reader = data_reader
        self.data_writer = data_writer
        self.error_reporter = error_reporter
        self.transformations = []

    def add_transformation(self, transformation):
        self.transformations.append(transformation)

    def run(self, file_path, output_file_path, format):
        try:
            data = self.data_reader.read(file_path)
            for transformation in self.transformations:
                data = transformation.apply(data)
            self.data_writer.write(data, output_file_path, format)
        except Exception as e:
            self.error_reporter.report(e)

# Usage example
if __name__ == "__main__":
    # Define the schema
    schema = Schema({
        "name": str,
        "age": int
    })

    # Create the data reader
    data_reader = DataReader(schema)

    # Create the data writer
    data_writer = DataWriter()

    # Create the error reporter
    error_reporter = ErrorReporter()

    # Create the pipeline
    pipeline = Pipeline(data_reader, data_writer, error_reporter)

    # Add transformations to the pipeline
    pipeline.add_transformation(FilteringTransformation(lambda x: x["age"] > 18))
    pipeline.add_transformation(MappingTransformation(lambda x: x["name"].upper()))
    pipeline.add_transformation(AggregationTransformation("mean"))

    # Run the pipeline
    pipeline.run("input.csv", "output.csv", "csv")

class TransformationChain:
    def __init__(self):
