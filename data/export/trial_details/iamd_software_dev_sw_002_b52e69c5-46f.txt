## Introduction

2. **TokenBucket**: The class implementing the token bucket algorithm.
3. **RateLimiterDecorator**: A decorator for easy application of rate limiters to functions.

	* `create_limiter(name, rate, burst_size)`: Creates a new rate limiter with the given name, rate, and burst size.
	* `get_limiter(name)`: Retrieves a rate limiter by name.
	* `limit(name, func, *args, **kwargs)`: Applies the rate limiter to a function.
2. **TokenBucket**:
	* `__init__(rate, burst_size)`: Initializes the token bucket with the given rate and burst size.
	* `allow()`: Returns `True` if a request is allowed, `False` otherwise.
3. **RateLimiterDecorator**:
	* `__init__(name)`: Initializes the decorator with the given rate limiter name.
	* `__call__(func)`: Applies the rate limiter to the function.

## Data Flow

1. The `RateLimiter` class maintains a dictionary of rate limiters, where each rate limiter is associated with a name.
2. The `TokenBucket` class implements the token bucket algorithm, which manages the available tokens and the rate at which they are replenished.
3. When a request is made, the `RateLimiter` class checks if the request is allowed by calling the `allow()` method of the corresponding `TokenBucket` instance.
4. If the request is allowed, the `TokenBucket` instance decrements the available tokens and the `RateLimiter` class calls the decorated function.

import threading
import time
from functools import wraps

class TokenBucket:
    def __init__(self, rate, burst_size):
        self.rate = rate
        self.burst_size = burst_size
        self.available_tokens = burst_size
        self.last_update = time.time()

    def allow(self):
        now = time.time()
        elapsed_time = now - self.last_update
        self.available_tokens = min(self.burst_size, self.available_tokens + elapsed_time * self.rate)
        self.last_update = now

        if self.available_tokens >= 1:
            self.available_tokens -= 1
            return True
        return False

class RateLimiter:
    def __init__(self):
        self.limiters = {}
        self.lock = threading.Lock()

    def create_limiter(self, name, rate, burst_size):
        with self.lock:
            if name in self.limiters:
                raise ValueError("Rate limiter with name '{}' already exists".format(name))
            self.limiters[name] = TokenBucket(rate, burst_size)

    def get_limiter(self, name):
        with self.lock:
            return self.limiters.get(name)

    def limit(self, name, func, *args, **kwargs):
        limiter = self.get_limiter(name)
        if limiter is None:
            raise ValueError("Rate limiter with name '{}' does not exist".format(name))

        if not limiter.allow():
            raise Exception("Rate limit exceeded")

        return func(*args, **kwargs)

    def decorator(self, name):
        def decorator(func):
            @wraps(func)
            def wrapper(*args, **kwargs):
                return self.limit(name, func, *args, **kwargs)
            return wrapper
        return decorator

# Example usage
if __name__ == "__main__":

    @rate_limiter.decorator("test")
