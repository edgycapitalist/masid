### Overview

The data pipeline is designed to read CSV files, apply a chain of transformations, and output the results to CSV or JSON. The pipeline will have a plugin system for custom transformations and provide clear error reporting for malformed data.

### Components

1. **Data Reader**
	* Responsible for reading CSV files with configurable schema validation.
	* Will use the `csv` and `pandas` libraries for reading and validating CSV files.
2. **Transformation Chain**
	* Applies a series of transformations to the data, including filtering, mapping, and aggregation.
	* Will use a plugin system to allow for custom transformations.
3. **Plugin System**
	* Allows for custom transformations to be added to the pipeline.
	* Will use a base class `Transformation` that custom plugins can inherit from.
4. **Data Writer**
	* Outputs the transformed data to CSV or JSON.
	* Will use the `csv` and `json` libraries for writing to CSV and JSON files.
5. **Error Reporter**
	* Provides clear error reporting for malformed data.
	* Will use a custom `ErrorReporter` class to handle error reporting.

### Data Flow

1. The `Data Reader` reads a CSV file and validates its schema.
2. The validated data is passed to the `Transformation Chain`.
3. The `Transformation Chain` applies a series of transformations to the data.
4. The transformed data is passed to the `Data Writer`.
5. The `Data Writer` outputs the data to CSV or JSON.
6. If any errors occur during the pipeline, the `Error Reporter` will report the errors.

### Interfaces

* `Data Reader`:
	+ `read_csv(file_path: str, schema: dict) -> pd.DataFrame`
* `Transformation Chain`:
	+ `apply_transformations(data: pd.DataFrame, transformations: list) -> pd.DataFrame`
* `Plugin System`:
	+ `Transformation`:
		- `apply(data: pd.DataFrame) -> pd.DataFrame`
* `Data Writer`:
	+ `write_csv(data: pd.DataFrame, file_path: str) -> None`
	+ `write_json(data: pd.DataFrame, file_path: str) -> None`
* `Error Reporter`:
	+ `report_error(error: Exception) -> None`

import csv
import json
import pandas as pd

class DataReader:
    def read_csv(self, file_path: str, schema: dict) -> pd.DataFrame:
        # Read CSV file and validate schema
        data = pd.read_csv(file_path)
        if not self._validate_schema(data, schema):
            raise ValueError("Invalid schema")
        return data

    def _validate_schema(self, data: pd.DataFrame, schema: dict) -> bool:
        # Validate schema
        for column, type_ in schema.items():
            if data[column].dtype != type_:
                return False
        return True

class Transformation:
    def apply(self, data: pd.DataFrame) -> pd.DataFrame:
        # Apply transformation
        raise NotImplementedError

class FilterTransformation(Transformation):
    def __init__(self, column: str, value: str):
        self.column = column
        self.value = value

    def apply(self, data: pd.DataFrame) -> pd.DataFrame:
        # Apply filter transformation
        return data[data[self.column] == self.value]

class MapTransformation(Transformation):
    def __init__(self, column: str, mapping: dict):
        self.column = column
        self.mapping = mapping

    def apply(self, data: pd.DataFrame) -> pd.DataFrame:
        # Apply map transformation
        data[self.column] = data[self.column].map(self.mapping)
        return data

class AggregationTransformation(Transformation):
    def __init__(self, column: str, aggregation: str):
        self.column = column
        self.aggregation = aggregation

    def apply(self, data: pd.DataFrame) -> pd.DataFrame:
        # Apply aggregation transformation
        if self.aggregation == "mean":
            return data.groupby(self.column).mean()
        elif self.aggregation == "sum":
            return data.groupby(self.column).sum()
        else:
            raise ValueError("Invalid aggregation")

class PluginSystem:
    def __init__(self):
        self.transformations = []

    def add_transformation(self, transformation: Transformation):
        self.transformations.append(transformation)

    def apply_transformations(self, data: pd.DataFrame) -> pd.DataFrame:
        for transformation in self.transformations:
            data = transformation.apply(data)
        return data

class DataWriter:
    def write_csv(self, data: pd.DataFrame, file_path: str) -> None:
        # Write data to CSV file
        data.to_csv(file_path, index=False)

    def write_json(self, data: pd.DataFrame, file_path: str) -> None:
        # Write data to JSON file
        data.to_json(file_path, orient="records")

class ErrorReporter:
    def report_error(self, error: Exception) -> None:
        # Report error
        print(f"Error: {error}")

	* Will use a base class `Transformation` that custom plugins can inherit from.

	* Will use the `pandas` library for writing to files.

import pandas as pd

class DataReader:
    def read_file(self, file_path: str, file_type: str) -> pd.DataFrame:
        # Read file and validate schema
        if file_type == "csv":
            return pd.read_csv(file_path)
        elif file_type == "json":
            return pd.read_json(file_path)
        else:

    def _validate_schema(self, data: pd.DataFrame, schema: dict) -> bool:
        # Validate schema
        for column, type_ in schema.items():
            if column not in data.columns:
                return False
            if type_ == "int" and not pd.api.types.is_integer_dtype(data[column]):
                return False
            if type_ == "str" and not pd.api.types.is_string_dtype(data[column]):
                return False
        return True

class Transformation:
    def apply(self, data: pd.DataFrame) -> pd.DataFrame:
        # Apply transformation
        raise NotImplementedError

    def __init__(self, column: str, value: str):
        self.column = column
        self.value = value

    def apply(self, data: pd.DataFrame) -> pd.DataFrame:

        return data[data[self.column] == self.value]

    def __init__(self, column: str, mapping: dict):
        self.column = column
        self.mapping = mapping

    def apply(self, data: pd.DataFrame) -> pd.DataFrame:
        # Apply map transformation
        data[self.column] = data[self.column].map(self.mapping)
        return data

    def __init__(self, column: str, aggregation: str):
        self.column = column
        self.aggregation = aggregation

    def apply(self, data: pd.DataFrame) -> pd.DataFrame:
        # Apply aggregation transformation
        if self.aggregation == "mean":
            return data.groupby(self.column).mean()
        elif self.aggregation == "sum":
            return data.groupby(self.column).sum()
        else:
