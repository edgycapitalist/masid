**URL Shortening Library Architecture Document**

## Introduction
The URL shortening library is designed to provide a lightweight and efficient solution for shortening URLs using a hash-based approach. The library will support reverse lookup, collision detection, input validation, and thread safety for concurrent usage.

## System Components

* **URL Shortener**: Responsible for shortening long URLs using a hash-based approach.
* **URL Mapper**: Maintains a mapping of short URLs to original URLs.
* **Collision Detector**: Detects and handles collisions when two different long URLs produce the same short URL.
* **Input Validator**: Validates the input URL format to ensure it conforms to standard URL syntax.

## Design

### Class Diagram

* `URLShortener`: Encapsulates the hash-based shortening logic.
	+ `shorten(url: str) -> str`: Returns a shortened version of the input URL.
* `URLMapper`: Maintains a mapping of short URLs to original URLs.
	+ `add_mapping(short_url: str, original_url: str) -> None`: Adds a new mapping to the database.
	+ `get_original_url(short_url: str) -> str`: Returns the original URL associated with the short URL.
* `CollisionDetector`: Detects and handles collisions.
	+ `detect_collision(short_url: str) -> bool`: Returns `True` if a collision is detected, `False` otherwise.
* `InputValidator`: Validates the input URL format.
	+ `validate_url(url: str) -> bool`: Returns `True` if the input URL is valid, `False` otherwise.

### Data Flow

1. The `URLShortener` receives a long URL as input and generates a shortened version using a hash-based approach.
2. The `URLMapper` adds a new mapping between the short URL and the original URL to the database.
3. The `CollisionDetector` checks for collisions by verifying if the short URL already exists in the database.
4. If a collision is detected, the `CollisionDetector` generates a new short URL and updates the mapping.
5. The `InputValidator` validates the input URL format to ensure it conforms to standard URL syntax.

### Technology Choices

* **Hashing Algorithm**: SHA-256 (Secure Hash Algorithm 256) will be used to generate the shortened URL.
* **Database**: A simple in-memory dictionary will be used to store the mappings between short URLs and original URLs.
* **Thread Safety**: The library will use a lock mechanism to ensure thread safety for concurrent usage.

## Implementation

```python
import hashlib
import threading
from urllib.parse import urlparse

class URLShortener:
    def __init__(self):
        self.mapper = URLMapper()
        self.collision_detector = CollisionDetector()
        self.input_validator = InputValidator()

    def shorten(self, url: str) -> str:
        if not self.input_validator.validate_url(url):
            raise ValueError("Invalid URL format")

        short_url = self.generate_short_url(url)
        if self.collision_detector.detect_collision(short_url):
            short_url = self.generate_new_short_url(url)

        self.mapper.add_mapping(short_url, url)
        return short_url

    def generate_short_url(self, url: str) -> str:
        # Use SHA-256 to generate a shortened version of the URL
        hash_object = hashlib.sha256(url.encode())
        return hash_object.hexdigest()[:6]

    def generate_new_short_url(self, url: str) -> str:
        # Generate a new short URL by appending a counter
        counter = 0
        while True:
            new_short_url = self.generate_short_url(url) + str(counter)
            if not self.collision_detector.detect_collision(new_short_url):
                return new_short_url
            counter += 1

class URLMapper:
    def __init__(self):
        self.mappings = {}
        self.lock = threading.Lock()

    def add_mapping(self, short_url: str, original_url: str) -> None:
        with self.lock:
            self.mappings[short_url] = original_url

    def get_original_url(self, short_url: str) -> str:
        with self.lock:
            return self.mappings.get(short_url)

class CollisionDetector:
    def __init__(self):
        self.mapper = URLMapper()

    def detect_collision(self, short_url: str) -> bool:
        return short_url in self.mapper.mappings

class InputValidator:
    def validate_url(self, url: str) -> bool:
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

# Example usage:
url_shortener = URLShortener()
short_url = url_shortener.shorten("https://www.example.com")
print(short_url)

original_url = URLMapper().get_original_url(short_url)
print(original_url)
```

## Introduction
The URL shortening library is designed to provide a lightweight and efficient solution for shortening URLs using a hash-based approach. The library will support reverse lookup, collision detection, input validation, and thread safety for concurrent usage.

## System Components

* **URL Shortener**: Responsible for shortening long URLs using a hash-based approach.
* **URL Mapper**: Maintains a mapping of short URLs to original URLs.
* **Collision Detector**: Detects and handles collisions when two different long URLs produce the same short URL.
* **Input Validator**: Validates the input URL format to ensure it conforms to standard URL syntax.

## Design

### Class Diagram

* `URLShortener`: Encapsulates the hash-based shortening logic.
	+ `shorten(url: str) -> str`: Returns a shortened version of the input URL.
* `URLMapper`: Maintains a mapping of short URLs to original URLs.
	+ `add_mapping(short_url: str, original_url: str) -> None`: Adds a new mapping to the database.
	+ `get_original_url(short_url: str) -> str`: Returns the original URL associated with the short URL.
* `CollisionDetector`: Detects and handles collisions.
	+ `detect_collision(short_url: str) -> bool`: Returns `True` if a collision is detected, `False` otherwise.
* `InputValidator`: Validates the input URL format.
	+ `validate_url(url: str) -> bool`: Returns `True` if the input URL is valid, `False` otherwise.

### Data Flow

1. The `URLShortener` receives a long URL as input and generates a shortened version using a hash-based approach.
2. The `URLMapper` adds a new mapping between the short URL and the original URL to the database.
3. The `CollisionDetector` checks for collisions by verifying if the short URL already exists in the database.
4. If a collision is detected, the `CollisionDetector` generates a new short URL and updates the mapping.
5. The `InputValidator` validates the input URL format to ensure it conforms to standard URL syntax.

### Technology Choices

* **Hashing Algorithm**: FNV-1a (Fowler-Noll-Vo 1a) will be used to generate the shortened URL, which is a faster and more efficient algorithm than SHA-256 for this purpose.
* **Database**: A trie data structure will be used to store the mappings between short URLs and original URLs, which provides faster lookup and insertion times.
* **Thread Safety**: The library will use a lock mechanism to ensure thread safety for concurrent usage.

```python
import threading
from urllib.parse import urlparse

class URLShortener:
    def __init__(self):
        self.mapper = URLMapper()
        self.collision_detector = CollisionDetector()
        self.input_validator = InputValidator()

    def shorten(self, url: str) -> str:
        if not self.input_validator.validate_url(url):
            raise ValueError("Invalid URL format")

        short_url = self.generate_short_url(url)
        if self.collision_detector.detect_collision(short_url):
            short_url = self.generate_new_short_url(url)

        self.mapper.add_mapping(short_url, url)
        return short_url

    def generate_short_url(self, url: str) -> str:
        # Use FNV-1a to generate a shortened version of the URL
        hash_object = fnv_1a.fnv_1a_hash(url.encode())
        return hash_object[:6]

    def generate_new_short_url(self, url: str) -> str:
        # Generate a new short URL by appending a counter
        counter = 0
        while True:
            new_short_url = self.generate_short_url(url) + str(counter)
            if not self.collision_detector.detect_collision(new_short_url):
                return new_short_url
            counter += 1

class URLMapper:
    def __init__(self):
        self.mappings = {}
        self.lock = threading.Lock()

    def add_mapping(self, short_url: str, original_url: str) -> None:
        with self.lock:
            self.mappings[short_url] = original_url

    def get_original_url(self, short_url: str) -> str:
        with self.lock:
            return self.mappings.get(short_url)

    def detect_collision(self, short_url: str) -> bool:
        return short_url in self.mapper.mappings

class InputValidator:
    def validate_url(self, url: str) -> bool:
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False
```

## Introduction
The URL shortening library is designed to provide a lightweight and efficient solution for shortening URLs using a hash-based approach. The library will support reverse lookup, collision detection, input validation, and thread safety for concurrent usage.

## System Components

* **URL Shortener**: Responsible for shortening long URLs using a hash-based approach.
* **URL Mapper**: Maintains a mapping of short URLs to original URLs.
* **Collision Detector**: Detects and handles collisions when two different long URLs produce the same short URL.
* **Input Validator**: Validates the input URL format to ensure it conforms to standard URL syntax.

## Design

### Class Diagram

* `URLShortener`: Encapsulates the hash-based shortening logic.
	+ `shorten(url: str) -> str`: Returns a shortened version of the input URL.
* `URLMapper`: Maintains a mapping of short URLs to original URLs.
	+ `add_mapping(short_url: str, original_url: str) -> None`: Adds a new mapping to the database.
	+ `get_original_url(short_url: str) -> str`: Returns the original URL associated with the short URL.
* `CollisionDetector`: Detects and handles collisions.
	+ `detect_collision(short_url: str) -> bool`: Returns `True` if a collision is detected, `False` otherwise.
* `InputValidator`: Validates the input URL format.
	+ `validate_url(url: str) -> bool`: Returns `True` if the input URL is valid, `False` otherwise.

### Data Flow

1. The `URLShortener` receives a long URL as input and generates a shortened version using a hash-based approach.
2. The `URLMapper` adds a new mapping between the short URL and the original URL to the database.
3. The `CollisionDetector` checks for collisions by verifying if the short URL already exists in the database.
4. If a collision is detected, the `CollisionDetector` generates a new short URL and updates the mapping.
5. The `InputValidator` validates the input URL format to ensure it conforms to standard URL syntax.

### Technology Choices

* **Hashing Algorithm**: FNV-1a (Fowler-Noll-Vo 1a) will be used to generate the shortened URL, which is a faster and more efficient algorithm than SHA-256 for this purpose.
* **Database**: A trie data structure will be used to store the mappings between short URLs and original URLs, which provides faster lookup and insertion times.
* **Thread Safety**: The library will use a lock mechanism to ensure thread safety for concurrent usage.

```python
import fnv_1a
import threading
from urllib.parse import urlparse

class URLShortener:
    def __init__(self):
        self.mapper = URLMapper()
        self.collision_detector = CollisionDetector()
        self.input_validator = InputValidator()

    def shorten(self, url: str) -> str:
        if not self.input_validator.validate_url(url):
            raise ValueError("Invalid URL format")

        short_url = self.generate_short_url(url)
        if self.collision_detector.detect_collision(short_url):
            short_url = self.generate_new_short_url(url)

        self.mapper.add_mapping(short_url, url)
        return short_url

    def generate_short_url(self, url: str) -> str:
        # Use FNV-1a to generate a shortened version of the URL
        hash_object = fnv_1a.fnv_1a_hash(url.encode())
        return hash_object[:6]

    def generate_new_short_url(self, url: str) -> str:
        # Generate a new short URL by appending a counter
        counter = 0
        while True:
            new_short_url = self.generate_short_url(url) + str(counter)
            if not self.collision_detector.detect_collision(new_short_url):
                return new_short_url
            counter += 1

class URLMapper:
    def __init__(self):
        self.mappings = {}
        self.lock = threading.Lock()

    def add_mapping(self, short_url: str, original_url: str) -> None:
        with self.lock:
            self.mappings[short_url] = original_url

    def get_original_url(self, short_url: str) -> str:
        with self.lock:
            return self.mappings.get(short_url)

class CollisionDetector:
    def __init__(self):
        self.mapper = URLMapper()

    def detect_collision(self, short_url: str) -> bool:
        return short_url in self.mapper.mappings

class InputValidator:
    def validate_url(self, url: str) -> bool:
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

def main():

    short_url = url_shortener.shorten("https://www.example.com")
    print(short_url)

    original_url = URLMapper().get_original_url(short_url)
```
