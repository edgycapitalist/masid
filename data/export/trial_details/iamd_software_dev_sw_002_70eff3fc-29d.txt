### Overview

The rate limiter module is designed to provide a flexible and thread-safe way to limit the rate at which requests are processed. The module supports the token bucket algorithm, configurable rate and burst size, multiple named rate limiters, and a decorator for easy application to functions.

### Module Structure

The rate limiter module consists of the following components:

*   `RateLimiter`: The main class that implements the token bucket algorithm and provides methods for checking and consuming tokens.
*   `RateLimiterDecorator`: A decorator that applies a rate limiter to a function.
*   `RateLimiterManager`: A class that manages multiple named rate limiters.

### Token Bucket Algorithm

The token bucket algorithm is a widely used rate limiting algorithm that works as follows:

*   Tokens are added to a bucket at a constant rate (requests per second).
*   Each request consumes one token from the bucket.
*   If the bucket is empty, the request is blocked until a token is available.

### Configurable Rate and Burst Size

The rate limiter module allows for configurable rate (requests per second) and burst size (maximum number of tokens in the bucket).

### Multiple Named Rate Limiters

The rate limiter module supports multiple named rate limiters, which can be used to limit the rate of requests for different users or endpoints.

### Thread-Safe Operation

The rate limiter module is designed to be thread-safe, using locks to synchronize access to the token bucket.

            return True

    def check(self) -> bool:
        """
        Check if a token is available in the bucket.

        :return: True if a token is available, False otherwise.
        """
        with self.lock:
            current_time = time.time()
            elapsed_time = current_time - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
            self.last_update = current_time
            return self.tokens >= 1

    def check_and_consume(self) -> bool:
        """
        Check if a token is available in the bucket and consume it if available.

        :return: True if the token was consumed successfully, False otherwise.
        """
        with self.lock:
            current_time = time.time()
            elapsed_time = current_time - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
            self.last_update = current_time
            if self.tokens < 1:
                return False
            self.tokens -= 1
            return True

    def consume_all(self) -> int:
        """
        Consume all available tokens in the bucket.

        :return: The number of tokens consumed.
        """
        with self.lock:
            current_time = time.time()
            elapsed_time = current_time - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
            self.last_update = current_time
            tokens_consumed = int(self.tokens)
            self.tokens = 0
            return tokens_consumed

class RateLimiterDecorator:
    def __init__(self, rate_limiter: RateLimiter, error_message: str = "Rate limit exceeded"):
        """
        Initialize a rate limiter decorator with the given rate limiter and error message.

        :param rate_limiter: The rate limiter to apply to the function.
        :param error_message: The error message to raise when the rate limit is exceeded.
        """
        self.rate_limiter = rate_limiter
        self.error_message = error_message

    def __call__(self, func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            if not self.rate_limiter.check_and_consume():
                raise Exception(self.error_message)
            return func(*args, **kwargs)
        return wrapper

class RateLimiterManager:
    def __init__(self):
        """
        Initialize a rate limiter manager.
        """
        self.rate_limiters = {}

    def get_rate_limiter(self, name: str, rate: int, burst_size: int) -> RateLimiter:
        """
        Get a rate limiter with the given name, rate, and burst size.

        :param name: The name of the rate limiter.
        :param rate: The rate at which tokens are added to the bucket (requests per second).
        :param burst_size: The maximum number of tokens in the bucket.
        :return: The rate limiter with the given name, or a new rate limiter if it does not exist.
        :raises ValueError: If the rate or burst size is less than or equal to zero.
        :raises TypeError: If the rate or burst size is not an integer.
        """
        if not isinstance(rate, int) or not isinstance(burst_size, int):
            raise TypeError("Rate and burst size must be integers")
        if rate <= 0 or burst_size <= 0:
            raise ValueError("Rate and burst size must be greater than zero")
        if name not in self.rate_limiters:
            self.rate_limiters[name] = RateLimiter(rate, burst_size)
        return self.rate_limiters[name]

    def add_rate_limiter(self, name: str, rate: int, burst_size: int):
        """
        Add a new rate limiter with the given name, rate, and burst size.

        :param name: The name of the rate limiter.
        :param rate: The rate at which tokens are added to the bucket (requests per second).
        :param burst_size: The maximum number of tokens in the bucket.
        """
        self.rate_limiters[name] = RateLimiter(rate, burst_size)

    def remove_rate_limiter(self, name: str):
        """
        Remove the rate limiter with the given name.

        :param name: The name of the rate limiter.
        """
        if name in self.rate_limiters:
            del self.rate_limiters[name]
