**Architecture Document: In-Memory Task Queue**

## Introduction
The in-memory task queue is designed to manage tasks with priorities, track task status, and support task retry with configurable max retries. The system will utilize a producer-consumer model, where tasks are added to the queue and consumed by worker threads.

## High-Level Design
The in-memory task queue will consist of the following components:

*   **TaskQueue**: The main interface for adding tasks and tracking task status.
*   **Task**: Represents a task with a unique ID, priority, and retry count.
*   **TaskStatus**: Enumerates the possible task statuses (pending, running, completed, failed).
*   **WorkerThread**: Consumes tasks from the queue and executes them.

## Module Decomposition
The system will be decomposed into the following modules:

*   **task_queue.py**: Implements the TaskQueue class.
*   **task.py**: Defines the Task class and TaskStatus enum.
*   **worker.py**: Implements the WorkerThread class.
*   **config.py**: Stores configuration settings, such as max retries and queue size.

## Interfaces
The following interfaces will be used:

*   **TaskQueue.add_task(task: Task)**: Adds a task to the queue.
*   **TaskQueue.get_task()**: Returns a task from the queue.
*   **TaskQueue.update_task_status(task_id: int, status: TaskStatus)**: Updates the status of a task.
*   **WorkerThread.start()**: Starts the worker thread.
*   **WorkerThread.stop()**: Stops the worker thread.

## Data Flow
The data flow between components is as follows:

1.  The TaskQueue receives tasks from external producers.
2.  The TaskQueue stores tasks in a priority queue.
3.  WorkerThreads consume tasks from the TaskQueue.
4.  WorkerThreads execute tasks and update task status.
5.  The TaskQueue tracks task status and provides task status updates to external consumers.

The system will utilize the following technologies:

*   **Python 3.x**: As the programming language.

*   **queue**: For implementing the in-memory task queue.

```python
import threading
import time
from queue import PriorityQueue
from typing import Callable, List
from enum import Enum

# Importing Task and TaskStatus from task.py
from task import Task, TaskStatus

class TaskQueue:
    def __init__(self, max_size: int = 100):
        """
        Initializes the TaskQueue with a maximum size.

        Args:
        max_size (int): The maximum number of tasks that can be stored in the queue.
        """
        self.max_size = max_size
        self.queue = PriorityQueue()
        self.lock = threading.Lock()
        self.tasks: List[Task] = []

    def add_task(self, task: Task):
        """
        Adds a task to the queue.

        Args:
        task (Task): The task to be added to the queue.
        """
        with self.lock:
            if len(self.tasks) < self.max_size:
                self.queue.put((task.priority, task))
                self.tasks.append(task)

    def get_task(self):
        """
        Retrieves a task from the queue.

        Returns:
        Task: The task retrieved from the queue, or None if the queue is empty.
        """
        with self.lock:
            if not self.queue.empty():
                _, task = self.queue.get()
                return task
            return None

    def update_task_status(self, task_id: int, status: TaskStatus):
        """
        Updates the status of a task.

        Args:
        task_id (int): The ID of the task to be updated.
        status (TaskStatus): The new status of the task.
        """
        with self.lock:
            for task in self.tasks:
                if task.task_id == task_id:
                    task.status = status
                    break

class WorkerThread(threading.Thread):
    def __init__(self, task_queue: TaskQueue):
        """
        Initializes a worker thread with a task queue.

        Args:
        task_queue (TaskQueue): The task queue to be used by the worker thread.
        """
        super().__init__()
        self.task_queue = task_queue
        self.running = False

    def start(self):
        """
        Starts the worker thread.
        """
        self.running = True
        super().start()

    def run(self):
        """
        Runs the worker thread, consuming tasks from the queue and executing them.
        """
        while self.running:
            task = self.task_queue.get_task()
            if task:
                task.status = TaskStatus.RUNNING
                try:
                    task.execute()
                except Exception as e:
                    print(f"Task {task.task_id} failed with error: {e}")
                finally:
                    self.task_queue.update_task_status(task.task_id, task.status)

    def stop(self):
        """
        Stops the worker thread.
        """
        self.running = False

def main():
    # Create a task queue
    task_queue = TaskQueue()

    # Create tasks
    task1 = Task(1, 1, lambda: print("Task 1 executed"))
    task2 = Task(2, 2, lambda: print("Task 2 executed"))
    task3 = Task(3, 3, lambda: print("Task 3 executed"))

    # Add tasks to the queue
    task_queue.add_task(task1)
    task_queue.add_task(task2)
    task_queue.add_task(task3)

    # Create worker threads
    worker1 = WorkerThread(task_queue)
    worker2 = WorkerThread(task_queue)

    # Start worker threads
    worker1.start()
    worker2.start()

    # Wait for 5 seconds
    time.sleep(5)

    # Stop worker threads
    worker1.stop()
    worker2.stop()

if __name__ == "__main__":
    main()
```

```python
import threading
import time
from queue import PriorityQueue
from typing import Callable, List
from enum import Enum

class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id: int, priority: int, func: Callable, max_retries: int = 3):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.max_retries = max_retries
        self.retries = 0
        self.status = TaskStatus.PENDING

    def execute(self):
        try:
            self.func()
            self.status = TaskStatus.COMPLETED
        except Exception as e:
            self.retries += 1
            if self.retries < self.max_retries:
                self.status = TaskStatus.PENDING
            else:
                self.status = TaskStatus.FAILED
            raise e

class TaskQueue:
    def __init__(self, max_size: int = 100):
        self.max_size = max_size
        self.queue = PriorityQueue()
        self.lock = threading.Lock()
        self.tasks: List[Task] = []

    def add_task(self, task: Task):
        with self.lock:
            if len(self.tasks) < self.max_size:
                self.queue.put((task.priority, task))
                self.tasks.append(task)

    def get_task(self):
        with self.lock:
            if not self.queue.empty():
                _, task = self.queue.get()
                return task
            return None

    def update_task_status(self, task_id: int, status: TaskStatus):
        with self.lock:
            for task in self.tasks:
                if task.task_id == task_id:
                    task.status = status
                    break

        super().__init__()

        self.running = False

            if task:
                task.status = TaskStatus.RUNNING
                try:

                except Exception as e:
                    print(f"Task {task.task_id} failed with error: {e}")
                finally:
```

```python
import threading
import time
```
```python
from queue import PriorityQueue
```
```python
from typing import Callable, List
from enum import Enum

class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id: int, priority: int, func: Callable, max_retries: int = 3):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.max_retries = max_retries
        self.retries = 0
        self.status = TaskStatus.PENDING

    def execute(self):
        try:
            self.func()
            self.status = TaskStatus.COMPLETED
        except Exception as e:
            self.retries += 1
            if self.retries < self.max_retries:
                self.status = TaskStatus.PENDING
            else:
                self.status = TaskStatus.FAILED
            raise e

class TaskQueue:
    def __init__(self, max_size: int = 100):
        self.max_size = max_size
        self.queue = PriorityQueue()
        self.lock = threading.Lock()
        self.tasks: List[Task] = []

    def add_task(self, task: Task):
        with self.lock:
            if len(self.tasks) < self.max_size:
                self.queue.put((task.priority, task))
                self.tasks.append(task)

    def get_task(self):
        with self.lock:
            if not self.queue.empty():
                _, task = self.queue.get()
                return task
            return None

    def update_task_status(self, task_id: int, status: TaskStatus):
        with self.lock:
            for task in self.tasks:
                if task.task_id == task_id:
                    task.status = status
                    break

class WorkerThread(threading.Thread):
    def __init__(self, task_queue: TaskQueue):
        super().__init__()
        self.task_queue = task_queue
        self.running = False

    def start(self):
        self.running = True
        super().start()

    def run(self):
        while self.running:
            task = self.task_queue.get_task()
            if task:
                task.status = TaskStatus.RUNNING
                try:
                    task.execute()
                except Exception as e:
                    print(f"Task {task.task_id} failed with error: {e}")
                finally:
                    self.task_queue.update_task_status(task.task_id, task.status)

    def stop(self):
        self.running = False

def main():
    task_queue = TaskQueue()
    task1 = Task(1, 1, lambda: print("Task 1 executed"))
    task2 = Task(2, 2, lambda: print("Task 2 executed"))
    task_queue.add_task(task1)
    task_queue.add_task(task2)
    worker = WorkerThread(task_queue)
    worker.start()
    time.sleep(2)
    worker.stop()

if __name__ == "__main__":
    main()
```
