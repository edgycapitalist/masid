## Introduction

This document outlines the design and implementation of a Python data pipeline that reads CSV files with configurable schema validation, applies a chain of transformations, supports a plugin system for custom transformations, and outputs results to CSV or JSON.

## System Components

1. **Data Reader**: Responsible for reading CSV files with configurable schema validation.
2. **Transformation Pipeline**: Applies a chain of transformations (filtering, mapping, aggregation) to the input data.
3. **Plugin System**: Allows for custom transformations to be added to the pipeline.
4. **Data Writer**: Outputs the transformed data to CSV or JSON.

## System Flow

1. The **Data Reader** reads a CSV file and performs schema validation.
2. The validated data is passed to the **Transformation Pipeline**.
3. The **Transformation Pipeline** applies a chain of transformations to the data.
4. The transformed data is passed to the **Data Writer**.
5. The **Data Writer** outputs the data to CSV or JSON.

## Transformation Pipeline

The transformation pipeline consists of a series of transformations that are applied to the input data. Each transformation is a separate module that can be added or removed from the pipeline as needed.

### Transformation Types

1. **Filtering**: Removes rows from the input data based on a set of conditions.
2. **Mapping**: Applies a transformation to each row of the input data.
3. **Aggregation**: Combines multiple rows of the input data into a single row.

### Plugin System

The plugin system allows for custom transformations to be added to the pipeline. Each plugin must implement the `Transformation` interface and provide a `transform` method that takes in a row of data and returns the transformed row.

## Error Reporting

The system provides clear error reporting for malformed data. If a schema validation error occurs during data reading, an error message is generated and logged. If a transformation error occurs during pipeline processing, an error message is generated and logged.

import csv
import json
from typing import List, Dict

# Data Reader
class DataReader:
    def __init__(self, schema: Dict):
        self.schema = schema

    def read(self, file_path: str) -> List[Dict]:
        with open(file_path, 'r') as file:
            reader = csv.DictReader(file)
            data = []
            for row in reader:
                if self.validate(row):
                    data.append(row)
                else:
                    raise ValueError("Schema validation error")
            return data

    def validate(self, row: Dict) -> bool:
        for key, value in self.schema.items():
            if key not in row:
                return False
            if not isinstance(row[key], value):
                return False
        return True

# Transformation Pipeline
class TransformationPipeline:
    def __init__(self):
        self.transformations = []

    def add_transformation(self, transformation):
        self.transformations.append(transformation)

    def apply(self, data: List[Dict]) -> List[Dict]:
        transformed_data = data
        for transformation in self.transformations:
            transformed_data = transformation.transform(transformed_data)
        return transformed_data

# Plugin System
class Transformation:
    def transform(self, data: List[Dict]) -> List[Dict]:
        raise NotImplementedError

class FilterTransformation(Transformation):
    def __init__(self, condition):
        self.condition = condition

    def transform(self, data: List[Dict]) -> List[Dict]:
        return [row for row in data if self.condition(row)]

class MapTransformation(Transformation):
    def __init__(self, mapping):
        self.mapping = mapping

    def transform(self, data: List[Dict]) -> List[Dict]:
        return [{**row, **self.mapping(row)} for row in data]

class AggregateTransformation(Transformation):
    def __init__(self, aggregation):
        self.aggregation = aggregation

    def transform(self, data: List[Dict]) -> List[Dict]:
        aggregated_data = {}
        for row in data:
            key = self.aggregation(row)
            if key not in aggregated_data:
                aggregated_data[key] = []
            aggregated_data[key].append(row)
        return [self.aggregate_rows(rows) for rows in aggregated_data.values()]

    def aggregate_rows(self, rows: List[Dict]) -> Dict:
        raise NotImplementedError

# Data Writer
class DataWriter:
    def __init__(self, output_format: str):
        self.output_format = output_format

    def write(self, data: List[Dict], file_path: str):
        if self.output_format == 'csv':
            with open(file_path, 'w') as file:
                writer = csv.DictWriter(file, fieldnames=data[0].keys())
                writer.writeheader()
                writer.writerows(data)
        elif self.output_format == 'json':
            with open(file_path, 'w') as file:
                json.dump(data, file)

# Example usage
if __name__ == '__main__':
    schema = {'name': str, 'age': int}
    data_reader = DataReader(schema)
    data = data_reader.read('input.csv')

    transformation_pipeline = TransformationPipeline()
    transformation_pipeline.add_transformation(FilterTransformation(lambda row: row['age'] > 18))
    transformation_pipeline.add_transformation(MapTransformation(lambda row: {'name': row['name'].upper()}))
    transformation_pipeline.add_transformation(AggregateTransformation(lambda row: row['name']))

    transformed_data = transformation_pipeline.apply(data)

    data_writer = DataWriter('csv')
    data_writer.write(transformed_data, 'output.csv')

1. **Data Reader**: Responsible for reading CSV files with configurable schema validation.
2. **Transformation Pipeline**: Applies a chain of transformations (filtering, mapping, aggregation) to the input data.
3. **Plugin System**: Allows for custom transformations to be added to the pipeline.
4. **Data Writer**: Outputs the transformed data to CSV or JSON.
5. **Error Handler**: Handles errors that occur during data reading, transformation, and writing.
6. **Logger**: Provides logging functionality to track the progress of the data pipeline.

## System Flow

1. The **Data Reader** reads a CSV file and performs schema validation.
2. The validated data is passed to the **Transformation Pipeline**.
3. The **Transformation Pipeline** applies a chain of transformations to the data.
4. The transformed data is passed to the **Data Writer**.
5. The **Data Writer** outputs the data to CSV or JSON.
6. The **Error Handler** catches and handles any errors that occur during the process.
7. The **Logger** logs the progress of the data pipeline.

## Transformation Pipeline

The transformation pipeline consists of a series of transformations that are applied to the input data. Each transformation is a separate module that can be added or removed from the pipeline as needed.

### Transformation Types

1. **Filtering**: Removes rows from the input data based on a set of conditions.
2. **Mapping**: Applies a transformation to each row of the input data.
3. **Aggregation**: Combines multiple rows of the input data into a single row.

### Plugin System

The plugin system allows for custom transformations to be added to the pipeline. Each plugin must implement the `Transformation` interface and provide a `transform` method that takes in a row of data and returns the transformed row.

## Error Handling

The system provides clear error reporting for malformed data. If a schema validation error occurs during data reading, an error message is generated and logged. If a transformation error occurs during pipeline processing, an error message is generated and logged. If a data writing error occurs, an error message is generated and logged.

import csv
import json
from typing import List, Dict
import logging

# Set up logging
logging.basicConfig(level=logging.INFO)

# Data Reader
class DataReader:
    def __init__(self, schema: Dict):
        self.schema = schema

    def read(self, file_path: str) -> List[Dict]:
        try:
            with open(file_path, 'r') as file:
                reader = csv.DictReader(file)
                data = []
                for row in reader:
                    if self.validate(row):
                        data.append(row)
                    else:
                        logging.error("Schema validation error")
                        raise ValueError("Schema validation error")
                return data
        except Exception as e:
            logging.error(f"Error reading file: {e}")
            raise

    def validate(self, row: Dict) -> bool:
        for key, value in self.schema.items():
            if key not in row:
                return False
            if not isinstance(row[key], value):
                return False
        return True

# Transformation Pipeline
class TransformationPipeline:
    def __init__(self):
        self.transformations = []

    def add_transformation(self, transformation):
        self.transformations.append(transformation)

    def apply(self, data: List[Dict]) -> List[Dict]:
        try:
            transformed_data = data
            for transformation in self.transformations:
                transformed_data = transformation.transform(transformed_data)
            return transformed_data
        except Exception as e:
            logging.error(f"Error applying transformations: {e}")
            raise

# Plugin System
class Transformation:
    def transform(self, data: List[Dict]) -> List[Dict]:
        raise NotImplementedError

class FilterTransformation(Transformation):
    def __init__(self, condition):
        self.condition = condition

    def transform(self, data: List[Dict]) -> List[Dict]:
        return [row for row in data if self.condition(row)]

class MapTransformation(Transformation):
    def __init__(self, mapping):
        self.mapping = mapping

    def transform(self, data: List[Dict]) -> List[Dict]:
        return [{**row, **self.mapping(row)} for row in data]

class AggregateTransformation(Transformation):
    def __init__(self, aggregation):
        self.aggregation = aggregation

    def transform(self, data: List[Dict]) -> List[Dict]:
        aggregated_data = {}
        for row in data:
            key = self.aggregation(row)
            if key not in aggregated_data:
                aggregated_data[key] = []
            aggregated_data[key].append(row)
        return [self.aggregate_rows(rows) for rows in aggregated_data.values()]

    def aggregate_rows(self, rows: List[Dict]) -> Dict:
        raise NotImplementedError

# Data Writer
class DataWriter:
    def __init__(self, output_format: str):
        self.output_format = output_format

    def write(self, data: List[Dict], file_path: str):
        try:
            if self.output_format == 'csv':
                with open(file_path, 'w') as file:
                    writer = csv.DictWriter(file, fieldnames=data[0].keys())
                    writer.writeheader()
                    writer.writerows(data)
            elif self.output_format == 'json':
                with open(file_path, 'w') as file:
                    json.dump(data, file)
        except Exception as e:
            logging.error(f"Error writing file: {e}")
            raise

# Logger
class Logger:
    def __init__(self):
        self.log_file = 'log.txt'

    def log(self, message: str):
        with open(self.log_file, 'a') as file:
            file.write(message + '\n')

    def log_error(self, error: str):
        with open(self.log_file, 'a') as file:
            file.write(f"ERROR: {error}\n")

# Example usage
if __name__ == '__main__':
    schema = {'name': str, 'age': int}
    data_reader = DataReader(schema)
    data = data_reader.read('input.csv')

    transformation_pipeline = TransformationPipeline()
    transformation_pipeline.add_transformation(FilterTransformation(lambda row: row['age'] > 18))
    transformation_pipeline.add_transformation(MapTransformation(lambda row: {'name': row['name'].upper()}))
    transformation_pipeline.add_transformation(AggregateTransformation(lambda row: row['name']))

    logger = Logger()
    logger.log("Starting data pipeline")
    try:
        transformed_data = transformation_pipeline.apply(data)
        logger.log("Data pipeline completed successfully")
    except Exception as e:
        logger.log_error(f"Error in data pipeline: {e}")

    data_writer = DataWriter('csv')
    data_writer.write(transformed_data, 'output.csv')

from solution import DataReader, TransformationPipeline, Transformation, FilterTransformation, MapTransformation, AggregateTransformation, DataWriter, Logger, ErrorHandler

def test_data_reader_with_valid_file():
    # Arrange
    schema = {'name': str, 'age': int}
    data_reader = DataReader(schema)
    file_path = 'valid_input.csv'

    # Act
    data = data_reader.read(file_path)

    # Assert
    assert isinstance(data, list)
    assert all(isinstance(row, dict) for row in data)

def test_data_reader_with_invalid_file():
    # Arrange
    schema = {'name': str, 'age': int}
    data_reader = DataReader(schema)
    file_path = 'non_existent_file.csv'

    # Act
    data = data_reader.read(file_path)

    # Assert
    assert data == []

def test_transformation_pipeline_with_valid_data():
    # Arrange
    transformation_pipeline = TransformationPipeline()
    transformation_pipeline.add_transformation(FilterTransformation(lambda row: row['age'] > 18))
    transformation_pipeline.add_transformation(MapTransformation(lambda row: {'name': row['name'].upper()}))
    data = [{'name': 'John', 'age': 25}, {'name': 'Jane', 'age': 30}]

    # Act
    transformed_data = transformation_pipeline.apply(data)

    # Assert
    assert isinstance(transformed_data, list)
    assert all(isinstance(row, dict) for row in transformed_data)

def test_filter_transformation_with_valid_condition():
    # Arrange

    data = [{'name': 'John', 'age': 25}, {'name': 'Jane', 'age': 30}]

    # Act
    transformed_data = filter_transformation.transform(data)

    # Assert

def test_data_writer_with_valid_output_format():
    # Arrange
    data_writer = DataWriter('csv')
    data = [{'name': 'John', 'age': 25}, {'name': 'Jane', 'age': 30}]
    file_path = 'output.csv'

    # Act
    data_writer.write(data, file_path)

    # Assert
    # Check if the file was written successfully
    with open(file_path, 'r') as file:
        assert file.read() != ''

def test_data_writer_with_invalid_output_format():
    # Arrange
    data_writer = DataWriter('invalid_format')
    data = [{'name': 'John', 'age': 25}, {'name': 'Jane', 'age': 30}]
    file_path = 'output.csv'

    # Act and Assert
