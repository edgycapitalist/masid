### Overview

*   `hash_generator`: A module responsible for generating a unique hash for each URL.
*   `collision_detector`: A module that detects and handles hash collisions.
*   `url_validator`: A module that validates the input URL format.
*   `thread_safe_storage`: A module that provides thread-safe storage for the URL mappings.

### Interfaces

The library will provide the following interfaces:

*   `shorten_url(long_url)`: Takes a long URL as input and returns a shortened version.
*   `lookup_short_url(short_url)`: Takes a short URL as input and returns the original URL.
*   `validate_url(url)`: Takes a URL as input and returns a boolean indicating whether the URL is valid.

### Data Flow

The data flow of the library is as follows:

1.  The user calls the `shorten_url` function with a long URL as input.
2.  The `url_validator` module checks if the input URL is valid. If not, it raises a `ValueError`.
3.  The `hash_generator` module generates a unique hash for the input URL.
4.  The `collision_detector` module checks if the generated hash already exists in the storage. If it does, it generates a new hash and repeats the check until a unique hash is found.
5.  The `thread_safe_storage` module stores the mapping between the short URL and the original URL.
6.  The `shorten_url` function returns the shortened URL to the user.
7.  The user calls the `lookup_short_url` function with a short URL as input.
8.  The `thread_safe_storage` module retrieves the original URL associated with the short URL and returns it to the user.

import hashlib
import threading
import sqlite3
from urllib.parse import urlparse

class URLShortener:
    def __init__(self, db_name):
        self.db_name = db_name
        self.lock = threading.Lock()
        self.conn = sqlite3.connect(db_name)
        self.cursor = self.conn.cursor()
        self.cursor.execute("""
            CREATE TABLE IF NOT EXISTS url_mappings
            (short_url TEXT PRIMARY KEY, long_url TEXT)
        """)
        self.conn.commit()

    def shorten_url(self, long_url):
        if not self.validate_url(long_url):
            raise ValueError("Invalid URL")

        hash_object = hashlib.sha256(long_url.encode())
        short_url = hash_object.hexdigest()[:6]

        with self.lock:
            self.cursor.execute("SELECT * FROM url_mappings WHERE short_url = ?", (short_url,))
            if self.cursor.fetchone():
                # Collision detected, generate a new hash
                short_url = self.generate_new_hash(long_url)

            self.cursor.execute("INSERT INTO url_mappings VALUES (?, ?)", (short_url, long_url))
            self.conn.commit()

        return short_url

    def lookup_short_url(self, short_url):
        with self.lock:
            self.cursor.execute("SELECT long_url FROM url_mappings WHERE short_url = ?", (short_url,))
            result = self.cursor.fetchone()
            if result:
                return result[0]
            else:
                return None

    def validate_url(self, url):
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

    def generate_new_hash(self, long_url):
        hash_object = hashlib.sha256(long_url.encode())
        return hash_object.hexdigest()[:6]

def main():
    shortener = URLShortener("url_shortener.db")
    long_url = "https://www.example.com/very/long/url"
    short_url = shortener.shorten_url(long_url)
    print(f"Shortened URL: {short_url}")
    print(f"Original URL: {shortener.lookup_short_url(short_url)}")

if __name__ == "__main__":
    main()
