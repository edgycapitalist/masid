### Overview

The in-memory task queue is designed to manage tasks with priorities, track their status, and support worker threads for task execution. The system will provide a robust and scalable solution for task management, ensuring efficient task processing and reliable error handling.

### System Components

1. **Task Queue**: The central component responsible for storing and managing tasks.
2. **Task**: Represents a single task with a unique ID, priority, and status.
3. **Worker Thread**: Consumes tasks from the task queue and executes them.
4. **Task Status Tracker**: Tracks the status of tasks (pending, running, completed, failed).
5. **Retry Mechanism**: Configurable max retries for failed tasks.

### System Flow

1. **Task Addition**: Tasks are added to the task queue with a specified priority.
2. **Task Retrieval**: Worker threads retrieve tasks from the task queue based on priority.
3. **Task Execution**: Worker threads execute tasks and update their status.
4. **Task Status Update**: The task status tracker updates the status of tasks (running, completed, failed).
5. **Retry Mechanism**: Failed tasks are retried up to a configurable max retries.

import threading
import queue
import time

class Task:
    def __init__(self, task_id, priority, func):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.status = "pending"

    def execute(self):
        try:
            self.func()
            self.status = "completed"
        except Exception as e:
            self.status = "failed"
            print(f"Task {self.task_id} failed: {e}")

class TaskQueue:
    def __init__(self):
        self.queue = queue.PriorityQueue()
        self.task_status_tracker = {}

    def add_task(self, task):
        self.queue.put((task.priority, task))
        self.task_status_tracker[task.task_id] = task.status

    def get_task(self):
        if self.queue.empty():
            return None
        _, task = self.queue.get()
        return task

    def update_task_status(self, task_id, status):
        if task_id in self.task_status_tracker:
            self.task_status_tracker[task_id] = status

class WorkerThread(threading.Thread):
    def __init__(self, task_queue):
        super().__init__()
        self.task_queue = task_queue

    def run(self):
        while True:
            task = self.task_queue.get_task()
            if task is None:
                break
            task.status = "running"
            self.task_queue.update_task_status(task.task_id, task.status)
            task.execute()
            self.task_queue.update_task_status(task.task_id, task.status)

class TaskStatusTracker:
    def __init__(self):
        self.task_status = {}

    def update_task_status(self, task_id, status):
        self.task_status[task_id] = status

def example_task():
    print("Example task executed")

def main():
    task_queue = TaskQueue()
    worker_thread = WorkerThread(task_queue)

    task1 = Task(1, 1, example_task)
    task2 = Task(2, 2, example_task)

    task_queue.add_task(task1)
    task_queue.add_task(task2)

    worker_thread.start()
    worker_thread.join()

if __name__ == "__main__":
    main()
