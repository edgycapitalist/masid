**Architecture Document: URL Shortener Library**

### Introduction
The URL Shortener Library is a Python library designed to provide a simple and efficient way to shorten URLs using a hash-based approach. The library will include the following features:

*   Accept a long URL and return a shortened version
*   Support a reverse lookup from short URL to original URL
*   Handle collision detection
*   Include input validation (valid URL format)
*   Be thread-safe for concurrent usage

### System Components
The URL Shortener Library will consist of the following components:

*   **URLValidator**: Responsible for validating the input URL format.
*   **URLShortener**: Uses a hash-based approach to shorten the input URL.
*   **URLDatabase**: Stores the mapping between short and long URLs.
*   **CollisionDetector**: Handles collisions by generating a new short URL.

### System Architecture
The system architecture is designed to be modular and scalable. The components interact as follows:

1.  The **URLValidator** checks the input URL format and returns an error if it's invalid.
2.  The **URLShortener** generates a short URL using a hash-based approach.
3.  The **CollisionDetector** checks if the generated short URL already exists in the **URLDatabase**. If it does, it generates a new short URL.
4.  The **URLDatabase** stores the mapping between the short and long URLs.

### Data Flow
The data flow is as follows:

1.  The user inputs a long URL.
2.  The **URLValidator** validates the input URL format.
3.  The **URLShortener** generates a short URL.
4.  The **CollisionDetector** checks for collisions and generates a new short URL if necessary.
5.  The **URLDatabase** stores the mapping between the short and long URLs.

### Technology Choices
The URL Shortener Library will use the following technologies:

*   **Python 3.9**: As the programming language.
*   **hashlib**: For generating hashes.
*   **threading**: For thread-safe concurrent usage.
*   **sqlite3**: For storing the URL database.

    *   `validate_url(url)`: Validates the input URL format.

import hashlib
import threading
import sqlite3
from urllib.parse import urlparse

class URLValidator:
    def validate_url(self, url):
        try:
            parsed_url = urlparse(url)
            return all([parsed_url.scheme, parsed_url.netloc])
        except ValueError:
            return False

class URLShortener:
    def __init__(self, url_database):
        self.url_database = url_database

    def shorten_url(self, url):
        hash_object = hashlib.sha256(url.encode())
        short_url = hash_object.hexdigest()[:6]
        return short_url

class URLDatabase:
    def __init__(self, db_name):
        self.db_name = db_name
        self.lock = threading.Lock()
        self.conn = sqlite3.connect(db_name)
        self.cursor = self.conn.cursor()
        self.cursor.execute("""
            CREATE TABLE IF NOT EXISTS urls
            (short_url TEXT PRIMARY KEY, long_url TEXT)
        """)
        self.conn.commit()

    def store_url(self, short_url, long_url):
        with self.lock:
            self.cursor.execute("INSERT INTO urls VALUES (?, ?)", (short_url, long_url))
            self.conn.commit()

    def get_long_url(self, short_url):
        with self.lock:
            self.cursor.execute("SELECT long_url FROM urls WHERE short_url = ?", (short_url,))
            result = self.cursor.fetchone()
            return result[0] if result else None

class CollisionDetector:
    def __init__(self, url_database):
        self.url_database = url_database

    def detect_collision(self, short_url):
        return self.url_database.get_long_url(short_url) is not None

class URLShortenerLibrary:
    def __init__(self, db_name):
        self.url_validator = URLValidator()
        self.url_database = URLDatabase(db_name)
        self.url_shortener = URLShortener(self.url_database)
        self.collision_detector = CollisionDetector(self.url_database)

    def shorten_url(self, url):
        if not self.url_validator.validate_url(url):
            raise ValueError("Invalid URL format")
        short_url = self.url_shortener.shorten_url(url)
        if self.collision_detector.detect_collision(short_url):
            # Handle collision by generating a new short URL
            short_url = self.url_shortener.shorten_url(url + str(hashlib.sha256(url.encode()).hexdigest()))
        self.url_database.store_url(short_url, url)
        return short_url

    def get_long_url(self, short_url):
        return self.url_database.get_long_url(short_url)

# Example usage
library = URLShortenerLibrary("url_database.db")
short_url = library.shorten_url("https://www.example.com")
print(short_url)
long_url = library.get_long_url(short_url)
print(long_url)

            try:
