## Overview

The URL Shortener Library is a Python library designed to provide a simple and efficient way to shorten URLs using a hash-based approach. The library supports reverse lookup, collision detection, input validation, and is thread-safe for concurrent usage.

## System Components

*   **URL Shortener**: The core component responsible for shortening URLs using a hash-based approach.
*   **URL Store**: A data structure used to store the mapping between shortened URLs and their corresponding original URLs.
*   **Collision Detector**: A mechanism to detect and handle collisions that may occur during the shortening process.
*   **Input Validator**: A component responsible for validating the format of input URLs.

## System Interfaces

*   `shorten_url(long_url: str) -> str`: Shortens a given long URL and returns the shortened version.
*   `get_original_url(short_url: str) -> str`: Retrieves the original URL corresponding to a given shortened URL.
*   `is_valid_url(url: str) -> bool`: Validates the format of a given URL.

## Data Flow

1.  The user provides a long URL to the `shorten_url` method.
2.  The `shorten_url` method uses a hash-based approach to generate a shortened URL.
3.  The `Collision Detector` checks if the generated shortened URL already exists in the `URL Store`.
4.  If a collision is detected, the `Collision Detector` generates a new shortened URL until a unique one is found.
5.  The `Input Validator` checks if the input URL is in a valid format.
6.  If the input URL is valid, the `URL Shortener` stores the mapping between the shortened URL and the original URL in the `URL Store`.
7.  The `get_original_url` method retrieves the original URL corresponding to a given shortened URL from the `URL Store`.

## Technology Choices

*   **Hashing Algorithm**: The library uses the SHA-256 hashing algorithm to generate shortened URLs.
*   **Data Structure**: The `URL Store` uses a dictionary (hash table) to store the mapping between shortened URLs and their corresponding original URLs.
*   **Thread Safety**: The library uses the `threading` module to ensure thread safety for concurrent usage.

## Implementation Code

```python
import hashlib
import threading
from urllib.parse import urlparse

class URLShortener:
    def __init__(self):
        self.url_store = {}
        self.lock = threading.Lock()

    def shorten_url(self, long_url: str) -> str:
        """
        Shortens a given long URL and returns the shortened version.
        
        Args:
        long_url (str): The long URL to be shortened.
        
        Returns:
        str: The shortened URL.
        """
        if not self.is_valid_url(long_url):
            raise ValueError("Invalid URL format")

        with self.lock:
            hash_object = hashlib.sha256(long_url.encode())
            short_url = hash_object.hexdigest()[:6]

            # Collision detection
            while short_url in self.url_store:
                hash_object = hashlib.sha256((long_url + str(threading.get_ident())).encode())
                short_url = hash_object.hexdigest()[:6]

            self.url_store[short_url] = long_url
            return short_url

    def get_original_url(self, short_url: str) -> str:
        """
        Retrieves the original URL corresponding to a given shortened URL.
        
        Args:
        short_url (str): The shortened URL.
        
        Returns:
        str: The original URL.
        """
        with self.lock:
            return self.url_store.get(short_url)

    def is_valid_url(self, url: str) -> bool:
        """
        Validates the format of a given URL.
        
        Args:
        url (str): The URL to be validated.
        
        Returns:
        bool: True if the URL is in a valid format, False otherwise.
        """
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False
