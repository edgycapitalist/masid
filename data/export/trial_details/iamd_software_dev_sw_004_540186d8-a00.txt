### Overview

The in-memory task queue is designed to manage tasks with priorities, support worker threads, track task status, and provide task retry with configurable max retries. The system will also include a mechanism for graceful shutdown.

### Components

1. **Task Queue**: A data structure to store tasks with their respective priorities.
2. **Task**: A unit of work that can be executed by a worker thread.
3. **Worker Thread**: A thread that consumes tasks from the task queue and executes them.
4. **Task Status Tracker**: A component that tracks the status of tasks (pending, running, completed, failed).
5. **Retry Mechanism**: A component that handles task retry with configurable max retries.

### Interfaces

1. **Task Queue Interface**: Provides methods to add tasks, consume tasks, and track task status.
2. **Worker Thread Interface**: Provides methods to execute tasks and update task status.
3. **Task Interface**: Provides methods to get task details (priority, status, etc.).

### Data Flow

1. **Task Addition**: A task is added to the task queue with its priority.
2. **Task Consumption**: A worker thread consumes a task from the task queue.
3. **Task Execution**: The worker thread executes the task and updates its status.
4. **Task Status Tracking**: The task status tracker updates the task status (pending, running, completed, failed).
5. **Task Retry**: If a task fails, the retry mechanism will retry the task up to the configurable max retries.

### Technology Choices

1. **Programming Language**: Python
2. **Data Structure**: PriorityQueue (from `queue` module) for task queue
3. **Threading**: `threading` module for worker threads
4. **Task Status Tracking**: Dictionary to store task status

import queue
import threading
from enum import Enum

class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, priority, func, args=None, kwargs=None):
        self.priority = priority
        self.func = func
        self.args = args
        self.kwargs = kwargs
        self.status = TaskStatus.PENDING
        self.retries = 0
        self.max_retries = 3  # configurable max retries

    def execute(self):
        try:
            self.func(*self.args, **self.kwargs)
            self.status = TaskStatus.COMPLETED
        except Exception as e:
            self.status = TaskStatus.FAILED
            self.retries += 1
            if self.retries < self.max_retries:
                # retry task
                self.status = TaskStatus.PENDING

class TaskQueue:
    def __init__(self):
        self.queue = queue.PriorityQueue()
        self.tasks = {}  # task_id -> Task
        self.lock = threading.Lock()

    def add_task(self, task):
        with self.lock:
            task_id = len(self.tasks) + 1
            self.tasks[task_id] = task
            self.queue.put((task.priority, task_id))

    def consume_task(self):
        with self.lock:
            if self.queue.empty():
                return None
            priority, task_id = self.queue.get()
            task = self.tasks[task_id]
            task.status = TaskStatus.RUNNING
            return task

    def update_task_status(self, task_id, status):
        with self.lock:
            task = self.tasks[task_id]
            task.status = status

class WorkerThread(threading.Thread):
    def __init__(self, task_queue):
        super().__init__()
        self.task_queue = task_queue

    def run(self):
        while True:
            task = self.task_queue.consume_task()
            if task is None:
                break
            task.execute()
            self.task_queue.update_task_status(task.id, task.status)

def main():
    task_queue = TaskQueue()

    # add tasks
    task1 = Task(1, lambda: print("Task 1 executed"))
    task2 = Task(2, lambda: print("Task 2 executed"))
    task_queue.add_task(task1)
    task_queue.add_task(task2)

    # start worker threads
    worker_threads = [WorkerThread(task_queue) for _ in range(2)]
    for thread in worker_threads:
        thread.start()

    # wait for worker threads to finish
    for thread in worker_threads:
        thread.join()

if __name__ == "__main__":
    main()

                self.task_queue.update_task_status(task.task_id, task.status)

def example_task():
    time.sleep(random.uniform(0.1, 1.0))
    if random.random() < 0.5:
        raise Exception("Example task failed")

def main():
    task_queue = TaskQueue()

    # add tasks
    for _ in range(10):
        task = Task(1, example_task)
        task_queue.add_task(task)

    # start worker threads
    worker_threads = [WorkerThread(task_queue) for _ in range(5)]
    for thread in worker_threads:
        thread.start()

    # wait for worker threads to finish
    time.sleep(10)
    task_queue.stop()
    for thread in worker_threads:
        thread.join()

if __name__ == "__main__":
    main()
