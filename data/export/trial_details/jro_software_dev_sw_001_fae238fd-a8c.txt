**URL Shortening Library Architecture Document**
=====================================================

### Overview

The URL shortening library provides a simple and efficient way to shorten URLs using a hash-based approach. The library is designed to be thread-safe, handling concurrent usage and supporting reverse lookup from short URL to original URL.

### Module Decomposition

The library consists of the following modules:

*   `url_shortener`: This module contains the main `URLShortener` class responsible for shortening URLs and handling reverse lookups.
*   `hash_generator`: This module contains the `HashGenerator` class responsible for generating unique hashes for URLs.
*   `collision_detector`: This module contains the `CollisionDetector` class responsible for detecting and handling collisions.
*   `url_validator`: This module contains the `URLValidator` class responsible for validating the input URL format.

### Interfaces

The library provides the following interfaces:

*   `shorten_url(long_url)`: This method takes a long URL as input and returns a shortened version.
*   `get_original_url(short_url)`: This method takes a short URL as input and returns the original URL.
*   `validate_url(url)`: This method takes a URL as input and returns a boolean indicating whether the URL is valid.

### Data Flow

The data flow of the library is as follows:

1.  The user provides a long URL to the `shorten_url` method.
2.  The `URLValidator` class validates the input URL format.
3.  If the URL is valid, the `HashGenerator` class generates a unique hash for the URL.
4.  The `CollisionDetector` class checks for collisions and handles them if necessary.
5.  The shortened URL is returned to the user.
6.  The user can use the `get_original_url` method to retrieve the original URL from the shortened URL.

The library uses the following technologies:

import hashlib
import threading
from urllib.parse import urlparse

class URLValidator:
    def __init__(self):
        pass

    def validate_url(self, url):
        """Validate the input URL format"""
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

class HashGenerator:
    def __init__(self):
        pass

    def generate_hash(self, url):
        """Generate a unique hash for the URL"""
        return hashlib.sha256(url.encode()).hexdigest()[:6]

class CollisionDetector:
    def __init__(self):
        self.url_map = {}
        self.lock = threading.Lock()

    def add_url(self, short_url, long_url):
        """Add a URL to the map and handle collisions"""
        with self.lock:
            if short_url in self.url_map:
                # Handle collision
                return False
            self.url_map[short_url] = long_url
            return True

    def get_original_url(self, short_url):
        """Get the original URL from the shortened URL"""
        with self.lock:
            return self.url_map.get(short_url)

class URLShortener:
    def __init__(self):
        self.url_validator = URLValidator()
        self.hash_generator = HashGenerator()
        self.collision_detector = CollisionDetector()

    def shorten_url(self, long_url):
        """Shorten a URL and return the shortened version"""
        if not self.url_validator.validate_url(long_url):
            raise ValueError("Invalid URL format")

        short_url = self.hash_generator.generate_hash(long_url)
        if not self.collision_detector.add_url(short_url, long_url):
            # Handle collision
            return self.shorten_url(long_url)

        return short_url

    def get_original_url(self, short_url):
        """Get the original URL from the shortened URL"""
        return self.collision_detector.get_original_url(short_url)

# Example usage
url_shortener = URLShortener()
long_url = "https://www.example.com"
short_url = url_shortener.shorten_url(long_url)
print(f"Shortened URL: {short_url}")
print(f"Original URL: {url_shortener.get_original_url(short_url)}")
