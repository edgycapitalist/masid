**Architecture Document: In-Memory Task Queue**

## Introduction
The in-memory task queue is designed to manage tasks with priorities, track task status, and provide task retry with configurable max retries. The system will support multiple worker threads to consume tasks and provide a mechanism for graceful shutdown.

## System Components
The system consists of the following components:

1. **Task Queue**: A data structure that stores tasks with their respective priorities.
2. **Worker Threads**: Threads that consume tasks from the task queue and execute them.
3. **Task Status Tracker**: A component that tracks the status of tasks (pending, running, completed, failed).
4. **Task Retry Manager**: A component that manages task retries with configurable max retries.
5. **Shutdown Manager**: A component that handles graceful shutdown of the system.

## System Design
The system design is based on the following principles:

1. **Modularity**: Each component is designed to be modular and independent, allowing for easy maintenance and modification.
2. **Thread Safety**: The system is designed to be thread-safe, ensuring that multiple worker threads can access the task queue and task status tracker without conflicts.
3. **Priority Queue**: The task queue is implemented as a priority queue, ensuring that tasks with higher priorities are executed first.

## Class Diagram
The system consists of the following classes:

1. **Task**: Represents a task with a unique ID, priority, and status.
2. **TaskQueue**: Represents the task queue, which stores tasks and provides methods for adding and consuming tasks.
3. **WorkerThread**: Represents a worker thread that consumes tasks from the task queue and executes them.
4. **TaskStatusTracker**: Represents the task status tracker, which tracks the status of tasks.
5. **TaskRetryManager**: Represents the task retry manager, which manages task retries with configurable max retries.
6. **ShutdownManager**: Represents the shutdown manager, which handles graceful shutdown of the system.

## Sequence Diagram
The system sequence diagram is as follows:

1. A task is added to the task queue with a priority.
2. A worker thread consumes the task from the task queue and executes it.
3. The task status tracker updates the task status to "running".
4. If the task fails, the task retry manager retries the task with a configurable max retries.
5. If the task completes successfully, the task status tracker updates the task status to "completed".
6. If the system is shut down, the shutdown manager stops all worker threads and ensures that all tasks are completed or failed.

```python
import threading
import queue
import time
from enum import Enum

class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id, priority, func, args, kwargs):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.args = args
        self.kwargs = kwargs
        self.status = TaskStatus.PENDING

class TaskQueue:
    def __init__(self):
        self.queue = queue.PriorityQueue()

    def add_task(self, task):
        self.queue.put((task.priority, task))

    def consume_task(self):
        return self.queue.get()[1]

class WorkerThread(threading.Thread):
    def __init__(self, task_queue, task_status_tracker, task_retry_manager):
        super().__init__()
        self.task_queue = task_queue
        self.task_status_tracker = task_status_tracker
        self.task_retry_manager = task_retry_manager

    def run(self):
        while True:
            task = self.task_queue.consume_task()
            self.task_status_tracker.update_task_status(task, TaskStatus.RUNNING)
            try:
                task.func(*task.args, **task.kwargs)
                self.task_status_tracker.update_task_status(task, TaskStatus.COMPLETED)
            except Exception as e:
                self.task_retry_manager.retry_task(task)
                self.task_status_tracker.update_task_status(task, TaskStatus.FAILED)

class TaskStatusTracker:
    def __init__(self):
        self.task_status = {}

    def update_task_status(self, task, status):
        self.task_status[task.task_id] = status

class TaskRetryManager:
    def __init__(self, max_retries):
        self.max_retries = max_retries

    def retry_task(self, task):
        if task.status == TaskStatus.FAILED and task.retries < self.max_retries:
            task.retries += 1
            task.status = TaskStatus.PENDING

class ShutdownManager:
    def __init__(self):
        self.shutdown = False

    def shutdown_system(self):
        self.shutdown = True

def main():
    task_queue = TaskQueue()
    task_status_tracker = TaskStatusTracker()
    task_retry_manager = TaskRetryManager(3)
    shutdown_manager = ShutdownManager()

    worker_threads = []
    for _ in range(5):
        worker_thread = WorkerThread(task_queue, task_status_tracker, task_retry_manager)
        worker_thread.start()
        worker_threads.append(worker_thread)

    task_queue.add_task(Task(1, 1, lambda: print("Task 1"), (), {}))
    task_queue.add_task(Task(2, 2, lambda: print("Task 2"), (), {}))

    time.sleep(5)

    shutdown_manager.shutdown_system()

    for worker_thread in worker_threads:
        worker_thread.join()

if __name__ == "__main__":
    main()
```

```python
import threading
import queue
import time
from enum import Enum
import logging

logging.basicConfig(level=logging.INFO)

class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id, priority, func, args, kwargs):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.args = args
        self.kwargs = kwargs
        self.status = TaskStatus.PENDING
        self.retries = 0

class TaskQueue:
    def __init__(self):
        self.queue = queue.PriorityQueue()

    def add_task(self, task):
        self.queue.put((task.priority, task))

    def consume_task(self):
        return self.queue.get()[1]

class WorkerThread(threading.Thread):
    def __init__(self, task_queue, task_status_tracker, task_retry_manager):
        super().__init__()
        self.task_queue = task_queue
        self.task_status_tracker = task_status_tracker
        self.task_retry_manager = task_retry_manager

    def run(self):
        while True:
            try:
                task = self.task_queue.consume_task()
                self.task_status_tracker.update_task_status(task, TaskStatus.RUNNING)
                try:
                    task.func(*task.args, **task.kwargs)
                    self.task_status_tracker.update_task_status(task, TaskStatus.COMPLETED)
                except Exception as e:
                    logging.error(f"Task {task.task_id} failed: {e}")
                    self.task_retry_manager.retry_task(task)
                    self.task_status_tracker.update_task_status(task, TaskStatus.FAILED)
            except queue.Empty:
                break

class TaskStatusTracker:
    def __init__(self):
        self.task_status = {}

    def update_task_status(self, task, status):
        self.task_status[task.task_id] = status

class TaskRetryManager:
    def __init__(self, max_retries):
        self.max_retries = max_retries

    def retry_task(self, task):
        if task.retries < self.max_retries:
            task.retries += 1
            task.status = TaskStatus.PENDING
            logging.info(f"Retrying task {task.task_id} (retry {task.retries}/{self.max_retries})")
        else:
            logging.info(f"Task {task.task_id} failed after {self.max_retries} retries")

class ShutdownManager:
    def __init__(self):
        self.shutdown = False

    def shutdown_system(self):
        self.shutdown = True

def main():
    task_queue = TaskQueue()
    task_status_tracker = TaskStatusTracker()
    task_retry_manager = TaskRetryManager(3)
    shutdown_manager = ShutdownManager()

    worker_threads = []
    for _ in range(5):

    task_queue.add_task(Task(1, 1, lambda: time.sleep(1), (), {}))
    task_queue.add_task(Task(2, 2, lambda: time.sleep(2), (), {}))

    time.sleep(5)

    shutdown_manager.shutdown_system()

    for worker_thread in worker_threads:
        worker_thread.join()

if __name__ == "__main__":
    main()
```

```python
import threading
import queue
import time
from enum import Enum
import logging

logging.basicConfig(level=logging.INFO)

class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id, priority, func, args, kwargs):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.args = args
        self.kwargs = kwargs
        self.status = TaskStatus.PENDING
        self.retries = 0

class TaskQueue:
    def __init__(self):
        self.queue = queue.PriorityQueue()

    def add_task(self, task):
        self.queue.put((task.priority, task))

    def consume_task(self):
        return self.queue.get()[1]

class WorkerThread(threading.Thread):
    def __init__(self, task_queue, task_status_tracker, task_retry_manager):
        super().__init__()
        self.task_queue = task_queue
        self.task_status_tracker = task_status_tracker
        self.task_retry_manager = task_retry_manager
```
```python
        self.shutdown = threading.Event()

    def run(self):
        while not self.shutdown.is_set():
            try:
                task = self.task_queue.consume_task()
                self.task_status_tracker.update_task_status(task, TaskStatus.RUNNING)
                try:
                    task.func(*task.args, **task.kwargs)
                    self.task_status_tracker.update_task_status(task, TaskStatus.COMPLETED)
                except Exception as e:
                    logging.error(f"Task {task.task_id} failed: {e}")
                    self.task_retry_manager.retry_task(task)
                    self.task_status_tracker.update_task_status(task, TaskStatus.FAILED)
            except queue.Empty:
                time.sleep(0.1)

    def stop(self):
        self.shutdown.set()

    def __init__(self):
        self.task_status = {}

    def update_task_status(self, task, status):
        self.task_status[task.task_id] = status

    def __init__(self, max_retries):
        self.max_retries = max_retries

    def retry_task(self, task):
        if task.retries < self.max_retries:
            task.retries += 1
            task.status = TaskStatus.PENDING
            logging.info(f"Retrying task {task.task_id} (retry {task.retries}/{self.max_retries})")
        else:
            logging.info(f"Task {task.task_id} failed after {self.max_retries} retries")

class ShutdownManager:
    def __init__(self):
        self.shutdown = False

    def shutdown_system(self, worker_threads):
        for worker_thread in worker_threads:
            worker_thread.stop()
        self.shutdown = True

def main():
```

```python
class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id, priority, func, args, kwargs):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.args = args
        self.kwargs = kwargs
        self.status = TaskStatus.PENDING
        self.retries = 0

class TaskQueue:
    def __init__(self):
        self.queue = queue.PriorityQueue()

    def add_task(self, task):
        self.queue.put((task.priority, task))

    def consume_task(self):
        return self.queue.get()[1]

class WorkerThread(threading.Thread):
    def __init__(self, task_queue, task_status_tracker, task_retry_manager):
        super().__init__()
        self.task_queue = task_queue
        self.task_status_tracker = task_status_tracker
        self.task_retry_manager = task_retry_manager
        self.shutdown = threading.Event()

    def run(self):
        while not self.shutdown.is_set():
            try:
                task = self.task_queue.consume_task()
                self.task_status_tracker.update_task_status(task, TaskStatus.RUNNING)
                try:
                    task.func(*task.args, **task.kwargs)
                    self.task_status_tracker.update_task_status(task, TaskStatus.COMPLETED)
                except Exception as e:
                    logging.error(f"Task {task.task_id} failed: {e}")
                    self.task_retry_manager.retry_task(task)
                    self.task_status_tracker.update_task_status(task, TaskStatus.FAILED)
            except queue.Empty:
                time.sleep(0.1)

    def stop(self):
        self.shutdown.set()

class TaskStatusTracker:
    def __init__(self):
        self.task_status = {}

    def update_task_status(self, task, status):
        self.task_status[task.task_id] = status

class TaskRetryManager:
    def __init__(self, max_retries):
        self.max_retries = max_retries

    def retry_task(self, task):
        if task.retries < self.max_retries:
            task.retries += 1
            task.status = TaskStatus.PENDING
            logging.info(f"Retrying task {task.task_id} (retry {task.retries}/{self.max_retries})")
        else:
            logging.info(f"Task {task.task_id} failed after {self.max_retries} retries")

class ShutdownManager:
    def __init__(self):
        self.shutdown = False

    def shutdown_system(self, worker_threads):
        for worker_thread in worker_threads:
            worker_thread.stop()
        self.shutdown = True

def main():
    task_queue = TaskQueue()
    task_status_tracker = TaskStatusTracker()
    task_retry_manager = TaskRetryManager(3)
    shutdown_manager = ShutdownManager()

    worker_threads = []
    for _ in range(5):
        worker_thread = WorkerThread(task_queue, task_status_tracker, task_retry_manager)
        worker_thread.start()
        worker_threads.append(worker_thread)

    task_queue.add_task(Task(1, 1, lambda: time.sleep(1), (), {}))
    task_queue.add_task(Task(2, 2, lambda: time.sleep(2), (), {}))

    time.sleep(5)

    shutdown_manager.shutdown_system(worker_threads)

    for worker_thread in worker_threads:
        worker_thread.join()

if __name__ == "__main__":
    main()
```
