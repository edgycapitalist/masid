**Architecture Document**

**Overview**
-----------

The proposed data pipeline is designed to handle CSV files with configurable schema validation, apply a series of transformations, and output the results in CSV or JSON format. The pipeline will also support a plugin system for custom transformations and provide clear error reporting for malformed data.

**Components**
------------

1. **Schema Validator**: Responsible for validating the schema of the input CSV files.
2. **Data Reader**: Reads the input CSV files and passes the data to the pipeline.
3. **Transformation Chain**: Applies a series of transformations (filtering, mapping, aggregation) to the data.
4. **Plugin System**: Allows for custom transformations to be added to the pipeline.
5. **Data Writer**: Outputs the transformed data in CSV or JSON format.
6. **Error Reporter**: Handles error reporting for malformed data.

**Data Flow**
-------------

1. The **Data Reader** reads the input CSV file and passes the data to the **Schema Validator**.
2. The **Schema Validator** checks the schema of the data against a predefined schema. If the schema is invalid, an error is reported to the **Error Reporter**.
3. The validated data is then passed to the **Transformation Chain**.
4. The **Transformation Chain** applies a series of transformations to the data. Each transformation is a separate plugin that can be added or removed from the chain.
5. The transformed data is then passed to the **Data Writer**.
6. The **Data Writer** outputs the transformed data in CSV or JSON format.

from abc import ABC, abstractmethod

class Transformation(ABC):
    @abstractmethod
    def transform(self, data):
        pass

class FilteringTransformation(Transformation):
    def __init__(self, filter_func):
        self.filter_func = filter_func

    def transform(self, data):
        return [x for x in data if self.filter_func(x)]

class MappingTransformation(Transformation):
    def __init__(self, map_func):
        self.map_func = map_func

    def transform(self, data):
        return [self.map_func(x) for x in data]

class AggregationTransformation(Transformation):
    def __init__(self, agg_func):
        self.agg_func = agg_func

    def transform(self, data):
        return [self.agg_func(x) for x in data]

class TransformationChain:
    def __init__(self):
        self.transformations = []

    def add_transformation(self, transformation):
        self.transformations.append(transformation)

    def transform(self, data):
        for transformation in self.transformations:
            data = transformation.transform(data)
        return data

**Overview**
-----------

The proposed data pipeline is designed to handle CSV files with configurable schema validation, apply a series of transformations, and output the results in CSV or JSON format. The pipeline will also support a plugin system for custom transformations and provide clear error reporting for malformed data.

**Components**
------------

1. **Schema Validator**: Responsible for validating the schema of the input CSV files.
2. **Data Reader**: Reads the input CSV files and passes the data to the pipeline.
3. **Transformation Chain**: Applies a series of transformations (filtering, mapping, aggregation) to the data.
4. **Plugin System**: Allows for custom transformations to be added to the pipeline.
5. **Data Writer**: Outputs the transformed data in CSV or JSON format.
6. **Error Reporter**: Handles error reporting for malformed data.

**Data Flow**
-------------

1. The **Data Reader** reads the input CSV file and passes the data to the **Schema Validator**.
2. The **Schema Validator** checks the schema of the data against a predefined schema. If the schema is invalid, an error is reported to the **Error Reporter**.
3. The validated data is then passed to the **Transformation Chain**.
4. The **Transformation Chain** applies a series of transformations to the data. Each transformation is a separate plugin that can be added or removed from the chain.
5. The transformed data is then passed to the **Data Writer**.
6. The **Data Writer** outputs the transformed data in CSV or JSON format.

def test_pipeline(tmp_path):
    schema = {"name": str, "age": int}
    input_file_path = tmp_path / "input.csv"
    output_file_path = tmp_path / "output.csv"
    output_format = "csv"

    # Create a sample input file
    data = {"name": ["John", "Jane"], "age": [30, 25]}
    pd.DataFrame(data).to_csv(input_file_path, index=False)

    pipeline = Pipeline(schema, str(input_file_path), str(output_file_path), output_format)
    pipeline.transformation_chain.add_transformation(FilteringTransformation(lambda x: x["age"] > 20))
    pipeline.run()

    # Check if the output file exists
    assert output_file_path.exists()

def test_schema_validator():
    schema = {"name": str, "age": int}
    data = {"name": ["John", "Jane"], "age": [30, 25]}
    schema_validator = SchemaValidator(schema)
    validated_data = schema_validator.validate(pd.DataFrame(data))
    assert isinstance(validated_data, pd.DataFrame)

def test_schema_validator_invalid_data():
    schema = {"name": str, "age": int}
    data = {"name": ["John", "Jane"], "age": ["thirty", 25]}
    schema_validator = SchemaValidator(schema)
    with pytest.raises(ValueError):
        schema_validator.validate(pd.DataFrame(data))

def test_data_reader(tmp_path):
    input_file_path = tmp_path / "input.csv"
    data = {"name": ["John", "Jane"], "age": [30, 25]}
    pd.DataFrame(data).to_csv(input_file_path, index=False)

    data_reader = DataReader(str(input_file_path))
    data = data_reader.read()

def test_error_reporter():
