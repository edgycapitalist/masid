**Introduction**

The URL shortening library is designed to provide a hash-based approach for shortening URLs, supporting reverse lookup, collision detection, input validation, and thread safety. This document outlines the high-level structure, module decomposition, interfaces, data flow, and technology choices.

**Module Decomposition**

The library consists of the following modules:

1. **UrlValidator**: responsible for validating the input URL format.
2. **HashGenerator**: generates a shortened URL using a hash-based approach.
3. **CollisionDetector**: detects and handles collisions between shortened URLs.
4. **UrlMapper**: stores and retrieves the mapping between shortened and original URLs.
5. **ThreadSafeUrlShortener**: provides a thread-safe interface for concurrent usage.

**Interfaces**

The following interfaces are defined:

1. **UrlShortenerInterface**: defines the methods for shortening and reverse lookup.
2. **UrlValidatorInterface**: defines the method for validating the input URL format.

**Data Flow**

The data flow is as follows:

1. The user provides a long URL to the **UrlShortenerInterface**.
2. The **UrlValidator** validates the input URL format.
3. If valid, the **HashGenerator** generates a shortened URL.
4. The **CollisionDetector** checks for collisions between the shortened URL and existing mappings.
5. If a collision is detected, the **HashGenerator** generates a new shortened URL.
6. The **UrlMapper** stores the mapping between the shortened and original URLs.
7. The user can retrieve the original URL using the **UrlShortenerInterface**.

**Technology Choices**

The library uses the following technologies:

1. **Python 3.x**: as the programming language.
2. **hashlib**: for generating hashes.
3. **threading**: for thread safety.

import hashlib
import threading
from urllib.parse import urlparse

class UrlValidator:
    def __init__(self):
        pass

    def is_valid_url(self, url):
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

class HashGenerator:
    def __init__(self):
        pass

    def generate_hash(self, url):
        return hashlib.sha256(url.encode()).hexdigest()[:6]

class CollisionDetector:
    def __init__(self):
        self.mappings = {}

    def is_collision(self, short_url):
        return short_url in self.mappings

    def add_mapping(self, short_url, original_url):
        self.mappings[short_url] = original_url

class UrlMapper:
    def __init__(self):
        self.mappings = {}

    def get_original_url(self, short_url):
        return self.mappings.get(short_url)

    def add_mapping(self, short_url, original_url):
        self.mappings[short_url] = original_url

class ThreadSafeUrlShortener:
    def __init__(self):
        self.url_validator = UrlValidator()
        self.hash_generator = HashGenerator()
        self.collision_detector = CollisionDetector()
        self.url_mapper = UrlMapper()
        self.lock = threading.Lock()

    def shorten_url(self, url):
        with self.lock:
            if not self.url_validator.is_valid_url(url):
                raise ValueError("Invalid URL format")

            short_url = self.hash_generator.generate_hash(url)
            while self.collision_detector.is_collision(short_url):
                short_url = self.hash_generator.generate_hash(url + str(self.collision_detector.mappings))

            self.collision_detector.add_mapping(short_url, url)
            self.url_mapper.add_mapping(short_url, url)
            return short_url

    def get_original_url(self, short_url):
        with self.lock:
            return self.url_mapper.get_original_url(short_url)

# Example usage:
url_shortener = ThreadSafeUrlShortener()
short_url = url_shortener.shorten_url("https://www.example.com")
print(short_url)
original_url = url_shortener.get_original_url(short_url)
print(original_url)
