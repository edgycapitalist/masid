## Overview

The data pipeline is designed to read CSV files, apply a chain of transformations, and output the results to CSV or JSON. The pipeline includes a plugin system for custom transformations and provides clear error reporting for malformed data.

## Components

1. **Data Reader**: Responsible for reading CSV files with configurable schema validation.
2. **Transformation Chain**: Applies a series of transformations to the data, including filtering, mapping, and aggregation.
3. **Plugin System**: Allows for custom transformations to be added to the pipeline.
4. **Data Writer**: Outputs the transformed data to CSV or JSON.

## Data Flow

1. The **Data Reader** reads a CSV file and performs schema validation.
2. The validated data is passed to the **Transformation Chain**, which applies the configured transformations.
3. The transformed data is then passed to the **Plugin System**, which applies any custom transformations.
4. The final transformed data is passed to the **Data Writer**, which outputs the data to CSV or JSON.

## Overview

The data pipeline is designed to read CSV files, apply a chain of transformations, and output the results to CSV or JSON. The pipeline includes a plugin system for custom transformations and provides clear error reporting for malformed data.

## Components

1. **Data Reader**: Responsible for reading CSV files with configurable schema validation.
2. **Transformation Chain**: Applies a series of transformations to the data, including filtering, mapping, and aggregation.
3. **Plugin System**: Allows for custom transformations to be added to the pipeline.
4. **Data Writer**: Outputs the transformed data to CSV or JSON.

## Data Flow

1. The **Data Reader** reads a CSV file and performs schema validation.
2. The validated data is passed to the **Transformation Chain**, which applies the configured transformations.
3. The transformed data is then passed to the **Plugin System**, which applies any custom transformations.
4. The final transformed data is passed to the **Data Writer**, which outputs the data to CSV or JSON.

## Technology Choices

* Python 3.9+
* pandas for data manipulation
* json for JSON output
* csv for CSV output
