## Overview
The rate limiter module is designed to control the frequency of requests to a system, preventing excessive usage and potential denial-of-service attacks. The module implements the token bucket algorithm, which allows for a configurable rate and burst size.

## Module Structure
The module consists of the following components:

1. **RateLimiter**: The core class responsible for managing the rate limiting logic.
2. **TokenBucket**: A helper class that implements the token bucket algorithm.
3. **RateLimiterDecorator**: A decorator that allows easy application of rate limiting to functions.

## Interfaces
The module exposes the following interfaces:

1. **RateLimiter**:
	* `__init__(name, rate, burst_size)`: Initializes a rate limiter with a given name, rate, and burst size.
	* `allow_request()`: Checks if a request is allowed based on the current token bucket state.
2. **TokenBucket**:
	* `__init__(rate, burst_size)`: Initializes a token bucket with a given rate and burst size.
	* `consume_tokens(num_tokens)`: Consumes a specified number of tokens from the bucket.
3. **RateLimiterDecorator**:
	* `__init__(rate_limiter)`: Initializes a decorator with a given rate limiter instance.
	* `__call__(func)`: Decorates a function with rate limiting logic.

## Data Flow
The data flow is as follows:

1. A client requests access to a rate-limited function.
2. The `RateLimiterDecorator` checks if the request is allowed by calling the `allow_request()` method of the associated `RateLimiter` instance.
3. If allowed, the `RateLimiter` instance consumes a token from the `TokenBucket` instance.
4. If not allowed, the `RateLimiterDecorator` raises a `RateLimitExceeded` exception.

## Implementation Code
```python
import threading
import time
from functools import wraps

class TokenBucket:
    def __init__(self, rate, burst_size):
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()

    def consume_tokens(self, num_tokens):
        current_time = time.time()
        elapsed_time = current_time - self.last_update
        self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
        self.last_update = current_time

        if num_tokens > self.tokens:
            return False
        self.tokens -= num_tokens
        return True

class RateLimiter:
    def __init__(self, name, rate, burst_size):
        self.name = name
        self.rate = rate
        self.burst_size = burst_size
        self.token_bucket = TokenBucket(rate, burst_size)
        self.lock = threading.Lock()

    def allow_request(self):
        with self.lock:
            return self.token_bucket.consume_tokens(1)

class RateLimiterDecorator:
    def __init__(self, rate_limiter):
        self.rate_limiter = rate_limiter

    def __call__(self, func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            if not self.rate_limiter.allow_request():
                raise RateLimitExceeded(f"Rate limit exceeded for {self.rate_limiter.name}")
            return func(*args, **kwargs)
        return wrapper

class RateLimitExceeded(Exception):
    pass

# Example usage:
rate_limiter = RateLimiter("example", 5, 10)
decorator = RateLimiterDecorator(rate_limiter)

@decorator
def example_function():
    print("Hello, world!")

example_function()

import threading
import time
from functools import wraps
import logging

logging.basicConfig(level=logging.INFO)

class TokenBucket:
    def __init__(self, rate, burst_size):
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()

    def consume_tokens(self, num_tokens):
        current_time = time.time()
        elapsed_time = current_time - self.last_update
        self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
        self.last_update = current_time

        if num_tokens > self.tokens:
            return False
        self.tokens -= num_tokens
        return True

class RateLimiter:
    def __init__(self, name, rate, burst_size):
        self.name = name
        self.rate = rate
        self.burst_size = burst_size
        self.token_bucket = TokenBucket(rate, burst_size)
        self.lock = threading.Lock()
        self.allowed_requests = 0
        self.denied_requests = 0

    def allow_request(self):
        with self.lock:
            if self.token_bucket.consume_tokens(1):
                self.allowed_requests += 1
                return True
            self.denied_requests += 1
            return False

    def get_metrics(self):
        with self.lock:
            return {
                "allowed_requests": self.allowed_requests,
                "denied_requests": self.denied_requests
            }

    def __init__(self, rate_limiter):
        self.rate_limiter = rate_limiter

    def __call__(self, func):

import time
from functools import wraps
import logging
import json

logging.basicConfig(level=logging.INFO)

class TokenBucket:
    def __init__(self, rate, burst_size):
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()

    def consume_tokens(self, num_tokens):
        current_time = time.time()
        elapsed_time = current_time - self.last_update
        self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
        self.last_update = current_time

        if num_tokens > self.tokens:
            return False
        self.tokens -= num_tokens
        return True

class RateLimiter:
    def __init__(self, name, rate, burst_size):
        self.name = name
        self.rate = rate
        self.burst_size = burst_size
        self.token_bucket = TokenBucket(rate, burst_size)
        self.lock = threading.Lock()
        self.allowed_requests = 0
        self.denied_requests = 0

    def allow_request(self):
        with self.lock:
            if self.token_bucket.consume_tokens(1):
                self.allowed_requests += 1
                return True
            self.denied_requests += 1
            return False

    def get_metrics(self):
        with self.lock:
            return {
                "allowed_requests": self.allowed_requests,
                "denied_requests": self.denied_requests
