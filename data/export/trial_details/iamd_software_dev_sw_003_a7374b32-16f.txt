## Introduction

The Python data pipeline is designed to read CSV files with configurable schema validation, apply a chain of transformations, and output results to CSV or JSON. The pipeline also supports a plugin system for custom transformations and provides clear error reporting for malformed data.

## System Components

1. **Data Ingestion**: Responsible for reading CSV files with configurable schema validation.
2. **Transformation Engine**: Applies a chain of transformations (filtering, mapping, aggregation) to the ingested data.
3. **Plugin System**: Allows for custom transformations to be integrated into the pipeline.
4. **Output Module**: Outputs the transformed data to CSV or JSON.
5. **Error Handling**: Provides clear error reporting for malformed data.

## System Flow

1. The **Data Ingestion** component reads a CSV file and validates its schema against a predefined configuration.
2. The validated data is then passed to the **Transformation Engine**, which applies a chain of transformations.
3. The **Transformation Engine** uses the **Plugin System** to integrate custom transformations into the pipeline.
4. The transformed data is then passed to the **Output Module**, which outputs the data to CSV or JSON.
5. Any errors encountered during the pipeline execution are handled by the **Error Handling** component.

## Technical Details

* The pipeline will use the `pandas` library for data manipulation and `json` library for JSON output.
* The plugin system will use a modular design, with plugins implemented as separate Python modules.
* Error handling will use a combination of try-except blocks and logging to provide clear error reporting.

```json

    "columns": [
```

```python
    """

    Returns:
        dict: Schema configuration.
    """
    with open(config_filename, "r") as f:

# Validate schema
def validate_schema(df, schema):
    """
    Validate the schema of a DataFrame against a predefined configuration.

    Args:
        df (pd.DataFrame): DataFrame to validate.
        schema (dict): Schema configuration.

    Raises:
        ValueError: If the schema is invalid.
    """
    for column in schema["columns"]:
        if column["name"] not in df.columns:
            raise ValueError(f"Column {column['name']} not found in DataFrame")
        if column["type"] == "int" and not pd.api.types.is_integer_dtype(df[column["name"]]):
            raise ValueError(f"Column {column['name']} has incorrect data type")
        if column["type"] == "string" and not pd.api.types.is_string_dtype(df[column["name"]]):
            raise ValueError(f"Column {column['name']} has incorrect data type")

# Apply transformations
def apply_transformations(df, transformations):
    """
    Apply a chain of transformations to a DataFrame.

    Args:
        df (pd.DataFrame): DataFrame to transform.
        transformations (list): List of transformation functions.

    Returns:
        pd.DataFrame: Transformed DataFrame.
    """
    for transformation in transformations:
        df = transformation(df)
    return df

# Output to CSV or JSON
def output_to_file(df, filename, format):
    """
    Output a DataFrame to a file in CSV or JSON format.

    Args:
        df (pd.DataFrame): DataFrame to output.
        filename (str): Path to the output file.
        format (str): Output format (CSV or JSON).
    """
    if format == "csv":
        df.to_csv(filename, index=False)
    elif format == "json":
        df.to_json(filename, orient="records")

# Filter by age transformation
def filter_by_age(df, age=18):
    """
    Filter a DataFrame by age.

    Args:
        df (pd.DataFrame): DataFrame to filter.
        age (int): Minimum age.

    Returns:
        pd.DataFrame: Filtered DataFrame.
```
