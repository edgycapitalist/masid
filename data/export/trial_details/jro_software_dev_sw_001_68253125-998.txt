**Architecture Document: URL Shortening Library**

## Overview

This document outlines the design and implementation of a Python library that provides URL shortening functionality using a hash-based approach. The library will support reverse lookup, collision detection, input validation, and thread safety for concurrent usage.

## System Components

1. **URL Shortener**: Responsible for shortening long URLs and storing the mapping between shortened and original URLs.
2. **URL Validator**: Verifies the input URL format to ensure it conforms to the expected format.
3. **Collision Detector**: Handles collisions that occur when two different long URLs are hashed to the same shortened URL.
4. **Reverse Lookup**: Allows retrieving the original URL from a shortened URL.

## Design

### URL Shortener

* The URL shortener will use a hash function (e.g., SHA-256) to generate a unique shortened URL for each input long URL.
* The shortened URL will be stored in a dictionary (e.g., `short_to_long`) with the original URL as the value.
* To ensure thread safety, the shortener will use a lock (e.g., `threading.Lock`) to synchronize access to the dictionary.

### URL Validator

* The URL validator will use a regular expression to check if the input URL conforms to the expected format (e.g., `^https?://[^\s]+$`).
* If the input URL is invalid, the validator will raise a `ValueError` exception.

### Collision Detector

* The collision detector will check if a shortened URL already exists in the `short_to_long` dictionary.
* If a collision is detected, the detector will generate a new shortened URL by appending a counter value to the original shortened URL.

### Reverse Lookup

* The reverse lookup will use the `short_to_long` dictionary to retrieve the original URL from a shortened URL.

## Implementation

```python
import hashlib
import re
import threading

class URLShortener:
    def __init__(self):
        self.short_to_long = {}
        self.long_to_short = {}
        self.lock = threading.Lock()

    def shorten(self, long_url):
        """Shorten a long URL and return the shortened URL."""
        with self.lock:
            if not self._is_valid_url(long_url):
                raise ValueError("Invalid URL format")

            hash_object = hashlib.sha256(long_url.encode())
            short_url = hash_object.hexdigest()[:6]

            if short_url in self.short_to_long:
                # Collision detected, generate a new shortened URL
                counter = 0
                while True:
                    new_short_url = f"{short_url}{counter}"
                    if new_short_url not in self.short_to_long:
                        short_url = new_short_url
                        break
                    counter += 1

            self.short_to_long[short_url] = long_url
            self.long_to_short[long_url] = short_url

            return short_url

    def lookup(self, short_url):
        """Retrieve the original URL from a shortened URL."""
        with self.lock:
            return self.short_to_long.get(short_url)

    def _is_valid_url(self, url):
        """Check if the input URL conforms to the expected format."""
        pattern = r"^https?://[^\s]+$"
        return bool(re.match(pattern, url))

# Example usage:
shortener = URLShortener()
long_url = "https://www.example.com/very/long/url"
short_url = shortener.shorten(long_url)
print(f"Shortened URL: {short_url}")
print(f"Original URL: {shortener.lookup(short_url)}")
```
