## Introduction

The URL shortening library is designed to provide a simple and efficient way to shorten URLs using a hash-based approach. The library will accept a long URL and return a shortened version, while also supporting reverse lookup from the short URL to the original URL. The library will handle collision detection and include input validation for valid URL formats. Additionally, the library will be designed to be thread-safe for concurrent usage.

## High-Level Design

The URL shortening library will consist of the following components:

1. **URLValidator**: Responsible for validating the input URL format.
2. **HashGenerator**: Responsible for generating a unique hash for the input URL.
3. **ShortURLStore**: Responsible for storing the mapping between the short URL and the original URL.
4. **CollisionDetector**: Responsible for detecting collisions between short URLs.

## Module Decomposition

The library will be decomposed into the following modules:

1. **url_validator.py**: Contains the `URLValidator` class.
2. **hash_generator.py**: Contains the `HashGenerator` class.
3. **short_url_store.py**: Contains the `ShortURLStore` class.
4. **collision_detector.py**: Contains the `CollisionDetector` class.
5. **url_shortener.py**: Contains the `URLShortener` class, which will be the main entry point for the library.

## Interfaces

The library will use the following interfaces:

1. **URLValidator**:
	* `is_valid(url: str) -> bool`: Returns `True` if the input URL is valid, `False` otherwise.
2. **HashGenerator**:
	* `generate_hash(url: str) -> str`: Returns a unique hash for the input URL.
3. **ShortURLStore**:
	* `store_short_url(short_url: str, original_url: str)`: Stores the mapping between the short URL and the original URL.
	* `get_original_url(short_url: str) -> str`: Returns the original URL for the given short URL.
4. **CollisionDetector**:
	* `detect_collision(short_url: str) -> bool`: Returns `True` if a collision is detected, `False` otherwise.

## Data Flow

The data flow for the library will be as follows:

1. The user inputs a long URL.
2. The `URLValidator` validates the input URL format.
3. If the input URL is valid, the `HashGenerator` generates a unique hash for the input URL.
4. The `ShortURLStore` stores the mapping between the short URL and the original URL.
5. If a collision is detected, the `CollisionDetector` will regenerate a new hash until a unique one is found.
6. The `URLShortener` returns the shortened URL.

## Technology Choices

The library will be implemented using Python 3.10 and will utilize the following technologies:

1. **hashlib**: For generating hashes.
2. **threading**: For ensuring thread safety.
3. **unittest**: For unit testing.

import threading

class ShortURLStore:
    def __init__(self):
        self.store = {}
        self.lock = threading.Lock()

    def store_short_url(self, short_url: str, original_url: str):
        with self.lock:
            self.store[short_url] = original_url

    def get_original_url(self, short_url: str) -> str:
        with self.lock:

### url_shortener.py
```python
from url_validator import URLValidator
from hash_generator import HashGenerator
from short_url_store import ShortURLStore
from collision_detector import CollisionDetector

class URLShortener:
    def __init__(self):
        self.validator = URLValidator()
        self.generator = HashGenerator()
        self.store = ShortURLStore()
        self.detector = CollisionDetector()

    def shorten_url(self, url: str) -> str:
        if not self.validator.is_valid(url):
            raise ValueError("Invalid URL format")

        short_url = self.generator.generate_hash(url)

        while self.detector.detect_collision(short_url, self.store.store):
            short_url = self.generator.generate_hash(url)

        self.store.store_short_url(short_url, url)

        return f"http://short.url/{short_url}"

    def get_original_url(self, short_url: str) -> str:
        return self.store.get_original_url(short_url)
```

import sqlite3
import threading
import logging

class ShortURLStore:
    def __init__(self, db_name: str):
        self.conn = sqlite3.connect(db_name)
        self.cursor = self.conn.cursor()
        self.lock = threading.Lock()
        self.logger = logging.getLogger(__name__)

        self.cursor.execute("""
            CREATE TABLE IF NOT EXISTS short_urls (
                short_url TEXT PRIMARY KEY,
                original_url TEXT
            )
        """)

        self.conn.commit()

    def store_short_url(self, short_url: str, original_url: str):
        with self.lock:
            try:
                self.cursor.execute("INSERT INTO short_urls VALUES (?, ?)", (short_url, original_url))
                self.conn.commit()
            except Exception as e:
                self.logger.error(f"Error storing short URL: {e}")

    def get_original_url(self, short_url: str) -> str:
        with self.lock:
            try:
                self.cursor.execute("SELECT original_url FROM short_urls WHERE short_url = ?", (short_url,))
                result = self.cursor.fetchone()
                return result[0] if result else None
            except Exception as e:
                self.logger.error(f"Error retrieving original URL: {e}")
                return None

    new_short_url = shortener.shorten_url("https://www.example.com")
