## Introduction

The in-memory task queue is a critical component of the overall project, enabling efficient task management and execution. This document outlines the high-level design and architecture of the task queue, ensuring seamless integration with other project components.

## Module Decomposition

The task queue will consist of the following modules:

1. **Task**: Represents a single task with associated metadata (priority, status, retries).
2. **TaskQueue**: Manages the task queue, providing methods for adding tasks, consuming tasks, and tracking task status.
3. **Worker**: Responsible for consuming tasks from the task queue and executing them.
4. **RetryPolicy**: Defines the retry strategy for failed tasks.

## Interfaces

The following interfaces will be used to facilitate communication between modules:

1. **TaskQueueInterface**: Defines methods for adding tasks, consuming tasks, and tracking task status.
2. **WorkerInterface**: Specifies methods for executing tasks and reporting task status.

## Data Flow

The data flow between modules is as follows:

1. **Task** -> **TaskQueue**: Tasks are added to the task queue with associated metadata.
2. **TaskQueue** -> **Worker**: The task queue provides tasks to worker threads for execution.
3. **Worker** -> **TaskQueue**: Worker threads report task status (completed, failed) to the task queue.
4. **TaskQueue** -> **RetryPolicy**: The task queue consults the retry policy to determine if a failed task should be retried.

The implementation will utilize the following technologies:

1. **Python**: As the primary programming language.
2. **threading**: For worker thread management.
3. **queue**: For task queue implementation.

import logging
import queue
import threading
import time

# Task class
class Task:
    def __init__(self, id, priority, func, *args, **kwargs):
        self.id = id
        self.priority = priority
        self.func = func
        self.args = args
        self.kwargs = kwargs
        self.status = 'pending'
        self.retries = 0

# TaskQueue class
class TaskQueue:
    def __init__(self, max_retries):
        self.queue = queue.PriorityQueue()
        self.max_retries = max_retries
        self.lock = threading.Lock()

    def add_task(self, task):
        with self.lock:
            self.queue.put((task.priority, task))

    def consume_task(self):
        with self.lock:
            try:
                _, task = self.queue.get(block=False)
                task.status = 'running'
                return task
            except queue.Empty:
                return None

    def report_task_status(self, task, status):
        with self.lock:
            task.status = status
            if status == 'failed' and task.retries < self.max_retries:
                task.retries += 1
                self.add_task(task)

# Worker class
class Worker(threading.Thread):
    def __init__(self, task_queue):
        super().__init__()
        self.task_queue = task_queue

    def run(self):
        while True:
            task = self.task_queue.consume_task()
            if task is None:
                break
            try:
                task.func(*task.args, **task.kwargs)
                self.task_queue.report_task_status(task, 'completed')
            except Exception as e:
                logging.error(f'Task {task.id} failed: {e}')
                self.task_queue.report_task_status(task, 'failed')

# RetryPolicy class
class RetryPolicy:
    def __init__(self, max_retries):
        self.max_retries = max_retries

    def should_retry(self, task):
        return task.retries < self.max_retries

# Example usage
def example_task(x, y):
    print(f'Example task: {x} + {y} = {x + y}')

task_queue = TaskQueue(max_retries=3)
task = Task(1, 1, example_task, 2, 3)
task_queue.add_task(task)

worker = Worker(task_queue)
worker.start()

# Graceful shutdown
def shutdown():
    worker.join()

import threading
import time
from typing import Callable

# Task class
class Task:
    def __init__(self, id: int, priority: int, func: Callable, *args, **kwargs):
        """
        Initialize a Task object.

        Args:
        id (int): Unique task ID.
        priority (int): Task priority.
        func (Callable): Task function to be executed.
        *args: Variable number of arguments to be passed to the task function.
        **kwargs: Keyword arguments to be passed to the task function.
        """
        self.id = id
        self.priority = priority
        self.func = func
        self.args = args
        self.kwargs = kwargs
        self.status = 'pending'
        self.retries = 0

# TaskQueue class
class TaskQueue:
    def __init__(self, max_retries: int, max_queue_size: int = 100):
        """
        Initialize a TaskQueue object.

        Args:
        max_retries (int): Maximum number of retries for a failed task.
        max_queue_size (int): Maximum size of the task queue (default=100).
        """
        self.queue = queue.PriorityQueue(max_queue_size)
        self.max_retries = max_retries
        self.lock = threading.Lock()
        self.shutdown_event = threading.Event()

    def add_task(self, task: Task):
        """
        Add a task to the task queue.

        Args:
        task (Task): Task object to be added to the queue.
        """
        with self.lock:
            try:
                self.queue.put((task.priority, task), block=False)
            except queue.Full:
                logging.error(f'Task queue is full. Cannot add task {task.id}.')

    def consume_task(self) -> Task:
        """
        Consume a task from the task queue.

        Returns:
        Task: Task object consumed from the queue, or None if the queue is empty.
        """
        with self.lock:
            try:
                _, task = self.queue.get(block=False)
                task.status = 'running'
                return task
            except queue.Empty:
                return None

    def report_task_status(self, task: Task, status: str):
        """
        Report the status of a task.

        Args:
        task (Task): Task object to report the status for.
        status (str): Status of the task (completed, failed).
        """
        with self.lock:
            task.status = status
            if status == 'failed' and task.retries < self.max_retries:
                task.retries += 1
                self.add_task(task)

    def shutdown(self):
        """
        Shutdown the task queue.
        """
        self.shutdown_event.set()

# Worker class
class Worker(threading.Thread):
    def __init__(self, task_queue: TaskQueue):
        """
        Initialize a Worker object.

        Args:
        task_queue (TaskQueue): Task queue to consume tasks from.
        """
        super().__init__()
        self.task_queue = task_queue

    def run(self):
        """
        Run the worker thread.
        """
        while not self.task_queue.shutdown_event.is_set():
            task = self.task_queue.consume_task()
            if task is None:
                time.sleep(0.1)  # Wait for 100ms before checking the queue again
                continue
            try:
                task.func(*task.args, **task.kwargs)
                self.task_queue.report_task_status(task, 'completed')
            except Exception as e:
                logging.error(f'Task {task.id} failed: {e}')
                self.task_queue.report_task_status(task, 'failed')

# RetryPolicy class
class RetryPolicy:
    def __init__(self, max_retries: int):
        """
        Initialize a RetryPolicy object.

        Args:
        max_retries (int): Maximum number of retries for a failed task.
        """
        self.max_retries = max_retries

    def should_retry(self, task: Task) -> bool:
        """
        Determine if a task should be retried.

        Args:
        task (Task): Task object to check for retry.

        Returns:
        bool: True if the task should be retried, False otherwise.
        """
        return task.retries < self.max_retries

import logging
import queue
import threading
import time
from typing import Callable

# Task class
class Task:
    def __init__(self, id: int, priority: int, func: Callable, *args, **kwargs):
        """
        Initialize a Task object.

        Args:
        id (int): Unique task ID.
        priority (int): Task priority.
        func (Callable): Task function to be executed.
        *args: Variable number of arguments to be passed to the task function.
        **kwargs: Keyword arguments to be passed to the task function.
        """
        self.id = id
        self.priority = priority
        self.func = func
        self.args = args
        self.kwargs = kwargs
        self.status = 'pending'
        self.retries = 0

# TaskQueue class
class TaskQueue:
    def __init__(self, max_retries: int, max_queue_size: int = 100):
        """
        Initialize a TaskQueue object.

        Args:
        max_retries (int): Maximum number of retries for a failed task.
        max_queue_size (int): Maximum size of the task queue (default=100).
        """
        self.queue = queue.PriorityQueue(max_queue_size)
        self.max_retries = max_retries
        self.lock = threading.Lock()
        self.shutdown_event = threading.Event()

    def add_task(self, task: Task):
        """
        Add a task to the task queue.

        Args:
        task (Task): Task object to be added to the queue.
        """
        with self.lock:
            try:
                self.queue.put((task.priority, task), block=False)
            except queue.Full:
                logging.error(f'Task queue is full. Cannot add task {task.id}.')

    def consume_task(self) -> Task:
        """
        Consume a task from the task queue.

        Returns:
        Task: Task object consumed from the queue, or None if the queue is empty.
        """
        with self.lock:
            try:
                _, task = self.queue.get(block=False)
                task.status = 'running'
                return task
            except queue.Empty:
                return None

    def report_task_status(self, task: Task, status: str):
        """
        Report the status of a task.

        Args:
        task (Task): Task object to report the status for.
        status (str): Status of the task (completed, failed).
        """
        with self.lock:
            task.status = status
            if status == 'failed' and task.retries < self.max_retries:
                task.retries += 1
                self.add_task(task)

    def shutdown(self):
        """
        Shutdown the task queue.
        """
        self.shutdown_event.set()

# Worker class
class Worker(threading.Thread):
    def __init__(self, task_queue: TaskQueue):
        """
        Initialize a Worker object.

        Args:
        task_queue (TaskQueue): Task queue to consume tasks from.
        """
        super().__init__()
        self.task_queue = task_queue

    def run(self):
        """
        Run the worker thread.
        """
        while not self.task_queue.shutdown_event.is_set():
            task = self.task_queue.consume_task()
            if task is None:
                time.sleep(0.1)  # Wait for 100ms before checking the queue again
                continue
            try:
                task.func(*task.args, **task.kwargs)
                self.task_queue.report_task_status(task, 'completed')
            except Exception as e:

                self.task_queue.report_task_status(task, 'failed')

# Task class
class Task:
    def __init__(self, id: int, priority: int, func: Callable, *args, **kwargs):
        """
        Initialize a Task object.

        Args:
        id (int): Unique task ID.
        priority (int): Task priority.
        func (Callable): Task function to be executed.
        *args: Variable number of arguments to be passed to the task function.
        **kwargs: Keyword arguments to be passed to the task function.
        """
        self.id = id
        self.priority = priority
        self.func = func
        self.args = args
        self.kwargs = kwargs
        self.status = 'pending'
        self.retries = 0

# TaskQueue class
class TaskQueue:
    def __init__(self, max_retries: int, max_queue_size: int = 100):
        """
        Initialize a TaskQueue object.

        Args:
        max_retries (int): Maximum number of retries for a failed task.
        max_queue_size (int): Maximum size of the task queue (default=100).
        """
        self.queue = queue.PriorityQueue(max_queue_size)
        self.max_retries = max_retries
        self.lock = threading.Lock()
        self.shutdown_event = threading.Event()

    def add_task(self, task: Task):
        """
        Add a task to the task queue.

        Args:
        task (Task): Task object to be added to the queue.
        """
        with self.lock:
            try:
                self.queue.put((task.priority, task), block=False)
            except queue.Full:
                logging.error(f'Task queue is full. Cannot add task {task.id}.')

    def consume_task(self) -> Task:
        """
        Consume a task from the task queue.

        Returns:
        Task: Task object consumed from the queue, or None if the queue is empty.
        """
        with self.lock:
            try:
                _, task = self.queue.get(block=False)
                task.status = 'running'
                return task
            except queue.Empty:
                return None

        """
        Report the status of a task.

        Args:
        task (Task): Task object to report the status for.
        status (str): Status of the task (completed, failed).

        with self.lock:
            task.status = status
            if status == 'failed' and task.retries < self.max_retries:
                task.retries += 1
                self.add_task(task)

    def shutdown(self):
        """
        Shutdown the task queue.
        """
        self.shutdown_event.set()

from typing import Callable

# Task class
class Task:
    def __init__(self, id: int, priority: int, func: Callable, *args, **kwargs):
        """
        Initialize a Task object.

        Args:
        id (int): Unique task ID.
        priority (int): Task priority.
        func (Callable): Task function to be executed.
        *args: Variable number of arguments to be passed to the task function.
        **kwargs: Keyword arguments to be passed to the task function.
        """
        self.id = id
        self.priority = priority
        self.func = func
        self.args = args
        self.kwargs = kwargs
        self.status = 'pending'
        self.retries = 0

# TaskQueue class
class TaskQueue:
    def __init__(self, max_retries: int, max_queue_size: int = 100):
        """
        Initialize a TaskQueue object.

        Args:
        max_retries (int): Maximum number of retries for a failed task.
        max_queue_size (int): Maximum size of the task queue (default=100).
        """
        self.queue = queue.PriorityQueue(max_queue_size)
        self.max_retries = max_retries
        self.lock = threading.Lock()
        self.shutdown_event = threading.Event()

    def add_task(self, task: Task):
        """
        Add a task to the task queue.

        Args:
        task (Task): Task object to be added to the queue.
        """
        with self.lock:
            try:
                self.queue.put((task.priority, task), block=False)
            except queue.Full:
                logging.error(f'Task queue is full. Cannot add task {task.id}.')

    def consume_task(self) -> Task:
        """
        Consume a task from the task queue.

        Returns:
        Task: Task object consumed from the queue, or None if the queue is empty.
        """
        with self.lock:
            try:
                _, task = self.queue.get(block=False)
                task.status = 'running'
                return task
            except queue.Empty:
                return None

    def report_task_status(self, task: Task, status: str):
        """
        Report the status of a task.

        Args:
        task (Task): Task object to report the status for.
        status (str): Status of the task (completed, failed).
        """
        with self.lock:
            task.status = status
            if status == 'failed' and task.retries < self.max_retries:
                task.retries += 1
                self.add_task(task)

    def shutdown(self):
        """
        Shutdown the task queue.
        """
        self.shutdown_event.set()

# Worker class
class Worker(threading.Thread):
    def __init__(self, task_queue: TaskQueue):
        """
        Initialize a Worker object.

        Args:
        task_queue (TaskQueue): Task queue to consume tasks from.
        """
        super().__init__()
        self.task_queue = task_queue

    def run(self):
        """
        Run the worker thread.
        """
        while not self.task_queue.shutdown_event.is_set():
            task = self.task_queue.consume_task()
            if task is None:
                time.sleep(0.1)  # Wait for 100ms before checking the queue again
                continue
            try:
                task.func(*task.args, **task.kwargs)
                self.task_queue.report_task_status(task, 'completed')
            except Exception as e:
                logging.error(f'Task {task.id} failed: {e}')
                self.task_queue.report_task_status(task, 'failed')

        Args:
        task (Task): Task object to check for retry.
