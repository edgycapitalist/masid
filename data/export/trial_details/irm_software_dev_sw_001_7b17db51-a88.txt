**Architecture Document: URL Shortening Library**
=====================================================

**Overview**
------------

The URL shortening library is designed to provide a hash-based approach for shortening URLs without relying on external services. The library will support reverse lookup, collision detection, input validation, and thread safety for concurrent usage.

**Module Decomposition**
----------------------

The library will consist of the following modules:

*   `url_shortener`: This module will contain the main logic for shortening and expanding URLs.
*   `hash_generator`: This module will generate a unique hash for each URL.
*   `url_validator`: This module will validate the input URL format.
*   `collision_detector`: This module will detect and handle collisions.
*   `thread_safe_cache`: This module will provide a thread-safe cache for storing the URL mappings.

**Interfaces**
-------------

The library will provide the following interfaces:

*   `shorten_url(long_url: str) -> str`: This method will shorten a given long URL and return the shortened version.
*   `expand_url(short_url: str) -> str`: This method will expand a given short URL and return the original long URL.

**Data Flow**
-------------

The data flow of the library will be as follows:

1.  The `shorten_url` method will receive a long URL as input and validate its format using the `url_validator` module.
2.  If the input URL is valid, the `hash_generator` module will generate a unique hash for the URL.
3.  The `collision_detector` module will check for collisions and handle them if necessary.
4.  The shortened URL will be stored in the `thread_safe_cache` along with the original long URL.
5.  The `expand_url` method will receive a short URL as input and retrieve the corresponding long URL from the `thread_safe_cache`.

**Technology Choices**
--------------------

The library will be implemented in Python 3.x, and the following technologies will be used:

*   `hashlib` library for generating hashes
*   `threading` library for thread safety
*   `urllib.parse` library for URL parsing and validation

**Implementation Code**
---------------------

```python
import hashlib
import threading
from urllib.parse import urlparse

class URLShortener:
    def __init__(self):
        self.cache = {}
        self.lock = threading.Lock()

    def shorten_url(self, long_url: str) -> str:
        """Shorten a given long URL and return the shortened version."""
        if not self._is_valid_url(long_url):
            raise ValueError("Invalid URL format")

        hash_object = hashlib.sha256(long_url.encode())
        short_url = hash_object.hexdigest()[:6]

        with self.lock:
            if short_url in self.cache:
                # Handle collision
                short_url += str(len(self.cache))

            self.cache[short_url] = long_url

        return f"http://short.url/{short_url}"

    def expand_url(self, short_url: str) -> str:
        """Expand a given short URL and return the original long URL."""
        short_url = short_url.split("/")[-1]
        with self.lock:
            return self.cache.get(short_url, None)

    def _is_valid_url(self, url: str) -> bool:
        """Validate the input URL format."""
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

# Example usage
shortener = URLShortener()
long_url = "https://www.example.com/very/long/url"
short_url = shortener.shorten_url(long_url)
print(f"Short URL: {short_url}")
print(f"Expanded URL: {shortener.expand_url(short_url)}")
```

```python
import hashlib
```
```python
import logging
```
```python
import redis
import threading
from urllib.parse import urlparse

class URLShortener:
    def __init__(self, redis_host='localhost', redis_port=6379):
        self.redis_client = redis.Redis(host=redis_host, port=redis_port, db=0)
        self.lock = threading.Lock()
        self.logger = logging.getLogger(__name__)

    def shorten_url(self, long_url: str) -> str:
        """Shorten a given long URL and return the shortened version."""
        if not self._is_valid_url(long_url):
            self.logger.error("Invalid URL format")
            raise ValueError("Invalid URL format")

        hash_object = hashlib.blake2b(long_url.encode())
        short_url = hash_object.hexdigest()[:6]

        with self.lock:
            if self.redis_client.exists(short_url):
                # Handle collision
                short_url += str(self.redis_client.incr('collision_counter'))
```
