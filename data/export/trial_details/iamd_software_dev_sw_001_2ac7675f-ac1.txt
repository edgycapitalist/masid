**Architecture Document: URL Shortening Library**

### Introduction
The URL shortening library is designed to provide a simple and efficient way to shorten URLs using a hash-based approach. The library will support reverse lookup, collision detection, input validation, and thread safety for concurrent usage.

### System Components
The library will consist of the following components:

1. **URL Shortener**: Responsible for shortening long URLs using a hash-based approach.
2. **URL Mapper**: Maintains a mapping of short URLs to their corresponding original URLs.
3. **Collision Detector**: Detects and resolves collisions that may occur during the shortening process.
4. **Input Validator**: Verifies the validity of the input URL format.
5. **Thread Safety Manager**: Ensures thread safety for concurrent usage.

### System Flow
The following is a high-level overview of the system flow:

1. The user provides a long URL to be shortened.
2. The input validator checks the validity of the input URL format.
3. If the input URL is valid, the URL shortener generates a shortened URL using a hash-based approach.
4. The collision detector checks for potential collisions with existing shortened URLs.
5. If a collision is detected, the collision detector resolves it by generating a new shortened URL.
6. The URL mapper stores the mapping between the shortened URL and the original URL.
7. The user can retrieve the original URL by providing the shortened URL.

### Implementation Details
The library will be implemented in Python, using the following technologies:

* **Hashing Algorithm**: SHA-256 will be used as the hashing algorithm to generate shortened URLs.
* **Data Storage**: A dictionary will be used to store the mapping between shortened URLs and original URLs.
* **Thread Safety**: The `threading` module will be used to ensure thread safety for concurrent usage.

### Code Implementation
```python
import hashlib
import threading
from urllib.parse import urlparse

class URLShortener:
    def __init__(self):
        self.url_mapper = {}
        self.lock = threading.Lock()

    def shorten_url(self, long_url):
        # Input validation
        if not self.is_valid_url(long_url):
            raise ValueError("Invalid URL format")

        # Generate shortened URL using hash-based approach
        shortened_url = self.generate_shortened_url(long_url)

        # Check for potential collisions
        if self.is_collision(shortened_url):
            # Resolve collision by generating a new shortened URL
            shortened_url = self.resolve_collision(shortened_url)

        # Store mapping between shortened URL and original URL
        with self.lock:
            self.url_mapper[shortened_url] = long_url

        return shortened_url

    def is_valid_url(self, url):
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

    def generate_shortened_url(self, long_url):
        # Use SHA-256 hashing algorithm to generate shortened URL
        hash_object = hashlib.sha256(long_url.encode())
        return hash_object.hexdigest()[:6]

    def is_collision(self, shortened_url):
        return shortened_url in self.url_mapper

    def resolve_collision(self, shortened_url):
        # Generate a new shortened URL by appending a random string
        import random
        random_string = str(random.randint(0, 1000000))
        return shortened_url + random_string

    def get_original_url(self, shortened_url):
        with self.lock:
            return self.url_mapper.get(shortened_url)

def main():
    url_shortener = URLShortener()
    long_url = "https://www.example.com"
    shortened_url = url_shortener.shorten_url(long_url)
    print(f"Shortened URL: {shortened_url}")
    print(f"Original URL: {url_shortener.get_original_url(shortened_url)}")

if __name__ == "__main__":
    main()
```

```python
    def __init__(self):
        self.url_mapper = {}
        self.lock = threading.Lock()
        self.counter = 0

    def shorten_url(self, long_url):
        # Input validation
        if not self.is_valid_url(long_url):

            shortened_url = self.resolve_collision(shortened_url)
```
