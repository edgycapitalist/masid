**Architecture Document: In-Memory Task Queue**
=============================================

### Overview

The in-memory task queue is designed to manage tasks with priorities, allowing worker threads to consume tasks while tracking task status and providing task retry with configurable max retries. The system will also support graceful shutdown.

### Components

1. **Task**: Represents a unit of work with a unique ID, priority, and retry count.
2. **TaskQueue**: Manages tasks, providing methods for adding tasks, consuming tasks, and tracking task status.
3. **Worker**: Consumes tasks from the task queue, executing tasks and updating task status.
4. **TaskStatus**: Enumerates task status (pending, running, completed, failed).

### Design

#### Task Queue

* The task queue will be implemented as a priority queue, using a data structure that supports efficient insertion and removal of tasks based on priority.
* Tasks will be stored in a dictionary, mapping task IDs to task objects.
* Task status will be tracked using a separate dictionary, mapping task IDs to task status.

#### Worker

* Workers will be implemented as threads, consuming tasks from the task queue and executing tasks.
* Workers will update task status after executing tasks.
* Workers will be designed to handle task failures, retrying tasks with configurable max retries.

#### Task Retry

* Task retry will be implemented using a separate thread, responsible for retrying failed tasks.
* The retry thread will use a timer to schedule retries, ensuring that tasks are retried after a configurable delay.

#### Shutdown

* The system will support graceful shutdown, allowing workers to complete executing tasks before shutting down.
* The shutdown process will involve stopping the task queue, stopping workers, and stopping the retry thread.

### Interfaces

#### TaskQueue

* `add_task(task)`: Adds a task to the task queue.
* `consume_task()`: Consumes a task from the task queue.
* `update_task_status(task_id, status)`: Updates the status of a task.
* `get_task_status(task_id)`: Returns the status of a task.

#### Worker

* `execute_task(task)`: Executes a task.
* `update_task_status(task_id, status)`: Updates the status of a task.

### Data Flow

1. Tasks are added to the task queue using the `add_task` method.
2. Workers consume tasks from the task queue using the `consume_task` method.
3. Workers execute tasks using the `execute_task` method.
4. Workers update task status using the `update_task_status` method.
5. The task queue tracks task status using the `get_task_status` method.

```python
import threading
import queue
import enum
import time

class TaskStatus(enum.Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id, priority, retry_count):
        self.task_id = task_id
        self.priority = priority
        self.retry_count = retry_count

class TaskQueue:
    def __init__(self):
        self.tasks = {}
        self.task_status = {}
        self.queue = queue.PriorityQueue()

    def add_task(self, task):
        self.tasks[task.task_id] = task
        self.task_status[task.task_id] = TaskStatus.PENDING
        self.queue.put((task.priority, task.task_id))

    def consume_task(self):
        if self.queue.empty():
            return None
        _, task_id = self.queue.get()
        task = self.tasks[task_id]
        self.task_status[task_id] = TaskStatus.RUNNING
        return task

    def update_task_status(self, task_id, status):
        self.task_status[task_id] = status

    def get_task_status(self, task_id):
        return self.task_status.get(task_id)

class Worker(threading.Thread):
    def __init__(self, task_queue):
        super().__init__()
        self.task_queue = task_queue

    def execute_task(self, task):
        try:
            # simulate task execution
            time.sleep(1)
            self.task_queue.update_task_status(task.task_id, TaskStatus.COMPLETED)
        except Exception as e:
            self.task_queue.update_task_status(task.task_id, TaskStatus.FAILED)
            # retry task if necessary
            if task.retry_count > 0:
                task.retry_count -= 1
                self.task_queue.add_task(task)

    def run(self):
        while True:
            task = self.task_queue.consume_task()
            if task is None:
                break
            self.execute_task(task)

class RetryThread(threading.Thread):
    def __init__(self, task_queue):
        super().__init__()
        self.task_queue = task_queue

    def run(self):
        while True:
            for task_id, status in self.task_queue.task_status.items():
                if status == TaskStatus.FAILED:
                    task = self.task_queue.tasks[task_id]
                    # retry task after a delay
                    time.sleep(5)
                    self.task_queue.add_task(task)
            time.sleep(1)

def main():
    task_queue = TaskQueue()
    worker = Worker(task_queue)
    retry_thread = RetryThread(task_queue)

    # add tasks to the task queue
    task_queue.add_task(Task(1, 1, 3))
    task_queue.add_task(Task(2, 2, 2))
    task_queue.add_task(Task(3, 3, 1))

    # start worker and retry thread
    worker.start()
    retry_thread.start()

    # shutdown after 10 seconds
    time.sleep(10)
    worker.join()
    retry_thread.join()

if __name__ == '__main__':
    main()

import threading
import queue
import enum
import time
import logging

logging.basicConfig(level=logging.INFO)

class TaskStatus(enum.Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id, priority, retry_count):
        self.task_id = task_id
        self.priority = priority
        self.retry_count = retry_count

class TaskQueue:
    def __init__(self):
        self.tasks = {}
        self.task_status = {}
        self.queue = queue.PriorityQueue()
        self.stop_event = threading.Event()

    def add_task(self, task):
        self.tasks[task.task_id] = task
        self.task_status[task.task_id] = TaskStatus.PENDING
        self.queue.put((task.priority, task.task_id))

    def consume_task(self):
        if self.stop_event.is_set():
            return None
        if self.queue.empty():
            return None
        _, task_id = self.queue.get()
        task = self.tasks[task_id]
        self.task_status[task_id] = TaskStatus.RUNNING
        return task

    def update_task_status(self, task_id, status):
        self.task_status[task_id] = status

    def get_task_status(self, task_id):
        return self.task_status.get(task_id)

    def stop(self):
        self.stop_event.set()

class Worker(threading.Thread):
    def __init__(self, task_queue):
        super().__init__()
        self.task_queue = task_queue

    def execute_task(self, task):
        try:
            # simulate task execution
            time.sleep(1)
            self.task_queue.update_task_status(task.task_id, TaskStatus.COMPLETED)
        except Exception as e:
            logging.error(f"Task {task.task_id} failed: {e}")
            self.task_queue.update_task_status(task.task_id, TaskStatus.FAILED)
            # retry task if necessary
            if task.retry_count > 0:
                task.retry_count -= 1
                self.task_queue.add_task(task)

    def run(self):
        while True:
            task = self.task_queue.consume_task()
            if task is None:
                break
            self.execute_task(task)

        super().__init__()
        self.task_queue = task_queue

import threading
import queue
import enum
import time
import logging

logging.basicConfig(level=logging.INFO)

class TaskStatus(enum.Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id, priority, retry_count):
        self.task_id = task_id
        self.priority = priority
        self.retry_count = retry_count

class TaskQueue:
    def __init__(self):
        self.tasks = {}
        self.task_status = {}
        self.queue = queue.PriorityQueue()
        self.stop_event = threading.Event()
        self.lock = threading.Lock()

    def add_task(self, task):
        with self.lock:
            self.tasks[task.task_id] = task
            self.task_status[task.task_id] = TaskStatus.PENDING
            self.queue.put((task.priority, task.task_id))

    def consume_task(self):
        if self.stop_event.is_set():
            return None
        if self.queue.empty():
            return None
        with self.lock:
            _, task_id = self.queue.get()
            task = self.tasks[task_id]
            self.task_status[task_id] = TaskStatus.RUNNING
            return task

    def update_task_status(self, task_id, status):
        with self.lock:
            self.task_status[task_id] = status

    def get_task_status(self, task_id):
        with self.lock:
            return self.task_status.get(task_id)

    def stop(self):
        self.stop_event.set()

class Worker(threading.Thread):
    def __init__(self, task_queue):
        super().__init__()
        self.task_queue = task_queue

    def execute_task(self, task):
        try:
            # simulate task execution
            time.sleep(1)
            self.task_queue.update_task_status(task.task_id, TaskStatus.COMPLETED)
        except Exception as e:
            logging.error(f"Task {task.task_id} failed: {e}")
            self.task_queue.update_task_status(task.task_id, TaskStatus.FAILED)
            # retry task if necessary
            if task.retry_count > 0:
                task.retry_count -= 1
                self.task_queue.add_task(task)

    def run(self):
        while True:
            task = self.task_queue.consume_task()
            if task is None:
                break
            self.execute_task(task)

class RetryThread(threading.Thread):

        super().__init__()
        self.task_queue = task_queue

class TaskQueue:
    def __init__(self, max_size=1000):
        self.tasks = {}
        self.task_status = {}
        self.queue = queue.PriorityQueue(max_size)
        self.stop_event = threading.Event()
        self.lock = threading.Lock()

    def add_task(self, task):
        if len(self.tasks) >= self.queue.max_size:
            raise Exception("Task queue is full")
        self.tasks[task.task_id] = task
        self.task_status[task.task_id] = TaskStatus.PENDING
        self.queue.put((task.priority, task.task_id))

    def consume_task(self):
        if self.stop_event.is_set():
            return None
        if self.queue.empty():
            return None
        with self.lock:
            _, task_id = self.queue.get()
            task = self.tasks[task_id]
            self.task_status[task_id] = TaskStatus.RUNNING
            return task

    def update_task_status(self, task_id, status):
        with self.lock:
            self.task_status[task_id] = status

    def get_task_status(self, task_id):
        with self.lock:
            return self.task_status.get(task_id)

    def stop(self):
        self.stop_event.set()

class Worker:
    def __init__(self, task_queue, task_timeout=60):
        self.task_queue = task_queue
        self.task_timeout = task_timeout

    def execute_task(self, task):
        try:
            # simulate task execution
            time.sleep(1)
            self.task_queue.update_task_status(task.task_id, TaskStatus.COMPLETED)
        except Exception as e:
            logging.error(f"Task {task.task_id} failed: {e}")
            self.task_queue.update_task_status(task.task_id, TaskStatus.FAILED)
            # retry task if necessary
            if task.retry_count > 0:
                task.retry_count -= 1
                self.task_queue.add_task(task)

    def run(self):
        while True:
            task = self.task_queue.consume_task()
            if task is None:
                break
            self.execute_task(task)

class RetryThread:
    def __init__(self, task_queue, max_retry_delay=300):
        self.task_queue = task_queue
        self.max_retry_delay = max_retry_delay

    def run(self):
        while True:
            if self.task_queue.stop_event.is_set():
                break
            with self.task_queue.lock:
                for task_id, status in self.task_queue.task_status.items():
                    if status == TaskStatus.FAILED:
                        task = self.task_queue.tasks[task_id]
                        # retry task after a delay
                        delay = min(self.max_retry_delay, task.retry_count * 60)
                        time.sleep(delay)
                        self.task_queue.add_task(task)
            time.sleep(1)
