### Overview

The rate limiter module is designed to manage the frequency of requests to a system, preventing abuse and ensuring fair usage. The module implements the token bucket algorithm, allowing for configurable rates and burst sizes. Multiple named rate limiters can be created, enabling per-user or per-endpoint rate limiting.

### Module Components

*   **RateLimiter**: The core class responsible for managing the token bucket and enforcing rate limits.
*   **TokenBucket**: A helper class implementing the token bucket algorithm.
*   **RateLimiterDecorator**: A decorator for easily applying rate limiting to functions.

### Interfaces

*   **RateLimiter**:
    *   `__init__(name, rate, burst_size)`: Initializes a new rate limiter with the given name, rate, and burst size.
    *   `allow()`: Checks if a request is allowed based on the current token bucket state.
    *   `get_tokens()`: Returns the current number of tokens in the bucket.
*   **TokenBucket**:
    *   `__init__(rate, burst_size)`: Initializes a new token bucket with the given rate and burst size.
    *   `add_tokens(tokens)`: Adds tokens to the bucket.
    *   `consume_tokens(tokens)`: Consumes tokens from the bucket.
*   **RateLimiterDecorator**:
    *   `__init__(rate_limiter)`: Initializes a new decorator with the given rate limiter.
    *   `__call__(func)`: Decorates the given function with rate limiting.

### Data Flow

1.  A new rate limiter is created with a name, rate, and burst size.
2.  The rate limiter is used to decorate a function, applying rate limiting to the function.
3.  When the decorated function is called, the rate limiter checks if the request is allowed based on the current token bucket state.
4.  If the request is allowed, the rate limiter consumes tokens from the bucket and proceeds with the function call.
5.  If the request is not allowed, the rate limiter raises a `RateLimitExceeded` exception.

import threading
import time
from functools import wraps

class TokenBucket:
    def __init__(self, rate, burst_size):
        """
        Initializes a new token bucket with the given rate and burst size.

        Args:
            rate (float): The rate at which tokens are added to the bucket (tokens per second).
            burst_size (int): The maximum number of tokens in the bucket.
        """
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()

    def add_tokens(self):
        """
        Adds tokens to the bucket based on the elapsed time since the last update.
        """
        now = time.time()
        elapsed = now - self.last_update
        self.last_update = now
        self.tokens = min(self.burst_size, self.tokens + elapsed * self.rate)

    def consume_tokens(self, tokens):
        """
        Consumes tokens from the bucket.

        Args:
            tokens (int): The number of tokens to consume.

        Returns:
            bool: True if the tokens were consumed, False if not enough tokens are available.
        """
        if self.tokens >= tokens:
            self.tokens -= tokens
            return True
        return False

class RateLimiter:
    def __init__(self, name, rate, burst_size):
        """
        Initializes a new rate limiter with the given name, rate, and burst size.

        Args:
            name (str): The name of the rate limiter.
            rate (float): The rate at which tokens are added to the bucket (tokens per second).
            burst_size (int): The maximum number of tokens in the bucket.
        """
        self.name = name
        self.token_bucket = TokenBucket(rate, burst_size)
        self.lock = threading.Lock()

    def allow(self):
        """
        Checks if a request is allowed based on the current token bucket state.

        Returns:
            bool: True if the request is allowed, False if not.
        """
        with self.lock:
            self.token_bucket.add_tokens()
            return self.token_bucket.consume_tokens(1)

    def get_tokens(self):
        """
        Returns the current number of tokens in the bucket.

        Returns:
            int: The current number of tokens.
        """
        with self.lock:
            self.token_bucket.add_tokens()
            return int(self.token_bucket.tokens)

class RateLimiterDecorator:
    def __init__(self, rate_limiter):
        """
        Initializes a new decorator with the given rate limiter.

        Args:
            rate_limiter (RateLimiter): The rate limiter to use for decoration.
        """
        self.rate_limiter = rate_limiter

    def __call__(self, func):
        """
        Decorates the given function with rate limiting.

        Args:
            func (callable): The function to decorate.

        Returns:
            callable: The decorated function.
        """
        @wraps(func)
        def wrapper(*args, **kwargs):
            if not self.rate_limiter.allow():
                raise RateLimitExceeded(self.rate_limiter.name)
            return func(*args, **kwargs)
        return wrapper

class RateLimitExceeded(Exception):
    def __init__(self, name):
        """
        Initializes a new rate limit exceeded exception.

        Args:
            name (str): The name of the rate limiter that exceeded its limit.
        """
        self.name = name
        super().__init__(f"Rate limit exceeded for {name}")

# Example usage:
rate_limiter = RateLimiter("example", 5, 10)
decorator = RateLimiterDecorator(rate_limiter)

@decorator
def example_function():
    print("Example function called")

example_function()

import time
import threading

class RateLimitExceeded(Exception):
    pass

class RateLimiter:
    def __init__(self, rate, burst_size):
        """
        Initializes a new rate limiter with the given rate and burst size.

        Args:
            rate (float): The rate at which tokens are added to the bucket (tokens per second).
            burst_size (int): The maximum number of tokens in the bucket.
        """
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()
        self.lock = threading.Lock()

    def allow(self):
        """
        Checks if a request is allowed based on the current token bucket state.

        Returns:
            bool: True if the request is allowed, False if not.
        """
        with self.lock:
            current_time = time.time()
            elapsed_time = current_time - self.last_update
            self.last_update = current_time
            self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
            if self.tokens >= 1:
                self.tokens -= 1
                return True
            return False

    def decorator(func):
        def wrapper(*args, **kwargs):
            if not limiter.allow():
                raise RateLimitExceeded("Rate limit exceeded")
            return func(*args, **kwargs)
        return wrapper
    return decorator
