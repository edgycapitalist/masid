**Introduction**

The rate limiter module is designed to limit the number of requests that can be made to a system within a certain time frame. This module uses the token bucket algorithm for rate limiting and supports multiple named rate limiters. The module is designed to be thread-safe and provides a decorator for easy application to functions.

**Module Components**

1. **Token Bucket Algorithm**: The token bucket algorithm is used to implement rate limiting. The algorithm uses a bucket that is filled with tokens at a certain rate. Each request consumes one token from the bucket. If the bucket is empty, the request is blocked until a token is available.
2. **Rate Limiter Class**: The `RateLimiter` class represents a single rate limiter. It has methods to consume tokens and check if a request is allowed.
3. **Rate Limiter Manager**: The `RateLimiterManager` class manages multiple named rate limiters. It provides methods to create, retrieve, and delete rate limiters.
4. **Decorator**: The `rate_limited` decorator is used to apply rate limiting to functions.

3. **Decorator Interface**: The `rate_limited` decorator has the following interface:

import time
from functools import wraps

class RateLimiter:
    def __init__(self, rate, burst_size):
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()

            current_time = time.time()
            elapsed_time = current_time - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
            self.last_update = current_time
            if self.tokens < 1:

            self.tokens -= 1
            return True

        with self.lock:
            current_time = time.time()

    def __init__(self):
        self.rate_limiters = {}
        self.lock = threading.Lock()

    def create_rate_limiter(self, name, rate, burst_size):
        with self.lock:
            if name in self.rate_limiters:
                raise ValueError("Rate limiter with name '{}' already exists".format(name))
            self.rate_limiters[name] = RateLimiter(rate, burst_size)

    def get_rate_limiter(self, name):
        with self.lock:
            return self.rate_limiters.get(name)

    def delete_rate_limiter(self, name):
        with self.lock:
            if name not in self.rate_limiters:
