### Introduction

The rate limiter module is designed to support token bucket algorithm for rate limiting. It provides configurable rate and burst size, multiple named rate limiters, a decorator for easy application to functions, and thread-safe operation.

### Module Components

*   **RateLimiter**: The main class responsible for managing rate limiters.
*   **TokenBucket**: A class implementing the token bucket algorithm.
*   **RateLimiterDecorator**: A decorator for easy application of rate limiting to functions.

### Module Interfaces

*   **RateLimiter**:
    *   `__init__(rate, burst_size)`: Initializes a rate limiter with the given rate and burst size.
    *   `acquire_token()`: Acquires a token from the rate limiter.
    *   `release_token()`: Releases a token back to the rate limiter.
*   **TokenBucket**:
    *   `__init__(rate, burst_size)`: Initializes a token bucket with the given rate and burst size.
    *   `acquire_token()`: Acquires a token from the token bucket.
    *   `release_token()`: Releases a token back to the token bucket.
*   **RateLimiterDecorator**:
    *   `__init__(rate_limiter)`: Initializes a decorator with the given rate limiter.
    *   `__call__(func)`: Applies the rate limiter to the given function.

### Data Flow

1.  The `RateLimiter` class is initialized with a rate and burst size.
2.  The `TokenBucket` class is initialized with the same rate and burst size.
3.  When a request is made, the `acquire_token` method is called on the `RateLimiter` instance.
4.  If a token is available, it is acquired and the request is allowed to proceed.
5.  If no token is available, the request is blocked until a token becomes available.
6.  The `release_token` method is called when a request is completed, releasing the token back to the rate limiter.

class TokenBucket:
    def __init__(self, rate, burst_size):
        """
        Initializes a token bucket with the given rate and burst size.

        :param rate: The rate at which tokens are added to the bucket (tokens per second)
        :param burst_size: The maximum number of tokens in the bucket
        """
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()

    def acquire_token(self):
        """
        Acquires a token from the token bucket.

        :return: True if a token is available, False otherwise
        """
        current_time = time.time()
        elapsed_time = current_time - self.last_update
        self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
        self.last_update = current_time

        if self.tokens < 1:
            return False
        self.tokens -= 1
        return True

    def release_token(self):
        """
        Releases a token back to the token bucket.

        :return: None
        """
        current_time = time.time()
        elapsed_time = current_time - self.last_update
        self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
        self.last_update = current_time
        if self.tokens < self.burst_size:
            self.tokens += 1

        Initializes a rate limiter with the given rate and burst size.

        rate (float): The rate at which tokens are added to the bucket.
        burst_size (int): The maximum number of tokens in the bucket.
