### Overview

The rate limiter module is designed to provide a flexible and thread-safe way to limit the rate at which requests are made to a system. It uses the token bucket algorithm to manage the rate limiting and supports multiple named rate limiters.

### Components

*   **RateLimiter**: The main class that implements the token bucket algorithm and provides methods for checking if a request is allowed and for getting the current rate limiter status.
*   **RateLimiterFactory**: A factory class that creates and manages multiple named rate limiters.
*   **RateLimiterDecorator**: A decorator class that provides an easy way to apply rate limiting to functions.

### Design

#### Token Bucket Algorithm

The token bucket algorithm works by adding tokens to a bucket at a constant rate. Each request consumes one token from the bucket. If the bucket is empty, the request is blocked until a token is added to the bucket.

#### RateLimiter Class

The `RateLimiter` class has the following attributes:

*   `rate`: The rate at which tokens are added to the bucket (requests per second).
*   `burst_size`: The maximum number of tokens in the bucket.
*   `tokens`: The current number of tokens in the bucket.
*   `last_update`: The last time the bucket was updated.

The `RateLimiter` class has the following methods:

*   `allow_request()`: Checks if a request is allowed based on the current token count and updates the token count if the request is allowed.
*   `get_status()`: Returns the current rate limiter status, including the number of tokens in the bucket and the last update time.

#### RateLimiterFactory Class

The `RateLimiterFactory` class has the following attributes:

*   `rate_limiters`: A dictionary of named rate limiters.

The `RateLimiterFactory` class has the following methods:

*   `create_rate_limiter(name, rate, burst_size)`: Creates a new named rate limiter with the specified rate and burst size.
*   `get_rate_limiter(name)`: Returns the named rate limiter with the specified name.

#### RateLimiterDecorator Class

The `RateLimiterDecorator` class has the following attributes:

*   `rate_limiter`: The rate limiter instance to use for rate limiting.

The `RateLimiterDecorator` class has the following methods:

*   `__call__(func)`: Decorates the specified function with rate limiting.

import threading
import time
from functools import wraps

class RateLimiter:
    def __init__(self, rate, burst_size):
        """
        Initializes a new RateLimiter instance.

        Args:
            rate (float): The rate at which tokens are added to the bucket (requests per second).
            burst_size (int): The maximum number of tokens in the bucket.
        """
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()
        self.lock = threading.Lock()

    def allow_request(self):
        """
        Checks if a request is allowed based on the current token count and updates the token count if the request is allowed.

        Returns:
            bool: True if the request is allowed, False otherwise.
        """
        with self.lock:
            now = time.time()
            elapsed = now - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed * self.rate)
            self.last_update = now

            if self.tokens < 1:
                return False

            self.tokens -= 1
            return True

    def get_status(self):
        """
        Returns the current rate limiter status, including the number of tokens in the bucket and the last update time.

        Returns:
            dict: The current rate limiter status.
        """
        with self.lock:
            return {
                "tokens": self.tokens,
                "last_update": self.last_update,
            }


class RateLimiterFactory:
    def __init__(self):
        """
        Initializes a new RateLimiterFactory instance.
        """
        self.rate_limiters = {}
        self.lock = threading.Lock()

    def create_rate_limiter(self, name, rate, burst_size):
        """
        Creates a new named rate limiter with the specified rate and burst size.

        Args:
            name (str): The name of the rate limiter.
            rate (float): The rate at which tokens are added to the bucket (requests per second).
            burst_size (int): The maximum number of tokens in the bucket.
        """
        with self.lock:
            if name not in self.rate_limiters:
                self.rate_limiters[name] = RateLimiter(rate, burst_size)

    def get_rate_limiter(self, name):
        """
        Returns the named rate limiter with the specified name.

        Args:
            name (str): The name of the rate limiter.

        Returns:
            RateLimiter: The named rate limiter instance.
        """
        with self.lock:
            return self.rate_limiters.get(name)


def rate_limiter_decorator(rate_limiter):
    """
    Decorates the specified function with rate limiting.

    Args:
        rate_limiter (RateLimiter): The rate limiter instance to use for rate limiting.

    Returns:
        function: The decorated function.
    """
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            if rate_limiter.allow_request():
                return func(*args, **kwargs)
            else:
                raise Exception("Rate limit exceeded")

        return wrapper

    return decorator


# Example usage
if __name__ == "__main__":
    factory = RateLimiterFactory()
    factory.create_rate_limiter("my_rate_limiter", 10, 100)
    rate_limiter = factory.get_rate_limiter("my_rate_limiter")

    @rate_limiter_decorator(rate_limiter)
    def my_function():
        print("Hello, world!")

    my_function()

### Overview

The rate limiter module is designed to provide a flexible and thread-safe way to limit the rate at which requests are made to a system. It uses the token bucket algorithm to manage the rate limiting and supports multiple named rate limiters.

### Components

*   **RateLimiter**: The main class that implements the token bucket algorithm and provides methods for checking if a request is allowed and for getting the current rate limiter status.
*   **RateLimiterFactory**: A factory class that creates and manages multiple named rate limiters.
*   **RateLimiterDecorator**: A decorator class that provides an easy way to apply rate limiting to functions.

### Design

#### Token Bucket Algorithm

The token bucket algorithm works by adding tokens to a bucket at a constant rate. Each request consumes one token from the bucket. If the bucket is empty, the request is blocked until a token is added to the bucket.

#### RateLimiter Class

The `RateLimiter` class has the following attributes:

*   `rate`: The rate at which tokens are added to the bucket (requests per second).
*   `burst_size`: The maximum number of tokens in the bucket.
*   `tokens`: The current number of tokens in the bucket.
*   `last_update`: The last time the bucket was updated.

The `RateLimiter` class has the following methods:

*   `allow_request()`: Checks if a request is allowed based on the current token count and updates the token count if the request is allowed.
*   `get_status()`: Returns the current rate limiter status, including the number of tokens in the bucket and the last update time.

#### RateLimiterFactory Class

The `RateLimiterFactory` class has the following attributes:

*   `rate_limiters`: A dictionary of named rate limiters.

The `RateLimiterFactory` class has the following methods:

*   `create_rate_limiter(name, rate, burst_size)`: Creates a new named rate limiter with the specified rate and burst size.
*   `get_rate_limiter(name)`: Returns the named rate limiter with the specified name.

#### RateLimiterDecorator Class

The `RateLimiterDecorator` class has the following attributes:

*   `rate_limiter`: The rate limiter instance to use for rate limiting.

The `RateLimiterDecorator` class has the following methods:

*   `__call__(func)`: Decorates the specified function with rate limiting.

import threading
import time
from functools import wraps

class RateLimiter:
    def __init__(self, rate, burst_size):
        """
        Initializes a new RateLimiter instance.

        Args:
            rate (float): The rate at which tokens are added to the bucket (requests per second).
            burst_size (int): The maximum number of tokens in the bucket.
        """
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()
        self.lock = threading.Lock()

    def allow_request(self):
        """
        Checks if a request is allowed based on the current token count and updates the token count if the request is allowed.

        Returns:
            bool: True if the request is allowed, False otherwise.
        """
        with self.lock:
            now = time.time()
            elapsed = now - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed * self.rate)
            self.last_update = now

            if self.tokens < 1:
                return False

            self.tokens -= 1
            return True

    def get_status(self):
        """
        Returns the current rate limiter status, including the number of tokens in the bucket and the last update time.

        Returns:
            dict: The current rate limiter status.
        """
        with self.lock:
            return {
                "tokens": self.tokens,
                "last_update": self.last_update,
            }


class RateLimiterFactory:
    def __init__(self):
        """
        Initializes a new RateLimiterFactory instance.
        """
        self.rate_limiters = {}
        self.lock = threading.Lock()

    def create_rate_limiter(self, name, rate, burst_size):
        """
        Creates a new named rate limiter with the specified rate and burst size.

        Args:
            name (str): The name of the rate limiter.
            rate (float): The rate at which tokens are added to the bucket (requests per second).
            burst_size (int): The maximum number of tokens in the bucket.
        """
        with self.lock:
            if name not in self.rate_limiters:
                self.rate_limiters[name] = RateLimiter(rate, burst_size)

    def get_rate_limiter(self, name):
        """
        Returns the named rate limiter with the specified name.

        Args:
            name (str): The name of the rate limiter.

        Returns:
            RateLimiter: The named rate limiter instance.
        """
        with self.lock:
            return self.rate_limiters.get(name)


def rate_limiter_decorator(rate_limiter):
    """
    Decorates the specified function with rate limiting.

    Args:
        rate_limiter (RateLimiter): The rate limiter instance to use for rate limiting.

    Returns:
        function: The decorated function.
    """
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            if rate_limiter.allow_request():
                return func(*args, **kwargs)
            else:
                raise Exception("Rate limit exceeded")

        return wrapper

    return decorator


# Example usage
if __name__ == "__main__":
    factory = RateLimiterFactory()
    factory.create_rate_limiter("my_rate_limiter", 10, 100)
    rate_limiter = factory.get_rate_limiter("my_rate_limiter")

    @rate_limiter_decorator(rate_limiter)
    def my_function():
        print("Hello, world!")

    my_function()

### Design

#### Token Bucket Algorithm

The token bucket algorithm works by adding tokens to a bucket at a constant rate. Each request consumes one token from the bucket. If the bucket is empty, the request is blocked until a token is added to the bucket.

#### RateLimiter Class

The `RateLimiter` class has the following attributes:

*   `rate`: The rate at which tokens are added to the bucket (requests per second).
*   `burst_size`: The maximum number of tokens in the bucket.
*   `tokens`: The current number of tokens in the bucket.
*   `last_update`: The last time the bucket was updated.

The `RateLimiter` class has the following methods:

*   `allow_request()`: Checks if a request is allowed based on the current token count and updates the token count if the request is allowed.
*   `get_status()`: Returns the current rate limiter status, including the number of tokens in the bucket and the last update time.

*   `rate_limiters`: A dictionary of named rate limiters.

The `RateLimiterFactory` class has the following methods:

*   `create_rate_limiter(name, rate, burst_size)`: Creates a new named rate limiter with the specified rate and burst size.
*   `get_rate_limiter(name)`: Returns the named rate limiter with the specified name.

#### RateLimiterDecorator Class

import threading
import time
from functools import wraps

class RateLimiter:
    def __init__(self, rate, burst_size):
        """
        Initializes a new RateLimiter instance.

        Args:
            rate (float): The rate at which tokens are added to the bucket (requests per second).
            burst_size (int): The maximum number of tokens in the bucket.
        """
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()
        self.lock = threading.Lock()

    def allow_request(self):
        """
        Checks if a request is allowed based on the current token count and updates the token count if the request is allowed.

        Returns:
            bool: True if the request is allowed, False otherwise.
        """
        with self.lock:
            now = time.time()
            elapsed = now - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed * self.rate)
            self.last_update = now

            if self.tokens < 1:
                return False

            self.tokens -= 1
            return True

    def get_status(self):
        """
        Returns the current rate limiter status, including the number of tokens in the bucket and the last update time.

        Returns:
            dict: The current rate limiter status.
        """
        with self.lock:
            return {
                "tokens": self.tokens,
                "last_update": self.last_update,
            }


class RateLimiterFactory:
    def __init__(self):
        """
        Initializes a new RateLimiterFactory instance.
        """
        self.rate_limiters = {}
        self.lock = threading.Lock()

    def create_rate_limiter(self, name, rate, burst_size):
        """
        Creates a new named rate limiter with the specified rate and burst size.

        Args:
            name (str): The name of the rate limiter.
            rate (float): The rate at which tokens are added to the bucket (requests per second).
            burst_size (int): The maximum number of tokens in the bucket.
        """
        with self.lock:
            if name not in self.rate_limiters:
                self.rate_limiters[name] = RateLimiter(rate, burst_size)

    def get_rate_limiter(self, name):
        """
        Returns the named rate limiter with the specified name.

        Args:
            name (str): The name of the rate limiter.

        Returns:
            RateLimiter: The named rate limiter instance.
        """
        with self.lock:
            return self.rate_limiters.get(name)


def rate_limiter_decorator(rate_limiter):
    """
    Decorates the specified function with rate limiting.

    Args:
        rate_limiter (RateLimiter): The rate limiter instance to use for rate limiting.

    Returns:
        function: The decorated function.

    return decorator
