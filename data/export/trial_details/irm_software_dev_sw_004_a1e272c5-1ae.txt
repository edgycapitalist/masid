### Overview

The in-memory task queue is designed to manage tasks with priorities, allowing worker threads to consume tasks while tracking their status. The system supports task retry with configurable max retries and provides a mechanism for graceful shutdown.

### Components

1. **Task**: Represents a unit of work with a unique ID, priority, and a callable function.
2. **Task Queue**: An in-memory data structure that stores tasks with their corresponding priorities.
3. **Worker Thread**: Consumes tasks from the task queue, executes them, and updates their status.
4. **Task Status Tracker**: Maintains the status of each task (pending, running, completed, failed).
5. **Retry Mechanism**: Allows tasks to be retried with a configurable maximum number of retries.

### System Design

1. **Task Queue Data Structure**: A priority queue (implemented using a heap) will be used to store tasks. The priority queue will ensure that tasks with higher priorities are executed first.
2. **Task Status Tracking**: A dictionary will be used to store the status of each task, where the key is the task ID and the value is the task status.
3. **Worker Thread**: Worker threads will be implemented using Python's `threading` module. Each worker thread will consume tasks from the task queue, execute them, and update their status.
4. **Retry Mechanism**: A task will be retried if it fails, up to a configurable maximum number of retries. The retry mechanism will be implemented using a decorator that wraps the task function.

### System Flow

1. **Task Addition**: A task is added to the task queue with its corresponding priority.
2. **Worker Thread Consumption**: A worker thread consumes a task from the task queue, executes it, and updates its status.
3. **Task Execution**: The task is executed, and its status is updated to "running".
4. **Task Completion**: If the task completes successfully, its status is updated to "completed". If the task fails, its status is updated to "failed", and it is retried if the maximum number of retries has not been reached.
5. **Retry**: If a task fails and the maximum number of retries has not been reached, the task is retried.

import threading
import heapq
import time
from functools import wraps

class Task:
    def __init__(self, task_id, priority, func, max_retries=3):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.max_retries = max_retries
        self.retries = 0
        self.status = "pending"

class TaskQueue:
    def __init__(self):
        self.queue = []
        self.task_status = {}

    def add_task(self, task):
        heapq.heappush(self.queue, (task.priority, task))
        self.task_status[task.task_id] = task.status

    def consume_task(self):
        if not self.queue:
            return None
        _, task = heapq.heappop(self.queue)
        return task

class WorkerThread(threading.Thread):
    def __init__(self, task_queue):
        super().__init__()
        self.task_queue = task_queue

    def run(self):
        while True:
            task = self.task_queue.consume_task()
            if task is None:
                break
            task.status = "running"
            self.task_queue.task_status[task.task_id] = task.status
            try:
                task.func()
                task.status = "completed"
            except Exception as e:
                task.status = "failed"
                task.retries += 1
                if task.retries < task.max_retries:
                    self.task_queue.add_task(task)
                else:
                    print(f"Task {task.task_id} failed after {task.max_retries} retries")
            finally:
                self.task_queue.task_status[task.task_id] = task.status

def retry(max_retries):
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            task = args[0]
            try:
                return func(*args, **kwargs)
            except Exception as e:
                task.retries += 1
                if task.retries < max_retries:
                    raise
                else:
                    raise Exception(f"Task {task.task_id} failed after {max_retries} retries")
        return wrapper
    return decorator

def main():
    task_queue = TaskQueue()
    worker_thread = WorkerThread(task_queue)

    def example_task(task):
        print(f"Executing task {task.task_id}")
        time.sleep(1)
        raise Exception("Example exception")

    task = Task(1, 1, example_task)
    task_queue.add_task(task)

    worker_thread.start()
    worker_thread.join()

if __name__ == "__main__":
    main()

import threading
import heapq
import time
from functools import wraps

class Task:
    def __init__(self, task_id, priority, func, max_retries=3):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.max_retries = max_retries
        self.retries = 0
        self.status = "pending"

class TaskQueue:
    def __init__(self, max_size=1000):
        self.queue = []
        self.task_status = {}
        self.lock = threading.Lock()
        self.max_size = max_size

    def add_task(self, task):
        with self.lock:
            if len(self.queue) >= self.max_size:
                raise Exception("Task queue is full")
            heapq.heappush(self.queue, (task.priority, task))
            self.task_status[task.task_id] = task.status

    def consume_task(self):
        with self.lock:
            if not self.queue:
                return None
            _, task = heapq.heappop(self.queue)
            return task

class WorkerThread(threading.Thread):
    def __init__(self, task_queue):
        super().__init__()
        self.task_queue = task_queue
        self.stop_event = threading.Event()

    def run(self):
        while not self.stop_event.is_set():
            task = self.task_queue.consume_task()
            if task is None:
                time.sleep(0.1)
                continue
            task.status = "running"
            self.task_queue.task_status[task.task_id] = task.status
            try:
                task.func()
                task.status = "completed"
            except Exception as e:
                task.status = "failed"
                task.retries += 1
                if task.retries < task.max_retries:
                    self.task_queue.add_task(task)
                else:
                    print(f"Task {task.task_id} failed after {task.max_retries} retries")
            finally:
                self.task_queue.task_status[task.task_id] = task.status

    def stop(self):
        self.stop_event.set()

    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            task = args[0]
            try:
                return func(*args, **kwargs)
            except Exception as e:
                task.retries += 1
                if task.retries < max_retries:
                    raise
                else:
                    raise Exception(f"Task {task.task_id} failed after {max_retries} retries")
        return wrapper
    return decorator

*   **WorkerThread**: Represents a worker thread that consumes tasks from the task queue. It has attributes such as `task_queue` and `stop_event`. It provides methods such as `start` and `stop`.

1.  **Task Creation**: A task is created with a `task_id`, `priority`, `func`, and `max_retries`.
