### Overview

The URL shortening library is designed to provide a lightweight, hash-based solution for shortening URLs. The library will utilize a combination of hashing and collision detection to ensure unique shortened URLs.

### Components

1. **URLValidator**: Responsible for validating the input URL format.
2. **URLShortener**: Generates a shortened URL using a hash-based approach.
3. **URLMapper**: Stores the mapping between shortened URLs and original URLs.
4. **CollisionDetector**: Detects and handles collisions between shortened URLs.

### Interfaces

1. **URLShortenerInterface**: Defines the interface for shortening URLs.
2. **URLMapperInterface**: Defines the interface for storing and retrieving URL mappings.

### Data Flow

1. The user provides a long URL to the **URLShortener**.
2. The **URLValidator** checks the input URL format.
3. The **URLShortener** generates a shortened URL using a hash-based approach.
4. The **URLMapper** stores the mapping between the shortened URL and the original URL.
5. The **CollisionDetector** checks for collisions between shortened URLs.
6. The user can retrieve the original URL by providing the shortened URL to the **URLMapper**.

### Technology Choices

**Implementation Code**
```python
import hashlib
import threading
from urllib.parse import urlparse

class URLValidator:
    def validate(self, url):
        try:
            result = urlparse(url)
            return all([result.scheme, result.netloc])
        except ValueError:
            return False

class URLShortener:
    def __init__(self, url_mapper):
        self.url_mapper = url_mapper
        self.lock = threading.Lock()

    def shorten(self, url):
        if not URLValidator().validate(url):
            raise ValueError("Invalid URL format")

        with self.lock:
            hash_object = hashlib.sha256(url.encode())
            shortened_url = hash_object.hexdigest()[:6]
            if self.url_mapper.exists(shortened_url):
                # Handle collision
                shortened_url = self._handle_collision(url, shortened_url)
            self.url_mapper.add_mapping(shortened_url, url)
            return shortened_url

    def _handle_collision(self, url, shortened_url):
        # Append a counter to the shortened URL
        counter = 0
        while self.url_mapper.exists(shortened_url + str(counter)):
            counter += 1
        return shortened_url + str(counter)

class URLMapper:
    def __init__(self):
        self.url_map = {}
        self.lock = threading.Lock()

    def add_mapping(self, shortened_url, url):
        with self.lock:
            self.url_map[shortened_url] = url

    def exists(self, shortened_url):
        with self.lock:
            return shortened_url in self.url_map

    def get_original_url(self, shortened_url):
        with self.lock:
            return self.url_map.get(shortened_url)

class URLShorteningLibrary:
    def __init__(self):
        self.url_mapper = URLMapper()
        self.url_shortener = URLShortener(self.url_mapper)

    def shorten(self, url):
        return self.url_shortener.shorten(url)

    def get_original_url(self, shortened_url):
        return self.url_mapper.get_original_url(shortened_url)

# Example usage
library = URLShorteningLibrary()
shortened_url = library.shorten("https://www.example.com")
print(shortened_url)
print(library.get_original_url(shortened_url))
```
