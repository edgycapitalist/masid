### Overview

The rate limiter module is designed to provide a flexible and thread-safe way to limit the rate at which requests are made to a system. The module uses the token bucket algorithm to implement rate limiting.

### Components

*   **Token Bucket Algorithm**: The token bucket algorithm is used to implement rate limiting. It works by adding tokens to a bucket at a fixed rate, and removing tokens from the bucket as requests are made. If the bucket is empty, requests are blocked until tokens are added.
*   **Rate Limiter Class**: The `RateLimiter` class is the core of the module. It provides methods for creating and managing rate limiters, as well as for checking if a request is allowed.

*   **Thread Safety**: The module is designed to be thread-safe, using locks to protect access to shared resources.

### Design

#### Token Bucket Algorithm

The token bucket algorithm is implemented as follows:

*   Tokens are added to the bucket at a fixed rate (requests per second).
*   Each request removes one token from the bucket.
*   If the bucket is empty, requests are blocked until tokens are added.

#### Rate Limiter Class

The `RateLimiter` class has the following methods:

*   `__init__(name, rate, burst_size)`: Initializes a new rate limiter with the given name, rate, and burst size.
*   `allow_request()`: Checks if a request is allowed, and returns `True` if it is, `False` otherwise.
*   `get_rate()`: Returns the rate of the rate limiter.
*   `get_burst_size()`: Returns the burst size of the rate limiter.

The `rate_limited` decorator is used to apply rate limiting to a function. It takes a `RateLimiter` object as an argument, and applies the rate limiter to the function.

import threading
import time
from functools import wraps

class RateLimiter:
    def __init__(self, name, rate, burst_size):
        """
        Initializes a new rate limiter with the given name, rate, and burst size.

        Args:
            name (str): The name of the rate limiter.
            rate (int): The rate of the rate limiter (requests per second).
            burst_size (int): The burst size of the rate limiter.
        """
        self.name = name
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()
        self.lock = threading.Lock()

    def allow_request(self):
        """
        Checks if a request is allowed, and returns True if it is, False otherwise.
        """
        with self.lock:
            now = time.time()
            elapsed = now - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed * self.rate)
            self.last_update = now
            if self.tokens < 1:
                return False
            self.tokens -= 1
            return True

    def get_rate(self):
        """
        Returns the rate of the rate limiter.
        """
        return self.rate

    def get_burst_size(self):
        """
        Returns the burst size of the rate limiter.
        """
        return self.burst_size

def rate_limited(rate_limiter):
    """
    Applies rate limiting to a function.

    Args:
        rate_limiter (RateLimiter): The rate limiter to apply to the function.

    Returns:
        A decorator that applies rate limiting to a function.
    """
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            if not rate_limiter.allow_request():
                raise Exception("Rate limit exceeded")
            return func(*args, **kwargs)
        return wrapper
