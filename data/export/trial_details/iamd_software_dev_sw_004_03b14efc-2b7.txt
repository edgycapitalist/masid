## Overview
The in-memory task queue is designed to manage tasks with priorities, track task status, and provide task retry functionality. The system consists of the following components:

*   **Task**: Represents a unit of work with a priority, status, and retry count.
*   **Task Queue**: Manages tasks and provides methods for adding, consuming, and tracking tasks.
*   **Worker**: Consumes tasks from the task queue and executes them.
*   **Task Status Tracker**: Tracks the status of tasks (pending, running, completed, failed).

## Component Design

### Task

*   **Class:** `Task`
*   **Attributes:**
    *   `id`: Unique task identifier
    *   `priority`: Task priority (integer)
    *   `status`: Task status (pending, running, completed, failed)
    *   `max_retries`: Maximum number of retries
    *   `retry_count`: Current retry count
    *   `func`: Task function to execute
    *   `args`: Task function arguments
    *   `kwargs`: Task function keyword arguments
*   **Methods:**
    *   `__init__`: Initializes the task with the given attributes
    *   `execute`: Executes the task function with the given arguments
    *   `retry`: Increments the retry count and updates the task status

### Task Queue

*   **Class:** `TaskQueue`
*   **Attributes:**
    *   `tasks`: List of tasks in the queue
    *   `lock`: Lock for thread-safe access to the queue
*   **Methods:**
    *   `__init__`: Initializes the task queue with an empty list of tasks
    *   `add_task`: Adds a task to the queue with the given priority
    *   `consume_task`: Consumes a task from the queue and returns it
    *   `update_task_status`: Updates the status of a task in the queue
    *   `shutdown`: Gracefully shuts down the task queue

### Worker

*   **Class:** `Worker`
*   **Attributes:**
    *   `task_queue`: Reference to the task queue
    *   `thread`: Worker thread
*   **Methods:**
    *   `__init__`: Initializes the worker with a reference to the task queue
    *   `start`: Starts the worker thread
    *   `run`: Consumes tasks from the task queue and executes them

### Task Status Tracker

*   **Class:** `TaskStatusTracker`
*   **Attributes:**
    *   `task_queue`: Reference to the task queue
*   **Methods:**
    *   `__init__`: Initializes the task status tracker with a reference to the task queue
    *   `get_task_status`: Returns the status of a task in the queue

import threading
import time
from collections import defaultdict
from enum import Enum
from typing import Callable, Any

class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, id: int, priority: int, max_retries: int, func: Callable, *args, **kwargs):
        self.id = id
        self.priority = priority
        self.status = TaskStatus.PENDING
        self.max_retries = max_retries
        self.retry_count = 0
        self.func = func
        self.args = args
        self.kwargs = kwargs

    def execute(self):
        try:
            self.func(*self.args, **self.kwargs)
            self.status = TaskStatus.COMPLETED
        except Exception as e:
            self.status = TaskStatus.FAILED
            self.retry_count += 1
            if self.retry_count < self.max_retries:
                self.status = TaskStatus.PENDING

    def retry(self):
        self.retry_count += 1
        if self.retry_count < self.max_retries:
            self.status = TaskStatus.PENDING

class TaskQueue:
    def __init__(self):
        self.tasks = []
        self.lock = threading.Lock()

    def add_task(self, task: Task):
        with self.lock:
            self.tasks.append(task)
            self.tasks.sort(key=lambda x: x.priority, reverse=True)

    def consume_task(self) -> Task:
        with self.lock:
            if self.tasks:
                return self.tasks.pop(0)
            return None

    def update_task_status(self, task: Task, status: TaskStatus):
        with self.lock:
            task.status = status

    def shutdown(self):
        with self.lock:
            self.tasks = []

class Worker:
    def __init__(self, task_queue: TaskQueue):
        self.task_queue = task_queue
        self.thread = threading.Thread(target=self.run)

    def start(self):
        self.thread.start()

    def run(self):
        while True:
            task = self.task_queue.consume_task()
            if task is None:
                break
            self.task_queue.update_task_status(task, TaskStatus.RUNNING)
            task.execute()
            self.task_queue.update_task_status(task, task.status)

class TaskStatusTracker:
    def __init__(self, task_queue: TaskQueue):
        self.task_queue = task_queue

    def get_task_status(self, task_id: int) -> TaskStatus:
        with self.task_queue.lock:
            for task in self.task_queue.tasks:
                if task.id == task_id:
                    return task.status
            return TaskStatus.COMPLETED

import logging
import threading

class Task:
    def __init__(self, priority, func):
        self.priority = priority
        self.func = func

    def execute(self):
        try:
            self.func()
        except Exception as e:
            logging.error(f"Task execution failed: {str(e)}")

class TaskQueue:
    def __init__(self):
        self.tasks = []
        self.lock = threading.Lock()

    def add_task(self, task):
        with self.lock:
            heapq.heappush(self.tasks, (-task.priority, task))

    def consume_task(self):
        with self.lock:
            if self.tasks:
                return heapq.heappop(self.tasks)[1]
            return None

    def cancel_task(self, task_id):
        with self.lock:
            for i, (priority, task) in enumerate(self.tasks):
                if task.id == task_id:
                    del self.tasks[i]
                    break
