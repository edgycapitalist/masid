### Overview

The data pipeline is designed to read CSV files, apply a chain of transformations, and output the results to CSV or JSON. The pipeline will support a plugin system for custom transformations and provide clear error reporting for malformed data.

### Modules

The data pipeline will consist of the following modules:

1. **Data Reader**: Responsible for reading CSV files with configurable schema validation.
2. **Transformation Chain**: Applies a chain of transformations (filtering, mapping, aggregation) to the data.
3. **Plugin System**: Allows for custom transformations to be added to the pipeline.
4. **Data Writer**: Outputs the results to CSV or JSON.
5. **Error Reporter**: Provides clear error reporting for malformed data.

### Interfaces

The following interfaces will be used to communicate between modules:

1. **Data**: A dictionary-like object that contains the data read from the CSV file.
2. **Transformation**: A function that takes in the data and returns the transformed data.
3. **Plugin**: A class that implements the `Transformation` interface and can be added to the pipeline.

### Data Flow

The data flow will be as follows:

1. The **Data Reader** reads the CSV file and validates the schema.
2. The **Data Reader** passes the data to the **Transformation Chain**.
3. The **Transformation Chain** applies each transformation in the chain to the data.
4. The **Transformation Chain** passes the transformed data to the **Data Writer**.
5. The **Data Writer** outputs the results to CSV or JSON.
6. If any errors occur during the pipeline, the **Error Reporter** will provide clear error reporting.

# Data Reader
class DataReader:
    def __init__(self, file_path, schema):
        """
        Initialize the DataReader with a file path and schema.

        Args:
            file_path (str): The path to the CSV file.
            schema (dict): A dictionary representing the schema of the data.
        """
        self.file_path = file_path
        self.schema = schema

    def read(self):
        """
        Read the CSV file and validate the schema.

        Returns:
            pd.DataFrame: The read data.

        Raises:
            ValueError: If the schema is invalid or the file cannot be read.
        """
        try:
            data = pd.read_csv(self.file_path)
            if not self._validate_schema(data):
                raise ValueError("Invalid schema")
            return data
        except Exception as e:
            raise ValueError("Error reading file: {}".format(str(e)))

    def _validate_schema(self, data):
        """
        Validate the schema of the data.

        Args:
            data (pd.DataFrame): The data to validate.

        Returns:
            bool: True if the schema is valid, False otherwise.
        """
        for column, dtype in self.schema.items():
            if column not in data.columns:
                return False
            if not all(isinstance(x, dtype) for x in data[column]):
                return False
        return True

        """
        Initialize the TransformationChain.

        """

        Args:
            data (pd.DataFrame): The data to validate.
