**Architecture Document: In-Memory Task Queue**

### Overview
The in-memory task queue is designed to manage tasks with priorities, track task status, and support task retry. The system consists of the following components:

*   **Task Queue:** A data structure to store tasks with priorities.
*   **Worker Threads:** Consume tasks from the task queue.
*   **Task Status Tracker:** Tracks the status of each task (pending, running, completed, failed).
*   **Task Retry Mechanism:** Retries failed tasks with a configurable max retries.

### Module Decomposition

The system is decomposed into the following modules:

*   `task_queue.py`: Implementation of the task queue data structure.
*   `worker.py`: Implementation of worker threads that consume tasks.
*   `task_status_tracker.py`: Implementation of the task status tracker.
*   `task_retry_mechanism.py`: Implementation of the task retry mechanism.
*   `main.py`: Main application that ties together all the modules.

### Interfaces

The following interfaces are defined:

*   `TaskQueueInterface`: Defines methods for adding tasks, consuming tasks, and tracking task status.
*   `WorkerInterface`: Defines methods for worker threads to consume tasks.
*   `TaskStatusTrackerInterface`: Defines methods for tracking task status.
*   `TaskRetryMechanismInterface`: Defines methods for retrying failed tasks.

### Data Flow

The data flow between components is as follows:

1.  **Task Addition:** A task is added to the task queue with a priority.
2.  **Task Consumption:** A worker thread consumes a task from the task queue.
3.  **Task Status Tracking:** The task status tracker updates the status of the task (pending, running, completed, failed).
4.  **Task Retry:** If a task fails, the task retry mechanism retries the task with a configurable max retries.

#### task_queue.py
```python
import heapq

class TaskQueue:
    def __init__(self):
        self.queue = []
        self.task_id = 0

    def add_task(self, task, priority):
        """Add a task to the queue with a priority."""
        self.task_id += 1
        heapq.heappush(self.queue, (priority, self.task_id, task))
        return self.task_id

    def consume_task(self):
        """Consume a task from the queue."""
        if not self.queue:
            return None
        _, _, task = heapq.heappop(self.queue)
        return task
```

class Worker:

        self.thread = threading.Thread(target=self.run)

        while True:
            task = self.task_queue.consume_task()
            if task is None:

            try:

import heapq

class Task:
    def __init__(self, id, priority, func):
        self.id = id
        self.priority = priority
        self.func = func

class TaskQueue:
    def __init__(self):
        self.queue = []
        self.task_id = 0

        self.task_id += 1
        task = Task(self.task_id, priority, func)
        heapq.heappush(self.queue, (task.priority, task))

        _, task = heapq.heappop(self.queue)
