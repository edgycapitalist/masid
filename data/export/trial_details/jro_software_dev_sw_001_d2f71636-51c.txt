### Overview

The URL shortening library is designed to provide a hash-based URL shortening functionality, allowing users to shorten long URLs and retrieve the original URL from the shortened version. The library will handle collision detection, input validation, and ensure thread safety for concurrent usage.

### Module Decomposition

The library will consist of the following modules:

*   **url_validator.py**: Responsible for validating the input URL format.
*   **hash_generator.py**: Generates a unique hash for the input URL.
*   **collision_detector.py**: Detects and handles collisions between hashes.
*   **url_shortener.py**: Provides the main URL shortening functionality.
*   **url_lookup.py**: Handles reverse lookup from shortened URL to original URL.
*   **thread_safe_wrapper.py**: Ensures thread safety for concurrent usage.

### Interfaces

The library will provide the following interfaces:

*   **`shorten_url(long_url: str) -> str`**: Accepts a long URL and returns a shortened version.
*   **`lookup_url(short_url: str) -> str`**: Accepts a shortened URL and returns the original URL.
*   **`validate_url(url: str) -> bool`**: Validates the input URL format.

### Data Flow

The data flow between modules will be as follows:

1.  The user calls the `shorten_url` method with a long URL.
2.  The `url_validator` module checks if the input URL is valid.
3.  If the URL is valid, the `hash_generator` module generates a unique hash for the URL.
4.  The `collision_detector` module checks if the generated hash is already in use.
5.  If a collision is detected, the `collision_detector` module generates a new hash.
6.  The `url_shortener` module stores the mapping between the original URL and the shortened URL.
7.  The `lookup_url` method retrieves the original URL from the shortened URL using the stored mapping.

### Technology Choices

The library will be implemented using Python 3.9 or higher, utilizing the `hashlib` library for hash generation and the `threading` library for thread safety.

import urllib.parse

def validate_url(url: str) -> bool:
    """
    Validates the input URL format.
    
    Args:
    url (str): The input URL to validate.
    
    Returns:
    bool: True if the URL is valid, False otherwise.
    """
    try:
        result = urllib.parse.urlparse(url)
        return all([result.scheme, result.netloc])
    except ValueError:
        return False

# hash_generator.py
import hashlib

def generate_hash(url: str) -> str:
    """
    Generates a unique hash for the input URL.
    
    Args:
    url (str): The input URL to generate a hash for.
    
    Returns:
    str: A unique hash for the input URL.
    """
    return hashlib.sha256(url.encode()).hexdigest()[:6]

# collision_detector.py
import threading

class CollisionDetector:
    def __init__(self):
        self.lock = threading.Lock()
        self.used_hashes = set()

    def is_collision(self, hash: str) -> bool:
        """
        Checks if the generated hash is already in use.
        
        Args:
        hash (str): The generated hash to check.
        
        Returns:
        bool: True if the hash is already in use, False otherwise.
        """
        with self.lock:
            return hash in self.used_hashes

    def add_hash(self, hash: str):
        """
        Adds a new hash to the set of used hashes.
        
        Args:
        hash (str): The new hash to add.
        """
        with self.lock:
            self.used_hashes.add(hash)

# url_shortener.py
import threading

class URLShortener:
    def __init__(self):
        self.lock = threading.Lock()
        self.url_map = {}
        self.collision_detector = CollisionDetector()

    def shorten_url(self, long_url: str) -> str:
        """
        Accepts a long URL and returns a shortened version.
        
        Args:
        long_url (str): The long URL to shorten.
        
        Returns:
        str: A shortened version of the input URL.
        """
        if not validate_url(long_url):
            raise ValueError("Invalid URL")

        hash = generate_hash(long_url)
        while self.collision_detector.is_collision(hash):
            hash = generate_hash(long_url)

        self.collision_detector.add_hash(hash)
        with self.lock:
            self.url_map[hash] = long_url

        return f"http://short.url/{hash}"

    def lookup_url(self, short_url: str) -> str:
        """
        Accepts a shortened URL and returns the original URL.
        
        Args:
        short_url (str): The shortened URL to lookup.
        
        Returns:
        str: The original URL corresponding to the shortened URL.
        """
        hash = short_url.split("/")[-1]
        with self.lock:
            return self.url_map.get(hash)

import urllib3

def validate_url(url: str) -> bool:
    """
    Validates the input URL format.
    
    Args:
    url (str): The input URL to validate.
    
    Returns:
    bool: True if the URL is valid, False otherwise.
    """
    try:
        result = urllib3.parse_url(url)
        return all([result.scheme, result.netloc])
    except ValueError:
        return False

# hash_generator.py
import hashlib

def generate_hash(url: str) -> str:
    """
    Generates a unique hash for the input URL.
    
    Args:
    url (str): The input URL to generate a hash for.
    
    Returns:
    str: A unique hash for the input URL.
    """
    return hashlib.sha512(url.encode()).hexdigest()[:6]

# collision_detector.py
import threading

class CollisionDetector:
    def __init__(self):
        self.lock = threading.Lock()
        self.used_hashes = set()

    def is_collision(self, hash: str) -> bool:
        """
        Checks if the generated hash is already in use.
        
        Args:
        hash (str): The generated hash to check.
        
        Returns:
        bool: True if the hash is already in use, False otherwise.
        """
        with self.lock:
            return hash in self.used_hashes

    def add_hash(self, hash: str):
        """
        Adds a new hash to the set of used hashes.
        
        Args:
        hash (str): The new hash to add.
        """
        with self.lock:
            self.used_hashes.add(hash)

import threading

class URLShortener:
    def __init__(self):
        self.lock = threading.Lock()
        self.url_map = {}
        self.collision_detector = CollisionDetector()

    def shorten_url(self, long_url: str) -> str:
        """
        Accepts a long URL and returns a shortened version.
        
        Args:
        long_url (str): The long URL to shorten.
        
        Returns:
        str: A shortened version of the input URL.
        """
        if not validate_url(long_url):
            raise ValueError("Invalid URL")

        hash = generate_hash(long_url)
        while self.collision_detector.is_collision(hash):
            hash = generate_hash(long_url)

        """
        Accepts a shortened URL and returns the original URL.
        
        Args:
        short_url (str): The shortened URL to lookup.
        
        Returns:
        str: The original URL corresponding to the shortened URL.
        """
        hash = short_url.split("/")[-1]
        with self.lock:
            return self.url_map.get(hash)

7.  The `lookup_url` method retrieves the original URL from the shortened URL using the stored mapping.

        str: A shortened version of the input URL.

        str: The original URL corresponding to the shortened URL.

            return hash in self.used_hashes

    def add_hash(self, hash: str):
        """
        Adds a new hash to the set of used hashes.
        
        Args:
        hash (str): The new hash to add.
        """
        with self.lock:
            self.used_hashes.add(hash)

# url_shortener.py
import threading

class URLShortener:
    def __init__(self):
        self.lock = threading.Lock()
        self.url_map = {}
        self.collision_detector = CollisionDetector()

    def shorten_url(self, long_url: str) -> str:
        """
        Accepts a long URL and returns a shortened version.
        
        Args:
        long_url (str): The long URL to shorten.
        
        Returns:
        str: A shortened version of the input URL.
        """
        if not validate_url(long_url):
            raise ValueError("Invalid URL")

        hash = generate_hash(long_url)
        while self.collision_detector.is_collision(hash):
            hash = generate_hash(long_url)

        self.collision_detector.add_hash(hash)
        with self.lock:
            self.url_map[hash] = long_url

        return f"http://short.url/{hash}"

    def lookup_url(self, short_url: str) -> str:
        """
        Accepts a shortened URL and returns the original URL.
        
        Args:
        short_url (str): The shortened URL to lookup.
        
        Returns:
        str: The original URL corresponding to the shortened URL.
        """
        hash = short_url.split("/")[-1]
        with self.lock:
            return self.url_map.get(hash)

# thread_safe_wrapper.py
import threading

class ThreadSafeWrapper:
    def __init__(self, url_shortener: URLShortener):
        self.url_shortener = url_shortener
        self.lock = threading.Lock()

    def shorten_url(self, long_url: str) -> str:
        with self.lock:
            return self.url_shortener.shorten_url(long_url)

    def lookup_url(self, short_url: str) -> str:
        with self.lock:
            return self.url_shortener.lookup_url(short_url)

from thread_safe_wrapper import ThreadSafeWrapper
