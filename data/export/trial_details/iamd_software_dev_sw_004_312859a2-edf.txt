## Introduction
The goal of this project is to design and implement an in-memory task queue in Python that supports task prioritization, worker threads, task status tracking, task retry, and graceful shutdown.

## System Components
The system consists of the following components:

1. **Task Queue**: An in-memory queue that stores tasks with their respective priorities.
2. **Task**: A unit of work that needs to be executed. Each task has a unique ID, priority, and a reference to the function that needs to be executed.
3. **Worker Threads**: A pool of threads that consume tasks from the task queue and execute them.
4. **Task Status Tracker**: A component that tracks the status of each task (pending, running, completed, failed).
5. **Task Retry Mechanism**: A component that retries failed tasks with a configurable maximum number of retries.

## System Flow
The system flow is as follows:

1. **Task Submission**: A task is submitted to the task queue with a priority.
2. **Task Queueing**: The task is added to the task queue based on its priority.
3. **Worker Thread Selection**: A worker thread is selected from the pool of available threads.
4. **Task Execution**: The selected worker thread consumes the task from the task queue and executes it.
5. **Task Status Update**: The task status is updated to "running" during execution and "completed" or "failed" upon completion.
6. **Task Retry**: If a task fails, the task retry mechanism is triggered, and the task is retried with a configurable maximum number of retries.
7. **Shutdown**: The system is shut down gracefully, ensuring that all tasks are completed or failed.

## Data Structures
The following data structures are used:

1. **Priority Queue**: A priority queue (implemented using `heapq`) is used to store tasks based on their priorities.
2. **Task Dictionary**: A dictionary is used to store task status and other relevant information.
3. **Worker Thread Pool**: A pool of worker threads (implemented using `threading`) is used to consume tasks from the task queue.

import heapq
import threading
from typing import Callable, Dict

class Task:
    def __init__(self, id: int, priority: int, func: Callable, max_retries: int = 3):
        self.id = id
        self.priority = priority
        self.func = func
        self.max_retries = max_retries
        self.retries = 0
        self.status = "pending"

class TaskQueue:
    def __init__(self):
        self.queue = []
        self.task_dict: Dict[int, Task] = {}
        self.lock = threading.Lock()

    def add_task(self, task: Task):
        with self.lock:
            heapq.heappush(self.queue, (task.priority, task.id))
            self.task_dict[task.id] = task

    def get_task(self) -> Task:
        with self.lock:
            if not self.queue:
                return None
            _, task_id = heapq.heappop(self.queue)
            return self.task_dict[task_id]

class WorkerThread(threading.Thread):
    def __init__(self, task_queue: TaskQueue):
        super().__init__()
        self.task_queue = task_queue

    def run(self):
        while True:
            task = self.task_queue.get_task()
            if task is None:
                break
            task.status = "running"
            try:
                task.func()
                task.status = "completed"
            except Exception as e:
                task.status = "failed"
                if task.retries < task.max_retries:
                    task.retries += 1
                    self.task_queue.add_task(task)

class TaskQueueManager:
    def __init__(self):
        self.task_queue = TaskQueue()
        self.worker_threads: List[WorkerThread] = []

    def start(self):
        for _ in range(5):  # Start 5 worker threads
            worker_thread = WorkerThread(self.task_queue)
            worker_thread.start()
            self.worker_threads.append(worker_thread)

    def stop(self):
        for worker_thread in self.worker_threads:
            worker_thread.join()

import heapq
import threading
from typing import Callable, Dict

class Task:
    def __init__(self, id: int, priority: int, func: Callable, max_retries: int = 3):
        """
        Initialize a Task object.

        Args:
        id (int): Unique task ID.
        priority (int): Task priority.
        func (Callable): Function to be executed.
        max_retries (int): Maximum number of retries. Defaults to 3.

        Raises:
        ValueError: If priority is a negative integer or func is not callable.
        """
        if priority < 0:
            raise ValueError("Priority must be a non-negative integer")
        if not callable(func):
            raise ValueError("Function reference is invalid or missing")
        self.id = id
        self.priority = priority
        self.func = func
        self.max_retries = max_retries
        self.retries = 0
        self.status = "pending"

class TaskQueue:
    def __init__(self):
        """
        Initialize a TaskQueue object.
        """
        self.queue = []
        self.task_dict: Dict[int, Task] = {}
        self.lock = threading.Lock()

    def add_task(self, task: Task):
        """
        Add a task to the task queue.

        Args:
        task (Task): Task to be added.

        Raises:
        ValueError: If task priority is a negative integer.
        """
        with self.lock:
            if task.priority < 0:
                raise ValueError("Priority must be a non-negative integer")
            heapq.heappush(self.queue, (task.priority, task.id))
            self.task_dict[task.id] = task

    def get_task(self) -> Task:
        """
        Get a task from the task queue.

        Returns:
        Task: Retrieved task or None if the queue is empty.
        """
        with self.lock:
            if not self.queue:
                return None
            _, task_id = heapq.heappop(self.queue)
            return self.task_dict[task_id]

class WorkerThread(threading.Thread):
    def __init__(self, task_queue: TaskQueue):
        """
        Initialize a WorkerThread object.

        Args:
        task_queue (TaskQueue): Task queue to consume tasks from.
        """
        super().__init__()
        self.task_queue = task_queue
        self.stop_event = threading.Event()

    def run(self):
        """
        Run the worker thread.
        """
        while not self.stop_event.is_set():
            task = self.task_queue.get_task()
            if task is None:
                continue
            task.status = "running"
            try:
                task.func()
                task.status = "completed"
            except Exception as e:
                task.status = "failed"
                if task.retries < task.max_retries:
                    task.retries += 1
                    self.task_queue.add_task(task)

    def stop(self):
        """
        Stop the worker thread.
        """
        self.stop_event.set()

    def __init__(self):
        """
        Initialize a TaskQueueManager object.
        """
        self.task_queue = TaskQueue()
        self.worker_threads: list[WorkerThread] = []

            if task.priority < 0:

        Args:

        self.task_queue = task_queue

        Stop the worker thread.
