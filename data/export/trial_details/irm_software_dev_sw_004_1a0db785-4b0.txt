### Overview

The in-memory task queue is designed to manage tasks with priorities, support worker threads, and provide task status tracking. The system will be implemented in Python, utilizing threading and queue data structures.

### Components

1. **Task**: Represents a single task with a unique ID, priority, and a callable function.
2. **Task Queue**: An in-memory queue that stores tasks and manages their priorities.
3. **Worker Thread**: A thread that consumes tasks from the task queue and executes them.
4. **Task Status Tracker**: Tracks the status of each task (pending, running, completed, failed).

### System Design

1. **Task Queue Structure**:
	* The task queue will be implemented using a priority queue (heapq) to ensure tasks are processed in order of priority.
	* Each task will be wrapped in a `Task` object, which contains the task ID, priority, callable function, and status.
2. **Worker Thread**:
	* Worker threads will be created and started when the system initializes.
	* Each worker thread will continuously consume tasks from the task queue and execute them.
3. **Task Status Tracking**:
	* The task status tracker will be implemented using a dictionary, where each task ID maps to its current status.
4. **Task Retry Mechanism**:
	* When a task fails, it will be retried a configurable number of times (max retries).
	* If all retries fail, the task will be marked as failed.

### Implementation Details

1. **Task Class**:
	* `__init__`: Initializes the task with a unique ID, priority, and callable function.
	* `execute`: Executes the task and updates its status.
2. **Task Queue Class**:
	* `__init__`: Initializes the task queue with a priority queue (heapq).
	* `add_task`: Adds a task to the queue with its priority.
	* `get_task`: Retrieves the next task from the queue based on priority.
3. **Worker Thread Class**:
	* `__init__`: Initializes the worker thread with a task queue and a task status tracker.
	* `run`: Continuously consumes tasks from the task queue and executes them.
4. **Task Status Tracker Class**:

import heapq
import threading
from typing import Callable

class Task:
    def __init__(self, task_id: str, priority: int, func: Callable):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.status = "pending"
        self.retries = 0

    def execute(self):
        try:
            self.func()
            self.status = "completed"
        except Exception as e:
            self.status = "failed"
            self.retries += 1
            if self.retries < 3:  # max retries
                raise

class TaskQueue:
    def __init__(self):
        self.queue = []

    def add_task(self, task: Task):
        heapq.heappush(self.queue, (task.priority, task))

    def get_task(self):
        if self.queue:
            return heapq.heappop(self.queue)[1]
        return None

class WorkerThread(threading.Thread):
    def __init__(self, task_queue: TaskQueue, task_status_tracker: dict):
        super().__init__()
        self.task_queue = task_queue
        self.task_status_tracker = task_status_tracker

    def run(self):
        while True:
            task = self.task_queue.get_task()
            if task:
                task.execute()
                self.task_status_tracker[task.task_id] = task.status

class TaskStatusTracker:
    def __init__(self):
        self.status = {}

    def update_status(self, task_id: str, status: str):
        self.status[task_id] = status

def main():
    task_queue = TaskQueue()
    task_status_tracker = TaskStatusTracker()

    # Add tasks to the queue
    task1 = Task("task1", 1, lambda: print("Task 1 executed"))
    task2 = Task("task2", 2, lambda: print("Task 2 executed"))
    task_queue.add_task(task1)
    task_queue.add_task(task2)

    # Create and start worker threads
    worker_threads = []
    for _ in range(2):
        worker_thread = WorkerThread(task_queue, task_status_tracker.status)
        worker_thread.start()
        worker_threads.append(worker_thread)

    # Wait for all tasks to complete
    while task_queue.queue:
        pass

    # Print final task status
    print("Final task status:")
    for task_id, status in task_status_tracker.status.items():
        print(f"{task_id}: {status}")

if __name__ == "__main__":
    main()

import heapq
import threading
from typing import Callable
from enum import Enum

class TaskStatus(Enum):
    PENDING = 1
    RUNNING = 2
    COMPLETED = 3
    FAILED = 4

class Task:
    def __init__(self, task_id: str, priority: int, func: Callable):
        self.task_id = task_id
        self.priority = priority
        self.func = func
        self.status = TaskStatus.PENDING
        self.retries = 0
        self.max_retries = 3  # configurable max retries

    def execute(self):
        try:
            self.func()
            self.status = TaskStatus.COMPLETED
        except Exception as e:
            self.status = TaskStatus.FAILED
            self.retries += 1
            if self.retries < self.max_retries:
                raise

    def retry(self):
        if self.retries < self.max_retries:
            self.execute()

class TaskQueue:
    def __init__(self):
        self.queue = []
        self.lock = threading.Lock()

    def add_task(self, task: Task):
        with self.lock:
            heapq.heappush(self.queue, (task.priority, task))

    def get_task(self):
        with self.lock:
            if self.queue:
                return heapq.heappop(self.queue)[1]
            return None

class WorkerThread(threading.Thread):
    def __init__(self, task_queue: TaskQueue, task_status_tracker: dict):
        super().__init__()
        self.task_queue = task_queue
        self.task_status_tracker = task_status_tracker

    def run(self):
        while True:
            task = self.task_queue.get_task()
            if task:
                try:
                    task.status = TaskStatus.RUNNING
                    self.task_status_tracker[task.task_id] = task.status
                    task.execute()
                    self.task_status_tracker[task.task_id] = task.status
                except Exception as e:
                    task.retry()
                    self.task_status_tracker[task.task_id] = task.status

class TaskStatusTracker:
    def __init__(self):
        self.status = {}
        self.lock = threading.Lock()

    def update_status(self, task_id: str, status: TaskStatus):
        with self.lock:
            self.status[task_id] = status

def main():
    task_queue = TaskQueue()
    task_status_tracker = TaskStatusTracker()

    # Add tasks to the queue
    task1 = Task("task1", 1, lambda: print("Task 1 executed"))
    task2 = Task("task2", 2, lambda: print("Task 2 executed"))
    task_queue.add_task(task1)
    task_queue.add_task(task2)

    # Create and start worker threads
    worker_threads = []
    for _ in range(2):
        worker_thread = WorkerThread(task_queue, task_status_tracker.status)
        worker_thread.start()
        worker_threads.append(worker_thread)

    # Wait for all tasks to complete
    while task_queue.queue:
        pass

    # Print final task status
    print("Final task status:")
    for task_id, status in task_status_tracker.status.items():
        print(f"{task_id}: {status}")

if __name__ == "__main__":
    main()
