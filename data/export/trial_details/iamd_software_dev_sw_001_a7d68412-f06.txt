## Introduction

The URL Shortener Library is a Python library that provides URL shortening functionality using a hash-based approach. The library aims to provide a simple and efficient way to shorten URLs while ensuring thread safety and handling collision detection.

## System Components

The URL Shortener Library consists of the following components:

1. **URLValidator**: Responsible for validating the input URL format.
2. **URLHasher**: Generates a shortened URL using a hash-based approach.
3. **URLDatabase**: Stores the mapping between shortened URLs and original URLs.
4. **URLShortener**: The main interface for shortening URLs and performing reverse lookups.

## System Design

The system design is based on the following principles:

1. **Modularity**: Each component is designed to be modular and independent, allowing for easy maintenance and extension.
2. **Thread Safety**: The library uses thread-safe data structures and locking mechanisms to ensure concurrent usage.
3. **Collision Detection**: The library detects and handles collisions between shortened URLs.

### URLValidator

The URLValidator component uses the `urllib.parse` module to validate the input URL format. It checks for the following:

* Valid scheme (e.g., http, https)
* Valid netloc (e.g., example.com)
* Valid path (e.g., /path/to/resource)

### URLHasher

The URLHasher component uses a hash-based approach to generate shortened URLs. It uses the `hashlib` library to generate a SHA-256 hash of the input URL. The hash is then truncated to a fixed length (e.g., 6 characters) to produce the shortened URL.

### URLDatabase

The URLDatabase component uses a dictionary to store the mapping between shortened URLs and original URLs. The dictionary is thread-safe using the `threading.Lock` mechanism.

### URLShortener

The URLShortener component is the main interface for shortening URLs and performing reverse lookups. It uses the URLValidator, URLHasher, and URLDatabase components to perform the following operations:

* Shorten URL: Validates the input URL, generates a shortened URL using the URLHasher, and stores the mapping in the URLDatabase.
* Reverse Lookup: Takes a shortened URL as input and returns the original URL from the URLDatabase.

```python
import hashlib
import threading
from urllib.parse import urlparse

class URLValidator:
    def __init__(self):
        self.schemes = ['http', 'https']

    def validate(self, url):
        try:
            parsed_url = urlparse(url)
            if parsed_url.scheme not in self.schemes:
                raise ValueError('Invalid scheme')
            if not parsed_url.netloc:
                raise ValueError('Invalid netloc')
            if not parsed_url.path:
                raise ValueError('Invalid path')
            return True
        except ValueError as e:
            raise ValueError('Invalid URL format') from e

class URLHasher:
    def __init__(self, length=6):
        self.length = length

    def hash(self, url):
        hash_object = hashlib.sha256(url.encode())
        return hash_object.hexdigest()[:self.length]

class URLDatabase:
    def __init__(self):
        self.database = {}
        self.lock = threading.Lock()

    def store(self, short_url, original_url):
        with self.lock:
            self.database[short_url] = original_url

    def retrieve(self, short_url):
        with self.lock:
            return self.database.get(short_url)

class URLShortener:
    def __init__(self):
        self.validator = URLValidator()
        self.hasher = URLHasher()
        self.database = URLDatabase()

    def shorten(self, url):
        if not self.validator.validate(url):
            raise ValueError('Invalid URL format')
        short_url = self.hasher.hash(url)
        if short_url in self.database.retrieve(short_url):
            raise ValueError('Collision detected')
        self.database.store(short_url, url)
        return short_url

    def reverse_lookup(self, short_url):
        return self.database.retrieve(short_url)

# Example usage
url_shortener = URLShortener()
original_url = 'https://example.com/path/to/resource'
short_url = url_shortener.shorten(original_url)
print(f'Shortened URL: {short_url}')
print(f'Original URL: {url_shortener.reverse_lookup(short_url)}')
```
