### Overview

The URL Shortening Library is a Python module that provides a simple and efficient way to shorten URLs using a hash-based approach. The library supports reverse lookup, collision detection, input validation, and is designed to be thread-safe for concurrent usage.

### Module Decomposition

The library consists of the following modules:

* `url_shortener`: The main module that provides the URL shortening functionality.
* `hash_generator`: A module that generates a unique hash for a given URL.
* `url_validator`: A module that validates the input URL format.
* `collision_detector`: A module that detects and handles collisions.
* `thread_safe_dict`: A thread-safe dictionary implementation for storing the URL mappings.

### Interfaces

The library provides the following interfaces:

* `shorten_url(long_url)`: Returns a shortened version of the input URL.
* `get_original_url(short_url)`: Returns the original URL corresponding to the input short URL.

### Data Flow

The data flow of the library is as follows:

1. The `shorten_url` function takes a long URL as input and passes it to the `url_validator` module for validation.
2. If the input URL is valid, the `hash_generator` module generates a unique hash for the URL.
3. The `collision_detector` module checks if the generated hash is already in use. If it is, a new hash is generated.
4. The `thread_safe_dict` module stores the mapping between the short URL and the original URL.
5. The `get_original_url` function takes a short URL as input and returns the corresponding original URL from the `thread_safe_dict`.

### Technology Choices

The library uses the following technologies:

* Python 3.x as the programming language.
* `hashlib` library for generating hashes.
* `urllib.parse` library for validating URLs.
* `threading` library for implementing thread safety.

### Implementation
```python
import hashlib
import urllib.parse
import threading

class UrlValidator:
    def __init__(self):
        self.allowed_schemes = ['http', 'https']

    def is_valid(self, url):
        try:
            parsed_url = urllib.parse.urlparse(url)
            return parsed_url.scheme in self.allowed_schemes
        except ValueError:
            return False

class HashGenerator:
    def __init__(self):
        self.hash_length = 6

    def generate_hash(self, url):
        return hashlib.sha256(url.encode()).hexdigest()[:self.hash_length]

class CollisionDetector:
    def __init__(self):
        self.hash_map = {}

    def is_collision(self, hash_value):
        return hash_value in self.hash_map

    def add_hash(self, hash_value, url):
        self.hash_map[hash_value] = url

class ThreadSafeDict:
    def __init__(self):
        self.lock = threading.Lock()
        self.dict = {}

    def get(self, key):
        with self.lock:
            return self.dict.get(key)

    def put(self, key, value):
        with self.lock:
            self.dict[key] = value

class UrlShortener:
    def __init__(self):
        self.url_validator = UrlValidator()
        self.hash_generator = HashGenerator()
        self.collision_detector = CollisionDetector()
        self.thread_safe_dict = ThreadSafeDict()

    def shorten_url(self, long_url):
        if not self.url_validator.is_valid(long_url):
            raise ValueError("Invalid URL")

        hash_value = self.hash_generator.generate_hash(long_url)
        while self.collision_detector.is_collision(hash_value):
            hash_value = self.hash_generator.generate_hash(long_url)

        self.collision_detector.add_hash(hash_value, long_url)
        self.thread_safe_dict.put(hash_value, long_url)

        return f"http://short.url/{hash_value}"

    def get_original_url(self, short_url):
        try:
            parsed_url = urllib.parse.urlparse(short_url)
            hash_value = parsed_url.path[1:]
            return self.thread_safe_dict.get(hash_value)
        except ValueError:
            return None

# Example usage
url_shortener = UrlShortener()
long_url = "https://www.example.com"
short_url = url_shortener.shorten_url(long_url)
print(f"Short URL: {short_url}")
print(f"Original URL: {url_shortener.get_original_url(short_url)}")
```
