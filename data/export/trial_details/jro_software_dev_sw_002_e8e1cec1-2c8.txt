### Overview

The rate limiter module is designed to regulate the frequency of requests to prevent abuse or overload of system resources. This module uses the token bucket algorithm to manage rate limiting, allowing for configurable rates and burst sizes. Multiple named rate limiters can be created to support various use cases, such as per-user or per-endpoint rate limiting.

### Module Structure

The rate limiter module consists of the following components:

*   `RateLimiter`: The core class responsible for managing the token bucket algorithm and enforcing rate limits.
*   `RateLimiterFactory`: A factory class that creates and manages multiple named rate limiters.
*   `ratelimit`: A decorator that applies rate limiting to functions.

### Interfaces

The rate limiter module exposes the following interfaces:

*   `RateLimiter`: `__init__(rate, burst_size)`, `allow_request()`, `get_remaining_tokens()`
*   `RateLimiterFactory`: `create_limiter(name, rate, burst_size)`, `get_limiter(name)`
*   `ratelimit`: `__call__(func, limiter_name)`

### Data Flow

The data flow between components is as follows:

1.  The `RateLimiterFactory` creates a new `RateLimiter` instance with the specified rate and burst size.
2.  The `ratelimit` decorator applies the rate limiter to a function, using the `RateLimiter` instance created by the `RateLimiterFactory`.
3.  When the decorated function is called, the `ratelimit` decorator checks if the request is allowed by calling the `allow_request()` method of the `RateLimiter` instance.
4.  If the request is allowed, the `RateLimiter` instance updates its internal state to reflect the consumed tokens.

### Technology Choices

The rate limiter module is implemented in Python, utilizing the following technologies:

*   `threading`: For thread-safe operation
*   `functools`: For creating the `ratelimit` decorator

import threading
import time
from functools import wraps

class RateLimiter:
    def __init__(self, rate, burst_size):
        """
        Initializes a rate limiter with the specified rate and burst size.

        Args:
            rate (int): The rate of requests per second.
            burst_size (int): The maximum number of requests allowed in a burst.
        """
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()
        self.lock = threading.Lock()

    def allow_request(self):
        """
        Checks if a request is allowed based on the current token bucket state.

        Returns:
            bool: True if the request is allowed, False otherwise.
        """
        with self.lock:
            current_time = time.time()
            elapsed_time = current_time - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
            self.last_update = current_time

            if self.tokens < 1:
                return False
            self.tokens -= 1
            return True

    def get_remaining_tokens(self):
        """
        Returns the number of remaining tokens in the bucket.

        Returns:
            int: The number of remaining tokens.
        """
        with self.lock:
            current_time = time.time()
            elapsed_time = current_time - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
            self.last_update = current_time
            return self.tokens


class RateLimiterFactory:
    def __init__(self):
        """
        Initializes a rate limiter factory.
        """
        self.limiters = {}
        self.lock = threading.Lock()

    def create_limiter(self, name, rate, burst_size):
        """
        Creates a new rate limiter with the specified name, rate, and burst size.

        Args:
            name (str): The name of the rate limiter.
            rate (int): The rate of requests per second.
            burst_size (int): The maximum number of requests allowed in a burst.
        """
        with self.lock:
            if name in self.limiters:
                raise ValueError("Rate limiter with the specified name already exists")
            self.limiters[name] = RateLimiter(rate, burst_size)

    def get_limiter(self, name):
        """
        Returns the rate limiter with the specified name.

        Args:
            name (str): The name of the rate limiter.

        Returns:
            RateLimiter: The rate limiter instance.
        """
        with self.lock:
            return self.limiters.get(name)


def ratelimit(limiter_name):
    """
    A decorator that applies rate limiting to a function.

    Args:
        limiter_name (str): The name of the rate limiter to use.

    Returns:
        callable: The decorated function.
    """
    factory = RateLimiterFactory()

    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            limiter = factory.get_limiter(limiter_name)
            if limiter is None:
                raise ValueError("Rate limiter with the specified name does not exist")
            if not limiter.allow_request():
                raise Exception("Rate limit exceeded")
            return func(*args, **kwargs)
        return wrapper
    return decorator

### Overview

The rate limiter module consists of the following components:

*   `RateLimiter`: The core class responsible for managing the token bucket algorithm and enforcing rate limits.

*   `ratelimit`: A decorator that applies rate limiting to functions.

4.  If the request is allowed, the `RateLimiter` instance updates its internal state to reflect the consumed tokens.

    def __init__(self, rate, burst_size):
        """
        Initializes a rate limiter with the specified rate and burst size.

        Args:
            rate (int): The rate of requests per second.
            burst_size (int): The maximum number of requests allowed in a burst.
        """
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()
        self.lock = threading.Lock()

    def allow_request(self):
        """
        Checks if a request is allowed based on the current token bucket state.

        Returns:
            bool: True if the request is allowed, False otherwise.
        """
        with self.lock:
            current_time = time.time()
            elapsed_time = current_time - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
            self.last_update = current_time

            if self.tokens < 1:
                logger.warning("Rate limit exceeded")
                raise RateLimitExceededError("Rate limit exceeded")
            self.tokens -= 1
            return True

        Returns the number of remaining tokens in the bucket.

        Returns:

import logging
import threading
import time
from functools import wraps

# Define a custom exception class for rate limit exceeded errors
class RateLimitExceededError(Exception):
    pass

# Define a logger for the rate limiter module
logger = logging.getLogger(__name__)

class RateLimiter:
    def __init__(self, rate, burst_size):
        """
        Initializes a rate limiter with the specified rate and burst size.

        Args:
            rate (int): The rate of requests per second.
            burst_size (int): The maximum number of requests allowed in a burst.
        """
        self.rate = rate
        self.burst_size = burst_size
        self.tokens = burst_size
        self.last_update = time.time()
        self.lock = threading.Lock()

    def allow_request(self):
        """
        Checks if a request is allowed based on the current token bucket state.

        Returns:
            bool: True if the request is allowed, False otherwise.
        """
        with self.lock:
            current_time = time.time()
            elapsed_time = current_time - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
            self.last_update = current_time

            if self.tokens < 1:
                logger.warning("Rate limit exceeded")
                raise RateLimitExceededError("Rate limit exceeded")
            self.tokens -= 1
            return True

    def get_remaining_tokens(self):
        """
        Returns the number of remaining tokens in the bucket.

        Returns:
            int: The number of remaining tokens.
        """
        with self.lock:
            current_time = time.time()
            elapsed_time = current_time - self.last_update
            self.tokens = min(self.burst_size, self.tokens + elapsed_time * self.rate)
            self.last_update = current_time
            return self.tokens


class RateLimiterFactory:
    def __init__(self):
        """
        Initializes a rate limiter factory.
        """
        self.limiters = {}
        self.lock = threading.Lock()

    def create_limiter(self, name, rate, burst_size):
        """
        Creates a new rate limiter with the specified name, rate, and burst size.

        Args:
            name (str): The name of the rate limiter.
            rate (int): The rate of requests per second.
            burst_size (int): The maximum number of requests allowed in a burst.
        """
        with self.lock:
            if name in self.limiters:
                raise ValueError("Rate limiter with the specified name already exists")
            self.limiters[name] = RateLimiter(rate, burst_size)

    def get_limiter(self, name):
        """
        Returns the rate limiter with the specified name.

        Args:
            name (str): The name of the rate limiter.

        Returns:
            RateLimiter: The rate limiter instance.
        """
        with self.lock:
            return self.limiters.get(name)


def ratelimit(limiter_name, error_message="Rate limit exceeded"):
    """
    A decorator that applies rate limiting to a function.

    Args:
        limiter_name (str): The name of the rate limiter to use.
        error_message (str): The error message to raise when the rate limit is exceeded.

    Returns:
        callable: The decorated function.
    """
    factory = RateLimiterFactory()

    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            limiter = factory.get_limiter(limiter_name)
            if limiter is None:
                raise ValueError("Rate limiter with the specified name does not exist")
            try:
                if not limiter.allow_request():
                    raise RateLimitExceededError(error_message)
                return func(*args, **kwargs)
            except RateLimitExceededError as e:
                logger.warning(f"Rate limit exceeded: {e}")
                raise
        return wrapper
    return decorator
