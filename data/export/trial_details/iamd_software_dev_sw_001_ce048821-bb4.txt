## Overview
The URL shortening library is designed to provide a simple and efficient way to shorten URLs using a hash-based approach. The library will support reverse lookup from short URL to original URL, handle collision detection, and include input validation for valid URL format. The library is designed to be thread-safe for concurrent usage.

## Module Decomposition
The library will consist of the following modules:

* `url_shortener`: This module will contain the main class `URLShortener` that provides the URL shortening functionality.
* `url_validator`: This module will contain a class `URLValidator` that validates the input URL format.
* `hash_generator`: This module will contain a class `HashGenerator` that generates a unique hash for the input URL.
* `database`: This module will contain a class `Database` that stores the mapping between the short URL and the original URL.

## Interfaces
The `URLShortener` class will have the following interfaces:

* `shorten_url(long_url)`: This method takes a long URL as input and returns a shortened version of the URL.
* `get_original_url(short_url)`: This method takes a short URL as input and returns the original URL.
* `is_valid_url(url)`: This method takes a URL as input and returns a boolean indicating whether the URL is valid.

## Data Flow
The data flow of the library is as follows:

1. The user calls the `shorten_url` method with a long URL as input.
2. The `URLShortener` class calls the `URLValidator` class to validate the input URL format.
3. If the URL is valid, the `URLShortener` class calls the `HashGenerator` class to generate a unique hash for the input URL.
4. The `URLShortener` class then stores the mapping between the short URL and the original URL in the `Database` class.
5. The `URLShortener` class returns the shortened URL to the user.
6. The user can then call the `get_original_url` method with the short URL as input to retrieve the original URL.

The library will be implemented in Python and will use the following technologies:

* `hashlib` library for generating unique hashes
* `sqlite3` library for storing the mapping between the short URL and the original URL
* `threading` library for ensuring thread safety

import hashlib
import sqlite3
import threading

class URLValidator:
    def is_valid_url(self, url):
        # Implement URL validation logic here
        # For simplicity, we will just check if the URL starts with http or https
        return url.startswith("http://") or url.startswith("https://")

class HashGenerator:
    def generate_hash(self, url):
        # Generate a unique hash for the input URL
        return hashlib.sha256(url.encode()).hexdigest()[:6]

class Database:
    def __init__(self, db_name):
        self.db_name = db_name
        self.conn = sqlite3.connect(db_name)
        self.cursor = self.conn.cursor()
        self.cursor.execute("CREATE TABLE IF NOT EXISTS url_mapping (short_url TEXT PRIMARY KEY, long_url TEXT)")

    def store_mapping(self, short_url, long_url):
        self.cursor.execute("INSERT INTO url_mapping VALUES (?, ?)", (short_url, long_url))
        self.conn.commit()

    def retrieve_mapping(self, short_url):
        self.cursor.execute("SELECT long_url FROM url_mapping WHERE short_url = ?", (short_url,))
        return self.cursor.fetchone()

class URLShortener:
    def __init__(self):
        self.validator = URLValidator()
        self.hash_generator = HashGenerator()
        self.database = Database("url_shortener.db")
        self.lock = threading.Lock()

    def shorten_url(self, long_url):
        if not self.validator.is_valid_url(long_url):
            raise ValueError("Invalid URL format")

        with self.lock:
            short_url = self.hash_generator.generate_hash(long_url)
            while self.database.retrieve_mapping(short_url) is not None:
                # Handle collision detection
                short_url = self.hash_generator.generate_hash(long_url + str(self.database.retrieve_mapping(short_url)))
            self.database.store_mapping(short_url, long_url)
        return short_url

    def get_original_url(self, short_url):
        with self.lock:
            mapping = self.database.retrieve_mapping(short_url)
            if mapping is None:
                raise ValueError("Short URL not found")
            return mapping[0]

    def is_valid_url(self, url):
        return self.validator.is_valid_url(url)

# Example usage
url_shortener = URLShortener()
long_url = "https://www.example.com/very/long/url"
short_url = url_shortener.shorten_url(long_url)
print(short_url)
print(url_shortener.get_original_url(short_url))

import sqlite3

class URLShortener:
    def __init__(self, db_name):
